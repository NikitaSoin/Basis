## Замечания по чек-листу

1. **(п.1, существенное) Таблица «эффект→канал» неполна — «уроненная эстафета» по токсичности бенефициара.** ИНСТ прямо пишет: санкционная токсичность бенефициара исключена из его премии, «её добавит geo-company-analyst». В valuation_translation этого эффекта нет ни в одном канале (принят ≈0 через E13, но решение не эксплицировано там, где его будет искать сборщик). Тест: каждая позиция ИНСТ «переданная ГЕО» должна встречаться строкой в valuation_translation, пусть с magnitude ≈0.

2. **(п.2, чисто)** Опосредованный ставочный канал последовательно не внесён в гео-премию — сильнейшая часть отчёта.

3. **(п.3, косметика)** Оба канала C (ИНСТ и ГЕО) частично питаются одним фактом неликвидности малого free float — величины малы, задвоение косметическое.

4. **(п.4, косметика)** Имплайд-рынок честен, оговорка про «отдельно от ИНСТ» не отвечает на вопрос «отдельно ли от рынка».

5. **(п.5, БЛОКЕР) Протокол C08 выполнен формально, не операционально.** Исход не сформулирован как одна метрика (три разных исхода смешаны в один набор весов). Тест величины есть только у К2; К1 закрыт декретом без применения (хотя применим — Best Buy/глобальный офлайн-ритейл под маркетплейсами); ~5% унаследовано из прецедента K27, не выведено. Риск зеркальный K27: якорение на прецеденте вместо измерения. Тест: развести атрибуцию по двум исходам, дать метод каждому каналу, добавить контроль К1.

6. **(п.6, косметика/существенное-минус)** Смешение «санкции» и «добровольный де-рискинг» брендов — механизм назван неточно, влияет на трактовку S1.

7. **(п.7, чисто)** Кейсы используются как якоря механизмов, не как доказательство катастрофы.

8. **(п.8, существенное)** Три места ложной точности: (а) атрибуция суммируется ровно в 100% без остатка; (б) «B ~1-2% FCF» — база не определена при отрицательном FCF; (в) канал C «0-3%» не заякорен историческим спредом (C06 §2).

9. **(п.9, существенное) Два балла ниже собственного описания.** E4=3.0 при описании якоря 4 (K17-картина дословно совпадает с якорем 4); E6=2.0 при описании якоря 3 («работа на замороженных версиях управляема» = текст якоря 3). E14 без обязательной пары чисел (отступление не заявлено явно).

10. **(п.10, чисто)** Nirvana fallacy не найден.

11. **(п.11, существенное) Ключевые санкционно-статусные факты без источников** — статус Гуцериева (Швейцария/ЕС), доля маркетплейсов, число магазинов; load-bearing E4 (SWIFT 20-40%) на слабом единственном источнике.

12. **(п.12, чисто)** Пробелы честно ведут к null/снижению confidence.

13. **(п.13, чисто)** Барометр свежий, триггеры не наступили.

14. **(п.14, существенное-минус) Однонаправленное использование нефинализированной сделки JD.com** — засчитана как смягчение в 4 местах, обратный сценарий (срыв сделки) не заложен симметрично в баллы.

15. **(п.15, существенное) Δ с legacy-версией не показана** — особенно важно для K27-кейса: приписывал ли legacy-блок обвал санкциям.

**Вне чек-листа:** `disagreements: []` при том, что сценарные сетки ИНСТ и ГЕО не синхронизированы (разные рамки взвешивания) — стоит зафиксировать для сборщика.

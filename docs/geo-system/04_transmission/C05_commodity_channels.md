# C05. Товарные каналы: строение дисконта, теневой флот, маршруты

**Слой:** знания. Микромеханика субиндекса G5 и полей E3/E7: из чего состоит дисконт, как живёт теневой флот, что где переваливается и какие метрики отслеживать. Пара к C09 (там — мир; здесь — российское экспортное плечо).

## 1. Строение дисконта (декомпозиция — всегда, «дисконт вообще» — никогда)

Дисконт Urals/ESPO к Brent = сумма компонент, каждая со своим драйвером: **(а) логистическая** — теневой фрахт дороже белого (старый флот, страховка вне P&I-клубов, длинные плечи) — драйвер: доступность судов, ставки; **(б) комплаенс-премия покупателя** — плата НПЗ Индии/Китая/Турции за санкционный риск — драйвер: волны энфорсмента (листинги, вторичные угрозы: после SDN мейджоров 10.2025 премия подскочила — K23); **(в) потолочная** — для «белого» контура цена ≤ потолка по построению ($60 → $47,6 с 09.2025 → спор о ~$44 [поиск]); **(г) сортовая/качественная** — стабильный фон, не гео. Правила: ESPO структурно уже Urals (короткое плечо, минуя все западные горловины — T10); спред ESPO−Urals = чистый индикатор логистическо-комплаенсной компоненты; сжатие дисконта = адаптация (T09 фаза), расширение = энфорсмент-волна → **пила, не ступень**; каждую точку данных относи к компоненте, прежде чем двигать G5.

## 2. Теневой флот — актив с санкционной амортизацией

Масштаб: сотни судов (оценки широко расходятся, порядок 600–1000+ [ОЦЕНКА]); возрастной, со страховкой альтернативных юрисдикций. Механика энфорсмента: листинг судна (США/ЕС/UK) → порты и покупатели шарахаются → простой (после волны 01.2025 значимая часть листингованных стояла месяцами [ОЦЕНКА-пресса]) → адаптация: перефлагирование, перепродажа «новым» владельцам, STS-перегрузки → судно возвращается в оборот с бóльшим трением. Следствия: (1) эффективность каждого листинга временна → энфорсмент требует ПОВТОРНЫХ волн (календарь пакетов = календарь пилы G5); (2) флот — расходуемый капитал: аварийные и экологические инциденты старых судов — растущий хвостовой риск (и повод для L4-мер по «экологическим» основаниям — B04); (3) для Совкомфлота и рынка фрахта: санкционная премия к ставкам = доходная сторона того же явления (K29-двойственность).

## 3. Маршруты и узлы (справочный слой, сверять к дате)

Нефть: Балтика (Приморск/Усть-Луга) → Индия: длинное плечо (Суэц либо мыс Доброй Надежды при красноморских рисках — C09), недели пути, максимум компонент (а)+(б); Новороссийск → Средиземноморье/Индия: Босфор-фактор; Козьмино (ESPO) → КНР: дни, минимальный дисконт; Мурманск — перевалка и арктическая ветка. Нефтепродукты: сложнее (эмбарго ЕС 02.2023) — Турция/Африка/Латам, часть через реэкспортные хабы. СПГ: Ямал — своя логика (ледовые челноки + перевалка), санкционные проекты (K14) — отдельный режим. Уголь/удобрения/зерно: восточное плечо упирается в ж/д (Восточный полигон — T10 «сухопутный пролив»), южное — в Босфор и портовые мощности. STS-зоны и их зачистки (меры ЕС против перевалок в своих ИЭЗ 2024+) — подвижная карта. Внутренний твист K26: удары по НПЗ смещают баланс «сырая на экспорт vs переработка внутрь» — экспорт сырой может РАСТИ при топливном кризисе внутри; читать связку, а не один ряд.

## 4. Метрики мониторинга (в B03-ритме)

Еженедельно/ежемесячно: дисконт Urals (Минфин — среднемесячный [ФАКТ-источник], Argus-цитаты в прессе), спред ESPO−Urals, морской экспорт нефти (танкер-трекинг через агентства), доля листингованных судов в отгрузках и их простой, ставки фрахта Aframax «серого» рынка, статус потолка и его энфорсмент-новости, платёжные схемы за нефть (после ГПБ-санкций — K18: посредники/валюты — прецеденты срывов). Триггерные пороги — в B03 §3 (№4–6, 15–16).

## 5. Анти-паттерны

Один «дисконт» без декомпозиции (лечится §1 — иначе спутаешь адаптацию с энфорсментом и просмотришь разворот). Экстраполяция дисконта трендом через энфорсмент-календарь. Смешение мирового цикла цены (G13/C08) с гео-компонентой (G5) — правило C08 №1 для нефтянки. Оценка флота по числу судов без учёта листинг-статуса и простоя. Статичная карта маршрутов (красноморский фактор перекраивает плечи — держать связку с C09). Игнор доходной стороны трения (фрахтовая премия кому-то платится — вторая сторона каждой издержки есть чей-то P&L).

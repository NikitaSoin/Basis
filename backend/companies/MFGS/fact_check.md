### 4. Синтез-вердикт (акцент: related-party Славнефть)
MFGS — добывающая дочка, чьи финансы на уровне эмитента **искажены related-party каналом**: нефть продаётся вверх по группе по внутренним ценам, маржа и деньги изымаются займами/расчётами, поэтому при выручке >200 млрд ₽ и положительной EBITDA эмитент хронически убыточен, а дивиденды по обычке отсутствуют годами. Для миноритария по обыкновенной акции это значит: ни дивидендного, ни прибыльного возврата — только курсовая переоценка малоликвидной бумаги при free float ~5% и риске squeeze-out. Рынок последовательно дисконтирует бумагу (P/B стабильно 0.22–0.24). Справедливый диапазон 385 (P/B, консерв.) — 558 (база) — 878 ₽ (EV/EBITDA, оптим.) держится ТОЛЬКО при допущении ЧД=0; если долг 101.8 млрд реален — equity отрицателен. Вывод: оценка по EV/EBITDA нерелевантна для миноритария (EBITDA не его), якорь — P/B-дисконт; качество прибыли низкое; статус — спекулятивный 3-й эшелон с высоким governance-риском, без «купить/продать».

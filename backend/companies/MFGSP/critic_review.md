# Критик-ревью карточки MFGSP (Славнефть-Мегионнефтегаз, прив.)

## 1. ВЕРДИКТ ЦЕЛОГО
**4/10 — добротный текст вокруг сломанного ядра.** Качественные вкладки (бизнес-модель, рынок, макро, гео, управление) написаны зрело и согласованно вокруг одного честного тезиса: «убыточная дочка, маржа уходит вверх по группе, преф без голоса и без дивидендов». Но финансовый блок — главный для бумаги «преф vs обычка» — внутренне разорван: `financials.json` посчитан как обыкновенная MFGS (по цене 392,7 ₽), а `financials_summary.md` — как преф (по цене 244 ₽). Два разных набора справедливых цен в одной карточке. Для бумаги, чья ВСЯ суть — дисконт префа к обычке, это критический дефект.

## 2. КАРТОЧКА КАК ЦЕЛОЕ

### Связность
Качественный контур связан образцово. Один сквозной тезис проходит через все текстовые вкладки: бизнес-модель («маржа уходит вверх по группе, амортизация съедает EBITDA») → управление («related-party потоки вверх, free float ~5%, squeeze-out риск») → гео («маржу распределяют санкционные совладельцы, держатель префа не влияет») → макро («попутный ветер слабо доходит до прибыли, нерв — маржа»). Это редкая для карточки цельность: инвестор реально понимает, ЧТО за бумага и ПОЧЕМУ она дёшева.

### Критическое противоречие — финансы расходятся сами с собой
- `financials.json` (обновлён 13 июня) считает справедливую цену через `current_price_for_calc = 392.702 ₽` и `капитализация 39064 млн` — это цена и капа **обыкновенной MFGS**, не префа. Диапазон справедливой цены: **conservative 385 / base 558 / optimistic 878 ₽**. CAPM-таргет 489 ₽.
- `financials_summary.md` (31 мая) считает по **цене префа 244 ₽**, P/B 0,195, и даёт коридор **188–376 ₽, центр 251 ₽**.
- Это два РАЗНЫХ ответа на главный вопрос карточки. Витрина покажет либо «преф стоит 244, справедливо ~251» (summary), либо «справедливо 385–878» (json). Второе — оценка обыкновенной акции, выданная за оценку префа. Для бумаги, существующей только ради сравнения с обычкой, это разрушает доверие.
- json даже не упоминает 244 ₽ и 48,6%-дисконт префа — то есть пересчёт от 13 июня **выбросил всю преф-специфику**, ради которой карточка и заведена отдельно от MFGS.

### Противоречие по убытку (РСБУ vs МСФО)
- business_model.md: чистый убыток 2024 = **−13,25 млрд** (РСБУ).
- financials.json / summary: чистый убыток 2024 = **−5,76 млрд** (МСФО).
Расхождение ×2,3 честно отмечено в summary, но мини-P&L бизнес-модели даёт читателю −13,25 как «якорь», а финблок — −5,76 как основу всех расчётов. Две разные «правды» о ключевом числе на двух соседних вкладках без перекрёстной ссылки.

### Противоречие по долгу
- business_model.md и EV/EBITDA-расчёт: чистый долг ~0.
- financials.json data_flags: источник даёт **долг 101,8 млрд при ЧД 0 одновременно** — противоречие, ЧД принят 0 как judgement.
- Сам json честно пишет: «если реальный ЧД 101,8 млрд — EV/EBITDA-оценка уходит НИЖЕ НУЛЯ». То есть базовый метод оценки (EV/EBITDA = 731 ₽) стоит на допущении, которое при ином прочтении источника обнуляет результат. Это не дыра анализа (отмечено честно), но это значит, что половина valuation-методов держится на воздухе.

### Ценность вкладок
- Бизнес-модель, управление, гео — несут реальную ценность, не дублируют.
- Макро и рынок частично пересекаются (Urals, дисконт, НДПИ-нагрузка повторяются в трёх вкладках), но каждый раз под своим углом — терпимо.
- **Финансы** — самая важная вкладка для этой бумаги — сейчас наименее надёжна. Парадокс: вокруг неё всё построено, а она сломана.

## 3. ПО ВКЛАДКАМ

**Бизнес-модель** — сильная. Честно маркирует якоря vs ориентиры (курсив), прямо говорит «преф = менее выгодный инструмент, чем обычка». Слабость: мини-P&L даёт убыток −13,25 (РСБУ), не согласован с финблоком (−5,76 МСФО); EBITDA «~29» и маржа «~14%» поданы как ориентир, но финблок берёт 29 000 как точную базу нормализации — мягкое расхождение «оценка → точное число».

**Финансы** — критично. (а) json считает преф по цене обычки — главный дефект. (б) Два набора справедливых цен (json 385–878 vs summary 188–376). (в) EV/EBITDA и CAPM формально «ok», но: EV/EBITDA нормализован на 2 точках из 6 лет (2019 и 2024), CAPM — это просто «цена × (1+Ke)», то есть не оценка стоимости, а требование доходности (сам json это признаёт в caveats — тогда зачем выдавать 489 ₽ как fair_value?). (г) DCF честно insufficient_data (нет ОДДС) — нормально. (д) Баланс — только итоги, вся детализация null; tangible equity неизвестен, а оценка опирается на book value — то есть оцениваем по капиталу, структуру которого не видим.

**Рынок** — хорошая, зрелая. Доли ВИНК и доля СН-МНГ (~0,71% / ~1,33%) честно помечены как требующие сверки с годовым отчётом. Тезис «медленная потеря доли» обоснован факторами. Слабость: цена Urals здесь $67,85 (2024), а в макро- и гео-вкладках фигурирует Urals ~$87 и даже ~$112 (апрель 2026) — разные временные срезы цены нефти в одной карточке без единой оговорки «по состоянию на».

**Макро** — сильная, читаемая, с уровнями достоверности. Urals ~$87, ставка 14,5%, рубль 72–73 — внутренне согласована. Вопрос только к стыковке с рынком (там $65–67).

**Гео** — самая глубокая вкладка. Сценарии (эскалация/статус-кво/деэскалация), даты SDN, потолок $44,10, теневой флот — всё конкретно и привязано к механизму «как доходит до префа». Urals здесь ~$112 (апрель 2026) — снова иной срез цены, чем в рынке ($65–67) и макро ($87). Три вкладки — три цены нефти.

**Управление** — сильная, прямо стыкуется с финансами и гео (related-party вывод маржи, free float 5%, squeeze-out). Честно отмечает расхождение года последней выплаты по префу (2012 vs 2013).

## 4. ФАКТИЧЕСКИЕ ОШИБКИ/ПРОТИВОРЕЧИЯ
1. **Финансы (json vs summary):** справедливая цена 385–878 ₽ (json, по цене обычки 392,7) vs 188–376 ₽ (summary, по цене префа 244). Главный дефект.
2. **Финансы:** `financials.json` считает преф MFGSP по цене и капитализации обыкновенной MFGS — преф-специфики (244 ₽, дисконт 48,6%) в json нет вообще.
3. **Чистый убыток 2024:** −13,25 млрд (бизнес-модель, РСБУ) vs −5,76 млрд (финансы, МСФО) — ×2,3, без явной перекрёстной сшивки в бизнес-модели.
4. **Цена Urals:** ~$65–67 (рынок) vs ~$87 (макро) vs ~$112 (гео) — разные срезы без единой даты «as of».
5. **Чистый долг:** 0 (judgement) при долге 101,8 млрд в источнике; базовый EV/EBITDA-метод обнуляется при ином прочтении (честно отмечено, но остаётся скрытой бомбой под оценкой).
6. **CAPM 489 ₽** подан как метод справедливой цены, хотя сам caveat говорит «это якорь стоимости капитала, не самостоятельная оценка» — методологическая нестыковка.

## 5. ЧЕГО НЕ ХВАТАЕТ ДО ЭКСПЕРТНОСТИ
1. **Главное: пересчитать financials.json по цене ПРЕФА (244 ₽), а не обычки.** Вся valuation должна быть преф-специфичной (P/B 0,195, дисконт к обычке как отдельный блок), и сойтись с summary в один диапазон. Сейчас это две разные карточки под одним тикером.
2. **Свести убыток 2024 к одной базе** (выбрать МСФО или РСБУ как основную, вторую дать справочно) — и согласовать мини-P&L бизнес-модели с финблоком.
3. **Единый срез цены нефти** по всем вкладкам с явной датой; либо явно оговорить, что рынок описывает 2024-факт, а макро/гео — текущий момент 2026.
4. **Решить судьбу долга 101,8 млрд:** добыть первичку slavneft.ru или явно вынести в шапку оценки «при ЧД 101,8 EV-оценка отрицательна» как равноправный сценарий, а не как сноску.
5. **Убрать CAPM из fair_value_range** или переименовать в «требуемая доходность» — не смешивать с оценкой стоимости.
6. Явный мини-блок «преф vs обычка»: цена 244 vs 502, P/B 0,195 vs 0,40, почему дисконт 48,6% — это и есть главный вопрос держателя MFGSP, сейчас он живёт только в summary, а не в данных.

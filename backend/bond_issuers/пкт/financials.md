## Финансы эмитента: сможет ли ПКТ расплатиться

Главный вопрос — сможет ли ПКТ обслуживать долг по выпуску 002P-01 (7,5 млрд ₽, купон 15,99%, погашение 15.11.2029).

**Рейтинг *(факт)* — важное расхождение между агентствами:**
- Эксперт РА — рейтинг облигаций 002P-01 на уровне **ruAA-** (подтверждён 05.11.2025), но **20.03.2026 установлен статус «под наблюдением»** — это не понижение, а пауза перед пересмотром из-за смены собственника (выход «Росатома»); высокая вероятность рейтингового действия в течение ~3 месяцев.
- АКРА — присвоило выпуску **AA(RU)** со стабильным прогнозом (выше на ступень), приравняв рейтинг эмиссии к рейтингу Global Ports.
- НРА — AA по эмитенту.

**Долговая нагрузка *(оценка)*.** Свежих консолидированных метрик по самому ПКТ нет. По группе Global Ports на конец 2022: Чистый долг/EBITDA ~2,4x, покрытие процентов ~3,0x — умеренная нагрузка *(данные устаревшие)*. По самому ПКТ за 2025: выручка 6,2 млрд ₽ (+15%), чистая прибыль 1,41 млрд ₽ (+16%) — операционная динамика положительная *(факт)*. Свежих данных по Долг/EBITDA и ликвидности к погашению нет *(нет данных)*.

**Флаги *(суждение)*:** жёлтый — статус «под наблюдением» по версии Эксперт РА из-за смены акционера; до закрытия сделки и пересмотра рейтинга профиль не определён окончательно. При этом сам бизнес прибылен, нагрузка группы исторически умеренная.

*Источники: raexpert.ru (20.03.2026); acra-ratings.ru (релиз 6306); cbonds; rusbonds. По состоянию на 10.06.2026. Профиль эмитента — общий для всех его облигационных выпусков.*

## Замечания по чек-листу

1. **(п.1) Двойной счёт внутри отчёта — канал A vs B, международный факторинг.** Потеря/невозможность роста международного факторинга заведена в оба канала. **Существенное.**

2. **(п.1/п.3, БЛОКЕР) Арифметика канала C не сходится однозначно.** «−8 п.п. (диапазон −7…−10)», но rationale даёт «отдельно ~2 п.п.» неясной аддитивности — либо недобор без объяснения, либо итог карточки 13+10=23% вместо заявленных ≈21%. ИНСТ уже держит ~2 п.п. за непрозрачность со скидкой — риск частичного дублирования с E15/S12. Тест: одна таблица «непрозрачность: ИНСТ X + ГЕО Y = Z, итог канала C = N». **Блокер.**

3. **(п.2/п.9, существенное) Канал A: подмена бенчмарка «индекс → сектор» + противоречивый санити-чек.** Дельта считается к сектору, а не к индексу как того требует C06 §2; «сохранение целого» как критерий валидности противоречит признанию, что прежний Ke был раздут. Тест: пересчитать дельту против индексной гео-экспозиции.

4. **(п.3) Пересечения с ИНСТ — в основном честно**, кроме стыка непрозрачности (см. замечание 2).

5. **(п.4/п.8, существенное) Канал C «якорится историческим спредом» — якоря фактически нет.** −8 п.п. = число, переданное ИНСТ, не независимо заякоренное. Тест: спред P/B санкционных/несанкционных банков до/после якорного события.

6. **(п.5, существенное) C08 без тестов величины для К3/К4 и без строки остатка.** CoR 9,53%/NPL 28% радикально выше сектора — часть «удара ставки» может быть идиосинкразией андеррайтинга (К4), а К2=70% завышен. Вывод «~5%» дан без [ОЦЕНКА, confidence, метод].

7. **(п.6) Эндогенность — чисто.**

8. **(п.7) Survivorship — практически чисто** (мелочь: сатурация без позитивного контроля событийными окнами). **Косметика.**

9. **(п.8, существенное) Ложная точность / ключевые числа без источника** (нормализованный убыток −124 млрд, CoR-удар +151 млрд, разовый доход +146 млрд, NPL 28%); `conclusion.range_low/high = null` — сценарная сборка C06 §4 не выполнена.

10. **(п.9, существенное) Якоря E-шкал: E1=4,5 противоречит собственному описанию** (адаптация пройдена → аргумент за ≤4, балл выше); **E13=2,5 не привязан к якорю** (Судариков не под санкциями, «зона внимания» не тянет на якорь-3).

11. **(п.10) Nirvana fallacy — чисто.**

12. **(п.11/п.12, существенное/граница) Проверка отрицательного факта по Сударикову** покрывает US+EU+UK, но источник проверки по EU/UK не приложен (честно помечено в data_flags — потому не блокер).

13. **(п.13) Свежесть — чисто.**

14. **(п.14) Пандитство — чисто** (конкурирующее объяснение ставка→CoR сделано главным).

15. **(п.15) Ретроспектива — чисто.**

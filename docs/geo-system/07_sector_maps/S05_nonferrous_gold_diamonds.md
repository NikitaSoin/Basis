# S05. Цветные металлы, золото, алмазы: секторная гео-карта

**Периметр:** Норникель, Русал, Полюс, АЛРОСА, En+.

## 1. Гео-профиль сектора
Сектор «глобальной незаменимости с исключениями»: E2 работает в обе стороны (K04 §3 — доля в мировом предложении = и уязвимость, и щит). **Норникель:** палладий/высокосортный никель — критичность для западных цепочек держала его ВНЕ блокировок дольше всех гигантов (LME-приёмка металла — живой индикатор); уязвимость — G13-спрос (EV-цикл убил палладиевую премию — глобальный, не гео-фактор!). **Русал:** K04-наследие (E13 вылечен, E1≈3 остаточный) + глиноземная логистика (Николаевский потерян-2022, австралийские запреты) + LME/US-пошлинные волны + C09-бенефициар (Ормуз-2026: выпадение Залива = ценовой ветер в паруса). **Полюс:** золото = антисанкционный товар по природе (внебиржевой, дробимый, восточносбытовой) — E-профиль лучший в секторе; риск смещён в ИНСТ-слой (аппетит государства к золотым активам — фон K31). **АЛРОСА:** G7-стайл удар по товару (санкции G7 на российские алмазы + трассировка-2024) при индийском огранном плече — эмпирика энфорсмент-эрозии на неучётном товаре.

## 2. Трансмиссия
G13 → цены (доминанта: EV/электроника/ювелирный циклы — декомпозиция обязательна); G3/G4 → точечно (пошлины, LME-правила, алмазная трассировка); C09 → шоковые премии (Залив-алюминий; Индонезия-никель как структурный медведь); G12 → экспортные пошлины «курсовые» (2023–24) и НДПИ-качели (золото под фискальным взглядом при рекордной цене — «где толсто»).

## 3. Сценарная карта
S1: умеренная пис-бета (санкционный слой тоньше нефтегазового); Русал/АЛРОСА — прямые бенефициары снятия товарных мер; крепкий рубль — общий встречный ветер. S2/S3: G13-циклы правят; внутрисекторная жизнь — от мировых балансов, не от фронта. S4: золото — единственный хедж сектора (Полюс = квази-защитный при слабом рубле+страхе); остальные — минус через глобальный risk-off.

## 4. Мониторинг
LME-статусы российского металла и склады, алмазная трассировка G7 (энфорсмент-волны), C09-строки (Залив/Индонезия/Чили по меди), EV-статистика (палладий), НДПИ/пошлинные строки бюджета, золотые интервенции/закупки ЦБ и Китая (спрос-фон).

## 5. Анти-паттерны / red-team флаги
Палладиевое падение «из-за санкций» (это EV — G13; контрольная группа: цены глобальны). Русал по памяти-2018 (E13 вылечен — nirvana-симметрия: не каркать на решённое, K12-дисциплина). Золото «вне политики» (фискально-институциональный слой реален). Алмазный «крах от санкций» без учёта лабораторного фактора (двойная декомпозиция: санкции × synthetic-цикл). Ормузская премия как постоянная (C09: режимы временны — снимать при деэскалации).

## 6. Дифференциация
Норникель — G13-циклическая машина с санкционным щитом незаменимости (щит ≤0,5 балла — K18 §6.2!); Русал — C09-чувствительный + остаточно-санкционный: двухдвигательный; Полюс — защитный профиль с ИНСТ-хвостом; АЛРОСА — товар под прямыми мерами: чистый полигон энфорсмент-эрозии (G5-методика на неэнергетическом товаре); En+ — Русал×энергетика: холдинговый дисконт поверх.

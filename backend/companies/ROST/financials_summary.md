# Финансы и оценка — Росинтер Ресторантс Холдинг (ROST)

*МСФО с IFRS 16, млн руб. Период 2021–2024. Цена 81.538 руб. на 2026-06-13. Данные: пресс-релиз 12m2024 + smart-lab. Методика v2.*

---

## 1. Динамика выручки и прибыли

Выручка растёт устойчиво: 6210 → 6280 → 7166 → 7766 млн руб (+8.4% в 2024). Двузначный рост 2023 года (+14.1%) связан с восстановлением посещаемости и открытием новых точек.

EBITDA при этом сжимается в марже: 38.2% (2021) → 43.5% (2022) → 31.8% (2023) → 30.6% (2024). Маржа упала из-за роста финансовых затрат (с 850 до 1339 млн за 2024 год на фоне высокой ставки) и сжатия валовой маржи (22.75% в 2024 против 26% в 2022).

**Разрыв reported/adjusted принципиален:**

- 2023: reported ЧП = 111.8 млн. После ETR-нормализации (+12.8) и исключения курсового убытка (+70.9) — **adj ЧП = 195.5 млн**.
- 2024: reported ЧП = 1.5 млн (за счёт признания ОНА при убытке до налога -45.5 млн). После ETR-нормализации (-35.6), возврата курсового убытка (+22.6) и обесценения (+54.8) — **adj ЧП = 43.3 млн**. Бизнес в 2024 году едва прибылен даже в нормализованном виде.

## 2. Баланс: отрицательный капитал, рост аренды

Собственный капитал устойчиво отрицательный: -3000 (2021) → -2533 млн (2024). Причина — накопленный убыток -6787 млн, лишь частично перекрытый акционерным капиталом и APIC. **P/B и дивидендная модель неприменимы**: дивиденды юридически невозможны при отрицательных чистых активах.

Главный структурный риск: аренда IFRS16 удвоилась за год — с 3.0 до 4.98 млрд руб. Это отражает агрессивное открытие ресторанов и/или переоформление договоров. При включении аренды в net_debt (9.88 млрд) коэффициент ND/EBITDA = 4.15x — опасно высокий для убыточного бизнеса. Текущая ликвидность: current ratio = 0.19 — критически низкая.

## 3. Качество денежного потока

CFO стабилен: 516 → 1100 → 1070 → 983 млн (2021-2024). Стандартный FCF (без аренды) выглядит приемлемо: 262 → 961 → 860 → 453 млн. Но это иллюзия: аренда IFRS16 вынесена в CFF (арендные платежи -1057/-1066 млн в год), не в CFO.

**FCF с арендными платежами: 2023 = -197 млн, 2024 = -322 млн (нормализованный capex).** Компания живёт за счёт займов от связанных лиц (1036 млн в 2024 при погашении 41 млн) — фактически финансируется основателем.

Капex 2024 = 530 млн (аномалия 2.5x к медиане 2021-2023 = 210 млн) — нормализован до 239 млн для FCF_normalized.

## 6. Справедливая стоимость

Методы применены по маршрутизации для ритейл/общепит с отрицательным капиталом:

| Метод | Статус | fair value |
|---|---|---|
| DCF (Гордон) | insufficient_data | equity отрицательна (-2047 млн) |
| EV/EBITDA forward (base: 4.33x, full ND) | ok | ~130 руб |
| EV/EBITDA forward (fin ND, 3.5x) | ok | ~272 руб |
| Relative peers | ok | подтверждает переоценённость при full ND |
| Historical P/E | insufficient_data | только 1 год; нет forward EPS |
| Historical P/B | not_applicable | капитал отрицательный |
| Dividend | not_applicable | дивиденды юридически невозможны |
| CAPM | ok | нет позитивного таргета при нулевом FCF/дивиденде |

**Ключевая неопределённость — трактовка IFRS16 аренды:**
- Если аренда = операционная (не включать в net_debt): EV/EBITDA 3.5x → 272 руб, 4.5x → 436 руб
- Если аренда = долг (включать в net_debt): только при 4.5x → 130 руб; ниже → equity отрицательна

**Коридор справедливой стоимости: 80–200 руб** (консервативный 80, база 130, оптимистичный 200 руб). Текущая цена 81.5 руб — у нижней границы коридора.

**Предупреждение**: оценка крайне неустойчива к допущениям. Текущая цена акции близка к «нулю» при консервативном сценарии и в 2-3 раза ниже — при оптимистичном. Это типично для высоколевериджированного убыточного бизнеса, где рыночная капитализация (1.33 млрд) мала относительно долга (4.9–9.9 млрд).

### (3) Противоречия (выявлены и устранены)
- **Главное:** governance.json утверждал, что «выплаты направляются на префы, обычка без дивиденда». ВЕБ опроверг: дивиденды НЕ платятся НИ по одному классу; последняя фактическая выплата по префу — 2011 г.; ГОСА 2021-2025 ежегодно отказывало по ОБОИМ классам (ст.43 ФЗ об АО — отрицательный капитал до 2025). Исправлено в governance.json (policy_text, share_classes).
- **Оценка велась как у обычки** (P/E forward 19,3 ₽, CAPM 30,7 ₽ на цене/числе обычки). Якоря исправлены на преф (цена 11,25 ₽).

### (3-bis) МЕТОД-ФИКС (2026-06-17) — УСТРАНЁН АРТЕФАКТ дивидендной модели
- **Артефакт:** прежний fair_value_range = charter DPS 2,14 ₽ ÷ плоская требуемая доходность 30/25/20% = 7,1 / 8,5 / 10,7 ₽. Неверно дважды: (а) дивиденд 2,14 ₽ по уставу **ФАКТИЧЕСКИ НЕ ПЛАТИТСЯ с 2011** — fair на невыплачиваемом потоке невозможен; (б) плоская завышенная ставка 25-30% давала искусственную цену.
- **Исправлено:** коридор переведён на **ИНТРИНСИК-методы** (class-agnostic per-share по экономике эмитента): historical P/E (база) + CAPM (справочный потолок). current_price = цена ПРЕФА 11,25 ₽; апсайд считается живьём, он сам отражает дисконт префа к обычке и governance/неликвидность. Метод `dividend` → **not_applicable** («не платится с 2011»). Дивиденд = 0 — отдельный качественный сигнал (н/п), не цена.

### (4) Синтез-вердикт по ПРЕФУ (обновлён)
- **Метод оценки:** интринсик per-share. Коридор: **16,6 ₽ (P/E backward, conservative) — 18,0 ₽ (база) — 22,0 ₽ (к CAPM-краю с дисконтом за ненадёжную бету)**. Сырой CAPM 30,7 ₽ — потолок, бета 0,8 для 3-го эшелона ненадёжна.
- **Цена префа 11,25 ₽ → апсайд к базе ≈ +60%, к conservative ≈ +48%.** Глубокий дисконт отражает бездивидендность + 3-й эшелон + LOW data quality + governance/related-party риск.
- **Спред к обычке:** преф 11,25 ₽ vs обычка 16,89 ₽ = **−33%** (преф дешевле).
- **Дивиденд (ФАКТ, н/п):** реализованная доходность **0%** — посл. выплата 2011 г.; ГОСА 2021-2025 отказывало (ст.43 ФЗ об АО, отрицательный капитал до 2025). Уставное право (10% ЧП / 25% капитала ≈ 2,14 ₽, потенц. 19%) — справочно. Исторической нормы выплат нет → доп-сигнал дивдоходности н/п.
- **Развилка ГОСА 2026:** объявят дивиденд по уставу → реальная доходность ~19%, апсайд-катализатор; откажут 15-й год → преф остаётся бездивидендной бумагой. Это качественный сигнал, не оценка.
- **Вывод:** бумага торгуется глубоко ниже интринсика экономики (P/E 2,1x), но дисконт оправдан 14-летним рекордом нулевых выплат + high governance/related-party + неликвидностью. Ставка на возобновление выплат. Достоверность LOW.

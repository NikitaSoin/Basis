# K12. 470-ФЗ и X5 (2023–2025): принудительная редомициляция — позитивный контроль системы

**Статус:** развёрнутый кейс. Обязательный «не-каркать»-тест бэктеста (V01 №8).

## 1. Паспорт
Тип: снятие периметрального риска государственным принуждением. Период: 08.2023 (закон) – 07.2025 (спецдивиденд). Оси: E9 (5→1 — единственный в базе пример полного схлопывания балла), C03 (навес и его абсорбция), T15 (внутренний спрос). Роли: позитивный контроль против деформации рамки; калибровка абсорбции навесов.

## 2. Хронология [ФАКТ]
04.08.2023 — закон 470-ФЗ об «экономически значимых организациях» (ЭЗО): суд может приостановить корпоративные права иностранной холдинговой структуры над российской операционной и принудительно «раздать» акции конечным бенефициарам-резидентам. Март–май 2024 — X5 первый флагманский кейс: Минпромторг → суд → права X5 Retail Group N.V. (Нидерланды) на «Корпоративный центр ИКС 5» приостановлены; торги расписками на Мосбирже остановлены (апрель 2024) на время трансформации. Резиденты-держатели получают акции ПАО напрямую; нерезидентам N.V. — механизм компенсации/выкупа с дисконтом. 09.01.2025 — старт торгов акциями ПАО: против страхов «навесного обвала» бумага абсорбирует предложение при высоких оборотах — внутренний спрос выкупает (компания без долга, лидер сектора, понятный кейс). 2025 — **спецдивиденд 648 ₽/акция** (выплачен летом) из накопленной за «беспереездные» годы подушки — материальная компенсация ожидания, гасившая остаток навеса.

## 3. Механика по каналам
E9-хирургия: голландский холдинг + расписки + «недружественная» прописка (максимум периметрального риска: заморозки, невыплаты, делистинговые угрозы) заменены прямым ПАО-владением — риск не «переоценён», а ЛИКВИДИРОВАН структурно. Цена решения — принуждение: права акционеров N.V. ограничены внешним для них законом (для западного наблюдателя — экспроприационный элемент; двойная оптика фиксируется честно: то, что для внутреннего миноритария — спасение, для внешнего — прецедент изъятия; второй слой скорится ИНСТ-системой). Навесная механика по C03: предложение (раздача + дисконтные покупатели внешнего контура) встретило спрос без иностранного бида — эмпирическое доказательство тезиса «внутренний рынок способен абсорбировать при сильном кейсе» (T15 §4).

## 4. Рыночная динамика
Старт-2025 без провала, быстрый набор капитализации, дивидендная переоценка — редкий в базе кейс, где событие гео-класса дало устойчивый ПЛЮС. Дифференциал к непереехавшим аналогам того периода — измеримая премия за решённый периметр [ОЦЕНКА, проверяемо].

## 5. Развязка
Полная и образцовая; шаблон применён дальше (последующие принуждения/переезды по ЭЗО-линии). Для рынка закрыт класс страхов «вечного нидерландского зависания» — по крайней мере для системно значимых.

## 6. Уроки для шкал
1. **Риск умеет сниматься — и система обязана это видеть:** после завершения трансформации E9=1–1,5; удержание высокого балла «по привычке» = деформация рамки (nirvana-предохранитель), бэктест это карает независимо от прочих результатов.
2. **Абсорбция навеса — калиброванный прецедент:** параметры (free-float, дисконтные держатели, подушка выплат, качество кейса) → шаблон прогноза для всех будущих разблокировок (C03 §3 наследует).
3. **«Подушка невыплат» = встроенный компенсатор:** накопленный за блокаду кэш — предсказуемый амортизатор навеса; проверять у каждого кандидата на переезд.
4. **Двойная оптика принуждения:** снятие риска для внутреннего контура ≠ снижение странового риска для внешнего — слои не смешивать (граница с ИНСТ).

## 7. Ex-ante тест
Двухточечный: (T1=запуск процедуры, 2024) — прогноз «E9 схлопнется; навес абсорбируем при [параметры]; ожидаемая премия к непереехавшим» против консенсусных страхов обвала; (T2=09.01.2025) — тест «не-каркать»: агент, выдавший высокий гео-риск ПОСЛЕ снятия причины, проваливает V01 §4 formально.

## 8. Перенос
На всю очередь периметральных историй (Ozon и далее): чек-лист «структура → процедура → навес-параметры → подушка»; на S1-разблокировки: X5 — нижняя граница оптимизма (сильнейший кейс), калибровать вниз от неё, не вверх.

## 9. Источники и метки
470-ФЗ, судебные акты и раскрытия X5 2024–25, параметры старта торгов и дивиденда (биржа/эмитент). Дифференциальные премии — [ОЦЕНКА], метод указан.

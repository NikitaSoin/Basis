### (3) Противоречия
- **НОВОЕ (факт 2026-06): Q1 2026 ЧП = 436.6 млн (-52.3% г/г)** из-за убытков нового банка КИБ «Евроальянс» (стадия запуска). Консенсус NP 2026 = 5127 млн (на котором стоит P/E base 256 руб) теперь ВЫГЛЯДИТ ОПТИМИСТИЧНЫМ → P/E-оценка завышена. Реалистичный NP 2026 ~3500-4500 млн. forecast.note обновлён.
- P/E base 256 руб (P/E 5x × forward EPS 51.27) расходится с P/BV×ROE 150 и CAPM 152 на >30%: P/E предполагает переоценку сектора + полный консенсус, оба под сомнением. Консервативный anchor — P/BV×ROE.
- Баланс не закрывается (data_quality LOW) → BVPS 145 и ROE — APPROX; реальные могут отличаться ±5-15%.

### (4) Синтез-вердикт (по P/BV×ROE + дивиденд)
Основной метод для МФО — **P/BV×ROE (Гордон):** justified P/B = (ROE 28% − g 2.5%)/(Ke 27.2% − g 2.5%) = 1.03x → fair ≈ **150 руб** (cons. ROE 24% → 127; opt. ROE 32% → 174). ROE ~28% лишь чуть выше Ke 27.2% → P/B ≈ 1.0 → бизнес создаёт стоимость с минимальной маржой. Текущий рыночный P/B = 1.02x — **акция оценена примерно справедливо**. Дивидендная проверка (DDM): рыночная implied yield 24% лежит между base (22%) и Ke (27.2%) → рынок уже закладывает регуляторный и концентрационный риск. **Вердикт: «справедливо оценена при текущих ставках», коридор 127–160 руб (база), верх 256 руб только при реализации консенсуса (теперь под вопросом из-за Q1-2026 банковского drag).** Ключевой катализатор роста — снижение КС ЦБ (↓Ke → ↑justified P/B и DDM); ключевой downside — регуляторное сжатие ROE<24% → fair < BVPS 145 + продолжение убытков банка Евроальянс.

## ВЕРДИКТ {light:gray}
ИА РТБ-1 (RU000A10EXM7) — облигация с ипотечным покрытием SPV-агента (структурный продукт, размещение 29.04.2026): на бирже 0 сделок, YTM/дюрация MOEX = 0 — рыночная доходность недостоверна, светофор-премия не строится. Кредит здесь определяется НЕ дефолтом SPV, а качеством пула закладных, старшинством транша и наличием поручителя/выкупа неприемлемых закладных оригинатором; по публичным данным структура (оригинатор, рейтинг выпуска, размер мл./стр. траншей) НЕ раскрыта прозрачно — поэтому gray, а не зелёный/жёлтый.

## АРИФМЕТИКА
- Фактический спред: **не строится** — на TQCB NUMTRADES=0, VALTODAY=0, YIELD=0, DURATION=0 (MOEX, 14.06.2026): нет цены сделки → «доходность за риск» рассчитать нельзя {факт}
- Оценочная YTM (режим A, флоатер): купон **15,8% год, ежемесячно, 24,67 ₽** на старте; MOEX помечает «Флоатер» — будущая ставка пересматривается. При прогнозе ЦБ средняя КС 14–14,5% в 2026 и 8–10% в 2027 (ОНЕГДКП, базовый сценарий) ожидаемый купон по траектории идёт ВНИЗ → оценочная YTM при покупке у номинала ≈ 13–15% с понижением; пометка: оценочная, от прогноза ЦБ {оценка}
- Требуемый спред: **не сопоставляется с G-кривой ОФЗ** — для ИЦБ кредитный риск замещён обеспечением (пул + поручитель), а реальная дюрация диктуется prepayment (досрочкой), не датой 2062. Сравнение с медианой рейтинговой группы неинформативно при неторгуемости {суждение}
- Премия: **gray** — неликвид + непрозрачная структура; светофор по премии неприменим

## ГЛАВНЫЕ АРГУМЕНТЫ
- **0 сделок на бирже, YIELD/DURATION = 0 (MOEX TQCB, 14.06.2026)** {факт} — бумага фактически неторгуемая (только что размещена 29.04.2026, ИЦБ часто оседают у институционалов «до погашения») → любая «доходность» бумажная, выйти по рыночной цене при новостях нельзя.
- **Эмитент — ипотечный SPV (ООО «Ипотечный агент РТБ-1»), деятельность по 152-ФЗ ограничена держанием пула закладных и обслуживанием долга по ИЦБ** {факт} — кредит держателя обеспечен залогом пула ипотек, а не балансом операционной компании; дефолт SPV как такового нерелевантен — риск в КАЧЕСТВЕ ПУЛА и старшинстве транша.
- **Оригинатор и рейтинг ВЫПУСКА в публичных источниках не подтверждены однозначно** {суждение} — «РТБ» предположительно отсылает к банку-оригинатору (вероятно Росбанк/иной банк-продавец закладных), но без раскрытия структуры (поручитель ДОМ.РФ? subordination? размер junior-транша?) кредитное качество транша оценить нельзя → консервативно gray.
- **Главные риски структурного продукта — prepayment и качество пула, не «дефолт эмитента»** {суждение} — досрочные погашения ипотек укорачивают бумагу непредсказуемо (реинвест-риск при падающей ставке), а ухудшение пула бьёт по младшим траншам в первую очередь; старший транш с поручителем ДОМ.РФ был бы AAA-квазириском, но это здесь НЕ подтверждено.

## ПРОВЕРКА ПОТЕРЯМИ
Для ИЦБ якорь PD×LGD считается не по SPV, а по ПУЛУ: историческая дефолтность качественной российской ипотеки низкая (~0,5–1,5%/год), LGD по обеспеченному жильём кредиту низкий (залог недвижимости, обычно 20–40% при реализации). Для СТАРШЕГО транша с поручительством ДОМ.РФ ожидаемые потери близки к нулю (квази-AAA). НО: без подтверждения, что анализируемая бумага — именно старший транш с поручителем (а не mezzanine/junior), нижнюю границу потерь зафиксировать нельзя. Якорь не закрывается из-за непрозрачности → консервативно gray. {оценка}

## БЛОК A — Платёжеспособность {score:2}
Для SPV-секьюритизации блок A = структура обеспечения и старшинство, а не корпфин.
- **A1 Источник платежей.** Денежный поток по ИЦБ = платежи заёмщиков по пулу закладных (ипотека), проходящие через SPV (pass-through), без собственного бизнеса агента {факт} — устойчивость потока = качество пула, а не баланс эмитента.
- **A2 Обеспечение и старшинство транша.** По 152-ФЗ требование держателя обеспечено залогом ипотечного покрытия; в типовой сделке есть старший (А) и младший (Б) транши, убытки пула поглощает младший первым {факт} — старший транш надёжен, младший — рисковый; конкретный класс этой бумаги по публичным данным НЕ подтверждён {суждение}.
- **A3 Поручитель / выкуп неприемлемых закладных.** В ИЦБ ДОМ.РФ поручитель — АО «ДОМ.РФ» (квази-госкредит), оригинатор выкупает несоответствующие закладные; для ИА РТБ-1 наличие поручителя ДОМ.РФ НЕ подтверждено {суждение} — если поручителя нет, риск выше типового ИЦБ ДОМ.РФ.
- **A4 Качество денежного потока.** Зависит от просрочки и prepayment по пулу; данных по конкретному пулу нет {суждение} — нет данных.
- **A5 Анти-рисование.** Для SPV неприменимо в классическом виде (нет «рисованного капитала»); риск-эквивалент — непрозрачность структуры и качества пула {суждение} — здесь это и есть ограничение: непрозрачно → блок не лучше середины.

## БЛОК B — Бизнес {score:2}
- **Модель.** SPV не ведёт бизнес — это «обёртка» для пула ипотек; источник денег — платежи розничных ипотечных заёмщиков {факт} — диверсифицированный пул из тысяч кредитов снижает идиосинкразический риск.
- **Позиция / концентрация.** Концентрация на одном классе актива (жилая ипотека РФ) и, вероятно, на одном банке-оригинаторе (обслуживание пула) {суждение} — сервисный риск: если оригинатор-сервисер уйдёт, потребуется backup-servicer.
- **Цикличность / на что заняли.** Привлечённое = рефинансирование ипотечного портфеля банка-оригинатора (классическая секьюритизация для разгрузки капитала) — нормальная цель, не «затыкание дыр» {суждение}.

## БЛОК C — Собственники и репутация {score:3}
- **Владельцы.** SPV по 152-ФЗ — техническая компания со специальной правоспособностью; бенефициар экономики — банк-оригинатор (предположительно из группы «РТБ», вероятно Росбанк/иной банк) {суждение} — конечный оригинатор публично не подтверждён → пометка.
- **Репутация / связанные стороны.** Для SPV каналы вывода активов структурно ограничены законом (деятельность только по сделке) {факт} — это плюс к C.
- **Прозрачность.** Низкая по публичным открытым источникам: рейтинг выпуска, структура траншей, поручитель не найдены в открытом доступе на дату анализа {суждение} — главный минус, тянет блок к 3.
- **История долга.** Нет (свежий выпуск 2026) {факт} — нет данных по обслуживанию.

## БЛОК D — Макро и отрасль {score:3}
- **Ключевая ставка (двойной канал prepayment).** Прогноз ЦБ: средняя КС 14–14,5% (2026) → 8–10% (2027), снижение {факт} — при падении ставки заёмщики рефинансируют ипотеку → ускорение prepayment → бумага гасится быстрее ожидаемого, реинвест-риск держателя (деньги вернут раньше, размещать под более низкую ставку) {суждение}.

- **Регулирование ипотеки.** Сворачивание льготных программ и ужесточение ЦБ по первичке/закредитованности заёмщиков влияет на качество новых пулов; на УЖЕ собранный пул влияет слабее {суждение}.

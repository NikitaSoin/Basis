# Аудит карточки — АКБ «Авангард» (AVAN)

## 1. ВЕРДИКТ ЦЕЛОГО
**4/10 — качественные блоки (гео, управление, рынки) сильные и честные, но финансовое ядро построено на реконструированных и взаимно противоречивых числах, а ключевые показатели (цена, прибыль, дивиденд, год) разъезжаются между вкладками настолько, что инвест-портрет рассыпается.**

## 2. КАРТОЧКА КАК ЦЕЛОЕ

**Главная системная проблема — РАССИНХРОН ОПОРНЫХ ЧИСЕЛ.** Это карточка, где одни и те же базовые величины в разных вкладках разные. Инвестор не может доверять ни одному числу, потому что соседняя вкладка его опровергает:

- **Цена акции** нигде не зафиксирована единообразно: financials.json meta.last_price = **null**, в summary текст «текущая цена 600 руб», метод PBV cross-check «текущая цена 625,52 ₽», CAPM «625,52 ₽», governance вообще без цены. Три значения (600 / 625,52 / null) в одном финансовом блоке.
- **Чистая прибыль и годы съехали.** financials.json: net_profit = [null, 0.281, 5.04, 5.85, null] для лет [2021–2025] — то есть **5,85 млрд приписана 2024 году**, а 2025 = null. Но business_model и summary говорят: ЧП **2024 = 5,04**, **2025 = 5,85**. То есть json сдвинут на год относительно текста. Прямое следствие: EPS, ROE, P/E в json посчитаны на «прибыли 2024 = 5,85», а это на самом деле 2025. summary честно признаёт «таблица Smart-Lab имела сдвиг столбцов на ~2 года» — но сдвиг не вычищен, он живёт в итоговых мультипликаторах.
- **EPS:** summary «72,5», json eps[3]=72,49 (приписан 2024), business_model P/E считает «600/72,5». При этом ЧП 5,85/80,7 млн = 72,5 — а это прибыль 2025, не 2024.
- **ROE:** business_model ~41%, summary ~43,5%, json roe_pct = 43,83 (за 2023!). Три значения, разнесены по разным годам.
- **Капитал:** business_model «11,5 (2023) и СК 2025 = 11,5 / прирост +9,5%», summary «РСБУ май-2026 ≈ 15,41 млрд», json total_equity 2023 = 11,5, 2024–2025 = null. BVPS 142,5 (из капитала-2023) используется как «last_bvps» в оценке 2026 — книга трёхлетней давности.

**Связность между качественными вкладками — наоборот, хорошая.** Гео, макро, governance и рынки согласованно ведут одну линию: внутренний рублёвый банк под санкциями, ядро дохода — комиссии (зарплатные/эквайринг), главная уязвимость — отток депозитов и спроса. Санкции (SDN 23.02.2024, ЕС 20-й пакет 14.05.2026) синхронны во всех вкладках. Это сильная сторона.

**Противоречие оценки и рынка не разрешено, а просто названо «аномалией».** Фундаментальный коридор 178–311 ₽ против рыночных ~600–625 ₽ — методы дают −50% downside, и вывод фактически «рынок неправ либо неликвиден». Для инвестора это не ответ. При free-float 0,3% рыночная цена действительно может быть фикцией — но тогда честнее сказать «оценка невозможна», чем выдавать коридор с точностью до десятых (178,2 ₽).

**Дивиденд — противоречие политики и метода.** governance: формализованной дивполитики НЕТ, выплаты — воля владельца. financials DDM при этом строит вечную модель Гордона с g=2% на «форвардном DPS 68,16» — то есть моделирует как стабильный поток то, что governance прямо называет нестабильным и негарантированным. Это методологический конфликт между вкладками.

**Ценность вкладок.** Все нужны, дублей нет. Но financials в текущем виде скорее вредит доверию, чем помогает — из-за съехавших чисел. Governance и гео — лучшие вкладки карточки.

## 3. ПО ВКЛАДКАМ

**Управление — 8/10.** Честно и предметно: «акция одного владельца», squeeze-out risk (ст. 84.8), связанные стороны через ФПГ, прецедент ЦБ-2018 (ограничение права голоса). Прямо назван weak. Это сильнейшая вкладка. Минус — DPS-числа (104,08 за 2023; 68,16 на 16.12.2024) переходят в финансовую оценку без сверки.

**Гео — 8/10.** Глубоко, нетривиально: разделение «сам факт санкций» vs «вторичные санкции на посредников» (60–80% отказов), буфер через НСПК/СБП, сценарии. Хорошо.

**Рынки — 7/10.** Грамотно: главный рынок + два под-рынка (зарплатные/эквайринг), доля сектора ~0,05% честно помечена как расчёт, тренд концентрации, сдвиг эквайринга к ecom-банкам. Слабее по конкретике доли самого банка (не раскрыта).

**Макро — 7/10.** Корректный разбор «помогает по марже, мешает по объёмам», ядро — эквайринг/потребление. Свежие числа (ставка 14,5%, рубль 72,6). Связно с бизнес-моделью.

**Бизнес-модель — 6/10.** Содержательная структура (комиссионная модель, сегменты, цепочка стоимости), но почти всё помечено «оценка» (доли расходов, CIR 77%, сегменты 80/20). Мини-P&L даёт ЧП 2024=5,04 / 2025=5,85 — и это РАСХОДИТСЯ с json (где 5,85 стоит на 2024). Депозиты: «упали на 29% за 2024 (115,6→82,1)» — но 115,6 это 2023, 82,1 это 2024; формулировка путает годы.

**Финансы — 3/10.** Самое слабое звено. Честно предупреждает «качество НИЗКОЕ, сдвиг столбцов, многое реконструировано» — но это предупреждение не оправдывает того, что итоговые мультипликаторы посчитаны на съехавших данных. Половина bank_pnl и bank_balance — null. NIM, CoR, CIR, Н1 в bank_metrics = null (хотя summary называет Н1=17,70% и CIR=77% — числа есть в тексте, но не заведены в json). Методология банковской оценки (P/BV×ROE, DDM, не-DCF) выбрана правильно — но входы ненадёжны.

## 4. ФАКТИЧЕСКИЕ ОШИБКИ / ПРОТИВОРЕЧИЯ
1. **Цена:** null (meta) / 600 (summary) / 625,52 (методы) — три значения.
2. **ЧП по годам съехала:** json кладёт 5,85 на 2024, текст — на 2025; json 2025 = null.
3. **ROE:** 41% (БМ) / 43,5% (summary) / 43,83% за 2023 (json) — разнесено по годам и блокам.
4. **Капитал:** 11,5 (2023, json/БМ) vs 15,41 (РСБУ май-2026, summary) — BVPS 142,5 в оценке-2026 построен на капитале 2023.
5. **EPS/P/E** посчитаны на прибыли, приписанной не тому году.
6. **Дивполитика:** governance — формальной нет; DDM моделирует вечный поток Гордона — методологический конфликт.
7. **Н1, CIR, NIM, CoR** есть в тексте summary/БМ, но = null в json (нет переноса в данные).
8. Депозиты «−29% за 2024» — в тексте БМ привязка к годам спутана.

## 5. ЧЕГО НЕ ХВАТАЕТ ДО ЭКСПЕРТНОСТИ
- **Поднять PDF МСФО** (e-disclosure / avangard.ru) и пересобрать financials.json с нуля — без этого блок недостоверен. Это приоритет №1.
- **Устранить сдвиг годов** в net_profit/eps/roe/p-e и зафиксировать одну цену акции во всех вкладках на одну дату.
- **Завести в json уже известные числа** (Н1 17,70%, CIR 77%, депозиты, капитал РСБУ) — они есть в тексте, но null в данных.
- **Решить, честно ли давать коридор 178–311 ₽** при free-float 0,3%: либо явный вывод «рыночная цена неинформативна, оценка невозможна», либо обоснование, почему фундаментал важнее. Точность до десятых вводит в заблуждение.
- **Согласовать DDM с реальностью дивполитики** (её нет — значит, либо сценарный диапазон выплат, либо отказ от вечной модели Гордона).

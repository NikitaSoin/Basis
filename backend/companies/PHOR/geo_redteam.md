# Red-team: ГЕО-КОМПАНИЯ PHOR (черновик от 2026-07-12)

## Специальные проверки

**Дословность handoff:** все числа совпадают построчно с geo_barometer.json → handoff. Систематическая ошибка 12/12 прошлых отчётов НЕ воспроизвелась по числам. Формальный дефект — см. замечание 1.

**Честность «пересечений с ИНСТ нет»:** подтверждена — institutions.json реально исключил санкционную WACC-компоненту, пошлины/логистику, навес нерезидентов из публикуемых чисел, заявления geo.json сверены дословно. Остаточная серая зона — замечание 3.

## Замечания

1. **(п.11) Цитата handoff — выборка 6 из 13 G-баллов с перефразированными ярлыками, названа «verbatim»; sector_flags не процитированы.** **Критичность: косметика (числа верны).**

2. **(п.1) БЛОКЕР — двойной счёт навеса нерезидентов.** Канал C («~5-7% дисконта») против E14/S1 (навес уже внутри пары S1 ~−3…+3%) и против собственного nonresident_overhang («в S3/S4 навес заперт, не давит»). Постоянный дисконт назначен за риск, который сама логика отчёта связывает только с S1/S2 (21% вероятность). **Критичность: блокер.**

3. **(п.3/п.1) «Неликвидность free float» в канале C без владельца в таблице A02 §1**, и без разграничения с ИНСТ-компонентой «эрозия оферты СМЗ/ЮГК» — риск частичного наложения при сборке. **Критичность: существенное.**

4. **(п.5/п.8) БЛОКЕР — атрибуция в geo_summary.md противоречит числам geo.json.** JSON: 35/45/5/5/10=100%. MD: «~45% рубль, ~35% цикл, ~10% ставка, ~10-15% гео» = 100-105%, состав каналов другой. Публикуемый текст противоречит собственным числам. **Критичность: блокер.**

5. **(п.5) Тест величины C08 §2.3 не выполнен ни для одного канала** — доли не выведены, нет сравнения с нероссийским аналогом (Mosaic/Nutrien), остаток не назван. **Критичность: существенное.**

6. **(п.9) E11=3.0 при собственной оговорке «флаг, не конвертируется в балл»** — противоречие якорю C07, исказит bottom-up свод медиан. **Критичность: косметика (обязательна к починке).**

7. **(п.9) E12=3.0 при описании, дословно совпадающем с якорем 4** («повторяющийся фискальный донор») — смягчение без теста. **Критичность: существенное.**

8. **(п.9/п.14) Нарратив «war-хедж» противоречит собственной паре E14** (S1≈0, S4 минус — это демпфированный даунсайд, не хедж). Заголовок сильнее данных, может быть прочитан пользователем неверно. **Критичность: существенное.**

9. **(п.4/п.8) Имплайд-рынок подменён брокерским консенсусом без источника** вместо C06 §5-процедуры (пис-беты/implied_market барометра). **Критичность: существенное.**

10. **(п.8) Каналы C (5-7%) и A (+0,5…1,0пп) без исторического якоря спредов** — нарушение C06 §2C. **Критичность: существенное.**

11. **(п.8/п.11) Ложная точность** («~72,5% экспорт» с десятичной при собственном признании реконструкции разбивки), число без источника (Бразилия ~1,7 млн т). **Критичность: косметика.**

12. **(п.11) E13 type=«факт» содержит интерпретацию мотива** (должно быть оценка/суждение). Остальные ключевые числа сверены — сходятся. **Критичность: косметика.**

13. **(п.8/C06§4) conclusion.range=null без минимальной чувствительности** (±10пп P(ключевого сценария) — выполнимо без DCF). **Критичность: существенное.**

**Чисто:** п.2 (курс/ставка не задвоены), п.6 (обратная причинность), п.7 (позитивный контроль — сама PHOR как «тихий» исход), п.10 (nirvana), п.12 (нет данных ≠ нет проблемы), п.13 (свежесть барометра), п.15 (первый прогон, с оговоркой о Δ санкционной премии против прошлого ИНСТ — стоит зафиксировать при сборке).

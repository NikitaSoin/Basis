## Замечания

1. **(п. 11/8, существенное) Включение Севергрупп в списки ЕС/Британии не подтверждено источниками** — приведённый Forbes-источник про личные санкции Мордашова, не про юрлицо. Тест: проверить EU Consolidated List / UK Sanctions List на «Severgroup»; переформулировать при отсутствии.

2. **(п. 3, БЛОКЕР) Канал C ГЕО частично пересекается с каналом 3 ИНСТ по «неликвидности тонкого free float».** ИНСТ уже держит ~6 п.п. multiple discount за «низкий free float ~17,7%» (внутренне-структурный факт с 2019, до санкций), а ГЕО добавляет «гео-неликвидность тонкого free float» поверх — риск двойного счёта при сборке. Тест: ограничить канал C ГЕО строго замороженной нерезидентской частью + санкционной недоступностью бумаги, тонкий флоат как таковой — только модификатор чувствительности.

3. **(п. 8/4, существенное) Величина канала C не заякорена историческим спредом** — обоснование круговое (унаследовано от старого multiple discount ИНСТ, который сам не заякорен). Тест: спред мультипликатора LENT к X5/Магниту до/после 06.2022 и до/после редомициляции.

4. **(п. 6, существенное) Эндогенность: дисконт «Ленты» к аналогам существовал ДО санкций** (free float с оферты 2019, отсутствие дивидендов) — пре-санкционный дисконт не отделён от санкционного приращения.

5. **(п. 1/8, существенное) Канал A +0,5…+1,0 п.п. не проходит собственный тест отчёта.** Компания чисто рублёвая, экспозиция ниже индекса — аргумент за нулевую/отрицательную дельту, а не положительную. Граница A/C размыта.

6. **(п. 5, существенное) C08 без тестов величин по каналам** — доли назначены без метода (нет теста К1 с нероссийским аналогом, нет чувствительности К2 к ставке).

7. **(п. 9, косметика) E13 помечен «факт», хотя балл 3,5 — интерполяция**, то есть оценка.

8. **(п. 9, косметика) E14=2,0 при симметричной паре ΔS1/ΔS4** — скорее якорь 1-1,5, не 2.

9. **(п. 11, косметика/граница) Доля 81,5% повышена до чистого [ФАКТ] без первички** — institutions.json сам флагует «сверить с последним раскрытием МКПАО».

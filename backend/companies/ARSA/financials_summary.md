# Финансы и оценка — ПАО УК Арсагера (ARSA)
*Методика v2 | МСФО 2022-2023 (первичка) + агрегатор 2024-2025 | data_quality: medium*

---

## 1. Динамика выручки и прибыли

**Выручка (комиссионный доход от управления)** растёт устойчиво: 38.1 → 62.2 → 72.0 → 77.0 млн руб за 2022-2025, темп замедляется с +63% до +7%.

**ЧП reported** — крайне волатильна из-за переоценки собственного инвестиционного портфеля (~296 млн руб):
- 2022: -83 млн (убыток переоценки -103.8 млн нивелировал весь операционный доход)
- 2023: +109 млн (доход переоценки +98.9 млн взлетел ЧП)
- 2024: -8.4 млн (агрегатор)
- 2025: -1.0 млн (агрегатор)

**ЧП adj** (без нереализованных переоценок, на нормативном ETR 25%):
- 2022 adj: -0.95 млн (комиссионный бизнес в ноль без портфельной переоценки)
- 2023 adj: +26.98 млн (операционная прибыль 9.7 млн + дивиденды портфеля 14.5 млн, after-tax)

**Мост 2022:** reported -83.1 млн + 82.2 млн (убыток переоценки FVPL add back нетто) = adj -0.95 млн.
**Мост 2023:** reported +108.8 млн - 81.8 млн (доход переоценки убран + ETR нормализована до 25%) = adj +26.98 млн.

Reported цифры без нормализации бессмысленны — переоценка портфеля полностью их искажает.

## 2. Баланс: качество и долг

NAV (= total_equity) по МСФО: 193.6 → 302.5 → 232 → 213 млн руб (2022→2025). BVPS-ряд: 1.56 → 2.44 → 1.87 → **1.72 руб** (последний — якорь оценки). Падение 2024-2025 — переоценка собственного портфеля на слабом рынке + дивиденды.

Финансовые активы FVPL (инвестпортфель) = 296.5 млн руб (91.6% активов 2023) — полностью mark-to-market. NAV отражает рыночную стоимость портфеля.

Долг в 2022 — ноль; в 2023 появился 13.1 млн (незначительный, debt/equity 0.10). Ликвидность высокая (current ratio 25x). НМА и гудвил ничтожны — tangible_equity = total_equity.

## 4. Мультипликаторы

P/B текущий = **4.53x** (цена 7.79 / BVPS 1.72, последний NAV 2025) — аномально высокий при ROE adj 10.9% << Ke 27.6%. P/E adj (2023) = 35.8x. P/S = 13.5x. Все мультипликаторы аномально высоки — следствие неликвидного ценообразования (free float 7%), а не переоценки бизнеса.

Теоретический fair P/B по Гордону: (ROE 10.9% - g 5%) / (Ke 27.6% - g 5%) = 0.26x. Рыночный P/B в ~17 раз выше теории.

## 6. Справедливая стоимость

Маршрутизация: инвестиционная компания с портфелем FVPL → основной метод P/B к NAV. DCF not_applicable (нет ОДДС; FCFF неадекватен для портфельных компаний).

| Метод | Оценка | Достоверность |
|---|---|---|
| NAV / P/B=0.8 (conservative) | 1.38 руб | Средняя |
| NAV / P/B=1.0 (base) | 1.72 руб | Средняя |
| NAV / P/B=1.2 (optimistic) | 2.06 руб | Низкая |
| Дивидендная (Гордон DPS) | 0.54-0.61 руб | Низкая |
| P/E adj 7x x EPS_fwd | 1.13 руб | Низкая |
| CAPM 12m | 9.69 руб | Нерелевантна |

**Базу P/B обосновали (не с потолка):** ~98% активов — портфель FVPL по рыночной стоимости, NAV уже = справедливая стоимость активов → логичный якорь P/B≈1.0; премия к 1.0 только за операционную ренту УК, но при ROE<<Ke и убытках 2024-2025 её НЕТ → base=1.0, не 1.3.

**Итоговый коридор: 1.38 – 1.72 – 2.06 руб.** Якорь — последний NAV (BVPS 1.72). Ke 27.6% включает governance-премию +1.3пп (концентрация + неликвидность).

**Синтез-вердикт: ДОРОГО к фундаменту** — цена 7.79 руб = P/B 4.53x, в ~4.5× выше NAV. Но это артефакт неликвидности (free float 7%), а не сигнал к сделке: при таком free float «справедливая цена» = ориентир NAV, не таргет.

Текущая цена 7.79 руб примерно в 2.5x выше базовой оценки. Оценка имеет ограниченную практическую ценность: при free float 7% рыночная цена определяется случайными сделками. Реализация справедливой стоимости возможна только при ликвидации или кратном росте free float.

## Отчёт по прогону

- Период: 2022-2023 первичка МСФО + 2024-2025 агрегатор
- Нормализовано: 2022 убыток переоценки FVPL +82.2 млн нетто; 2023 доход переоценки -74.2 млн + ETR 18.6% → 25%
- Методы: NAV/P/B (основной), дивидендная, P/E adj (низкая достоверность)
- not_applicable: DCF (нет ОДДС, портфельный профиль), relative_peers (нет аналогов РФ)
- Расхождение методов >100% — нестабильность прибыли, неликвидность, отсутствие аналогов
- data_flags: ОДДС null; 2022 ЧП расчётная; 2024-2025 агрегатор; аномальный P/B
- peers.json: не обновлялся — нет публичных аналогов в секторе

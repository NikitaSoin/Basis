# Финансы и оценка — ПАО «Ашинский метзавод» (AMEZ)

**Качество данных: medium.** МСФО 2020–2025 (2022 отсутствует; 2025 — H1). Баланс частичный: доступен только капитал; активы, обязательства, долг — не раскрыты. Торги приостановлены с 17.03.2025 (squeeze-out «Урал-ВК»); цена 67.47 ₽ — по сделкам с практически нулевым free float, нерепрезентативна.

---

## 1. Динамика выручки и прибыли

Выручка прошла полный цикл: рост 63% в 2021 (40 млрд ₽) и 16% в 2023 (46.5 млрд), затем спад −20% в 2024 (37.3 млрд) и −27% в 2025 H1 (27.2 млрд). Валовая маржа сжалась с 31.5% в 2023 до 12.3% в 2025 — даунцикл спроса на спецсталь.

**Reported vs adjusted.** Ключевые расхождения:

- **2021:** reported ЧП 11.7 млрд → adj ЧП **7.7 млрд**. Причина: строка «Прочие операционные доходы» содержит разовый доход +4 626 млн (в 92× больше нормы). Природа не установлена из агрегата — certainty=judgement. Плюс нормализация ETR 21.7% → 25%.
- **2023:** reported ЧП 9.7 млрд → adj ЧП **9.6 млрд** (−123 млн). Сверхналог 486 млн включён в фактическую уплату; нормализация через устойчивую ETR 25%.
- **2024:** reported ЧП 4.8 млрд → adj ЧП **4.9 млрд** (+61 млн). Небольшая нормализация ETR 25.9% → 25%.
- **2025 H1:** reported = adj = **−1.4 млрд** (убыток genuine; корректировок нет).
- **2020:** незначительная ETR-коррекция (−43 млн).

EBITDA adj: пик 11.7 млрд (2023) → 5.1 млрд (2024) → −273 млн (2025 H1). В 2021 adj EBITDA 10.6 млрд (без разового дохода).

## 3. Денежный поток

FCF reported: 2.8 млрд (2020) → 6.9 млрд (2023) → **−6.0 млрд** (2024). CFO 2024 = −4.1 млрд аномален при adj ЧП 4.9 млрд: нормализован через WC-корректировку +9.9 млрд (вероятно дебиторка/авансы при падении выручки). FCF_normalized 2024 = **3.9 млрд**.

## 6. Справедливая стоимость

Маршрутизация: циклический металлург → EV/EBITDA + P/E(adj) + P/B; DCF осторожно (убыток 2025); дивидендная — not_applicable (спецдивиденд 2024 разовый).

| Метод | Результат | Примечание |
|-------|-----------|------------|
| DCF (Гордон, FCF1=3200 млн, Ke=23.6%, g=3%) | **31.2 ₽** | implied EV/EBITDA 3.0× |
| DCF conservative (FCF1=2500 млн) | **24.3 ₽** | |
| Исторический P/E adj (медиана 5.7× × EPS 9.77 ₽) | **55.2 ₽** | backward, нет forward EPS |
| Исторический P/B (среднее 2.77 × BVPS 19.76 ₽) | **54.7 ₽** | |
| Относительная peers (EV/EBITDA 5.5× × EBITDA 5100 млн) | **56.3 ₽** | суждение, peers.json не сформирован |
| CAPM 12м (Ke=23.6%) | **83.4 ₽** | механический, справочно |
| Дивидендная | not_applicable | нет устойчивого DPS |

**Расхождение DCF (31.2 ₽) vs мультипликаторные (54–56 ₽): 43% > 30%.** Причина: высокая Ke=23.6% при ОФЗ 14.6%. Мультипликаторные методы опираются на исторические периоды с более низкими ставками и прибыльным бизнесом.

**Коридор справедливой стоимости:** conservative **24 ₽** — base **55 ₽** — optimistic **56 ₽**.

Текущая цена 67.47 ₽ **выше базового коридора** по всем фундаментальным методам, кроме CAPM (механический). Апсайд сайт считает от живой цены.

**Предупреждение:** оценка носит академический характер. Торги приостановлены с 17.03.2025; миноритарии выкуплены по 71 ₽ (squeeze-out); делистинг вероятен. Практическое значение справедливой стоимости для инвестора — минимальное: возможности для входа/выхода по рыночной цене нет.

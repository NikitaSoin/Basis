### 🔴 Вердикт — риск существенно недоплачен; доходность нерелевантна как премия

YTM ~72% — не премия за риск, а ценник дистресса: рынок торгует бумагу по ~64,5% номинала, закладывая высокую вероятность дефолта *(стоп-правило №5 + преддефолтный рейтинг CCC)*. Спред «72% за зерно из Саратовской области» арифметически широкий, но это сигнал «риск не покрывается никакой ставкой», а не выгодная сделка.

> ⚠️ Расхождение: действующий рейтинг — **CCC(RU) от АКРА, развивающийся, под наблюдением, понижен 04.05.2026**. Часть агрегаторов показывает прежний BB-(RU). За недели бумага прошла BB- → CCC (из «ВДО с риском» в преддефолт).

### Арифметика (с оговоркой)

- **Фактический спред:** ~59 п.п. над ОФЗ; YTM ~72%, цена ~64,5% *(оценка, доходность бумажная)*.
- **Требуемый спред:** группа CCC/агро в базе практически пуста → деградация до «только рейтинговая группа CCC», точность ограничена; для CCC теоретически >25–35 п.п., но при PD группы CCC никакой конечный спред не «оплачивает» риск.
- **Премия:** формально +20…+30 п.п., но по стоп-правилу №5 это дисконт цены на грани recovery, не премия.

### Аргументы (факт → механизм → вывод)

1. **Блок A = 5:** выручка-2025 −57% (1 333 → 578 млн ₽), чистый убыток −614 млн ₽, Долг/FFO 2,9x → 9,8x *(факт, РСБУ, качество ограничено)* → неурожай обнулил поток, долг остался → платить из бизнеса нечем, ICR <1, проедает себя.
2. **Блок B/D:** монокультурная агрофирма (зерновые/масличные, ~110 тыс. га, Саратовская обл.) *(факт)* → один плохой сезон бьёт по всей выручке; нет диверсификации, восстановление зависит от ещё не собранного урожая → денежный поток контролирует погода, не менеджмент.
3. **Блок A3/E:** стена погашений ~200 млн ₽ по соседнему выпуску в июле 2026 без подтверждённого источника; по БО-04 погашение 11.11.2027 *(факт, АКРА)* → АКРА указывает на «отсутствие определённости с источниками погашения»; риск рефинансирования материализуется в ближайшие месяцы.
4. **Блок C/F:** крупный нерешённый судебный иск + «под наблюдением» + развивающийся прогноз *(факт)* → исход иска может подорвать ликвидность; рынок закладывает негатив (цена 64,5%).

### Проверка ожидаемыми потерями

Для CCC PD >20%, LGD ~70% → минимально ~14 п.п., при дистрессе CCC реалистичная PD выше. Даже спред 59 п.п. — не «заработок», а компенсация того, что значимая часть тела может не вернуться. Оба якоря: вопрос смещается с «какая доходность» на «сколько удастся вернуть» *(оценка/суждение)*.

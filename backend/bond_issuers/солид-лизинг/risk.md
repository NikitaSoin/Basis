## ВЕРДИКТ {light:orange}
Флоатер КС+4% торгуется по 88% — рынок требует за этот лизинговый ВДО без действующего рейтинга эффективный спред порядка +18–20 п.п. к КС; контрактные +4 п.п. риск не покрывают, дисконт цены — это и есть доплата рынка за риск, и она оправдана (ICR<1, рейтинг отозван, неликвид). Премии «с запасом» нет — справедливо-оранжево. По состоянию на 11.06.2026.

## АРИФМЕТИКА
- **Тип:** флоатер, купон = **КС ЦБ + 4,00 п.п.** (купонные периоды 5–14; ранее КС+5). {факт} — для флоатера плата за риск = спред купона к базе, НЕ YTM-метод.
- **Контрактный спред к базе:** **+4,0 п.п.** к ключевой ставке. {факт}
- **Эффективный спред с учётом дисконта:** цена ~**88,35%** при дюрации ~0,89 г и погашении 27.08.2027 даёт прибавку от подтягивания к номиналу ≈ **+13–14 п.п. годовых** {оценка} — суммарно рынок зашивает ≈ **КС + 18 п.п.** эффективной доходности.
- **Требуемый спред:** группа B/лизинг-ВДО, рейтинг отозван → деградация до «скоринг Basis + нет рейтинга +1 п.п.». Для B+/B по лизингу с ICR<1 требуемая надбавка к КС ориентировочно **+15–18 п.п.** {оценка}
- **Премия:** факт ~+18 п.п. vs требуемое ~+15–18 п.п. → **около нуля / лёгкий минус**, перекрыто стоп-правилами. {суждение}

## ГЛАВНЫЕ АРГУМЕНТЫ
- **ICR опустился ниже 1 (РСБУ, 2025)** {факт} — проценты к уплате превышают прибыль от продаж → обслуживание на рефинансе, а не из потока (блок A=4–5).
- **Кредитный рейтинг НРА B+ ОТОЗВАН 19.09.2025** {факт} — эмитент сейчас БЕЗ действующего рейтинга; Эксперт РА держал ruB- и отозвал ещё в 2022 → потеря внешнего мониторинга, +1 п.п. (§5).
- **D/E ≈ 3,3, долг ~1,94 млрд при капитале ~0,15 млрд, FCF отрицательный** {факт} — высокий леверидж + отрицательный денежный поток = долговая пирамида на рефинансировании (блок A).
- **Неликвид: оборот ~0,1 млн ₽/день, только квалы, дисконт 88%** {факт} — выйти при первых новостях нельзя; дисконт = рыночная доплата за риск+неликвид, а не «подарок».

# S06. Удобрения, химия, лес: секторная гео-карта

**Периметр:** ФосАгро, Акрон, КуйбышевАзот, НКНХ/КОС (СИБУР-контур), Сегежа.

## 1. Гео-профиль сектора
**Удобрения** — привилегированный гео-класс: продовольственные carve-out'ы (C02 §2, T17-агролобби) делают прямые блокировки маловероятными почти в любом режиме → E1 структурно низкий (2–2,5) при живых косвенных осях: E3 (балтийская перевалка — B04 L4-чувствительность; аммиакопровод мёртв), E4 (расчёты — общий налог K17), торговые барьеры нового типа (пошлины ЕС на российские удобрения 2025+ — «санкции без санкций», G11-строка). C09-профиль двусторонний: газовая себестоимость = глобальное преимущество РФ (дешёвый газ против европейских производителей — структурная маржа), ближневосточные шоки (Ормуз-2026) = ценовые премии азотной группе. **Химия (СИБУР-контур):** E5/E6 — катализаторы/лицензии (T12-корзины), экспорт полимеров в Азию с дисконтной логистикой. **Лес (Сегежа):** K28 — терминальная реализация E2×leverage; сектор-предупреждение.

## 2. Трансмиссия
G13/C09 → мировые цены (газ-Европа = себестоимость конкурентов: их боль = наша маржа — редкий канал «чужого страдания»); G11 → пошлинная эрозия премиального рынка (медленная, календарная); G5-аналог: дисконты и фрахт балкерного плеча; G12 → экспортные пошлины/НДПИ-качели (сектор посещался «курсовыми» пошлинами — фискальная мишень второй очереди).

## 3. Сценарная карта
S1: нейтрально-умеренный плюс (санкционного навеса почти нет — снимать нечего; крепкий рубль — минус марже; логистическая нормализация — плюс); НЕ пис-корзина. S2/S3: мировой цикл цен правит; carve-out-стабильность = «тихая гавань» гео-невинности с оговоркой E4/E3-трения. S4: L4–L5-хвост по балтийской перевалке (Усть-Луга-зависимые) + глобальная продовольственная премия частично компенсирует.

## 4. Мониторинг
Мировые цены карбамид/DAP/аммиак и европейские газовые (себестоимостной спред — главный ряд), пошлинный календарь ЕС, C09-строки Залива (Ормуз-режимы), балтийская логистика (B04-маркеры), фискальные строки (пошлины/НДПИ), для Сегежи-класса — K28-каталог предвестников.

## 5. Анти-паттерны / red-team флаги
Перенос нефтяной санкционной драмы на удобрения (carve-out-класс — иная механика). Игнор «пошлин без санкций» (эрозия ЕС-премии реальна, просто медленна). Ормузская премия как структурная (режимная — снимать по деэскалации, C09 §6). Сегежа-соблазн «дно пройдено» без пересчёта от SPO-якоря (K28 §6.5). Забытый спред себестоимости (главное преимущество сектора живёт в ЧУЖИХ издержках — мониторить европейский газ, не только свои цены).

## 6. Дифференциация
ФосАгро — фосфатная ниша + образцовая логистическая перестройка: эталон carve-out-класса; Акрон — азотная группа: максимальная C09-чувствительность (газ/аммиак-шоки); КуйбышевАзот — внутренне-ориентированный микс (капролактам: E5-зависимости); СИБУР-контур — технологическая ось (T12-корзины лицензий) при азиатском сбыте; Сегежа — пост-K28 состояние: торгуется реструктуризационной механикой, гео-ярлык вторичен.

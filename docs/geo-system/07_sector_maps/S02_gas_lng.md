# S02. Газ и СПГ: секторная гео-карта

**Периметр:** Газпром, НОВАТЭК.

## 1. Гео-профиль сектора
Сектор двух несравнимых профилей под одной вывеской. **Газпром:** E2 — терминальная потеря премиального рынка (K22: труба в ЕС мертва как система, «Турецкий поток» — остаток с W7-риском единственности); E9/E12 — государственный инструмент (K08: дивиденды = f(бюджет)); китайская надежда с вассальной ценой (СС-2 — T08 §4: шкалу двигает только контракт с формулой). **НОВАТЭК:** growth-профиль под технологической блокадой — E5×E6 (Arc7-флот, лицензии сжижения: K14, T12), сегментный E1 (проекты в SDN при менее блокированной материнской), уникальная T10-география (свой океан), обнуляемая флотом.

## 2. Трансмиссия
G11 → рыночный доступ Газпрома (энергоразрыв ЕС: запрет СПГ-2027 — календарный триггер); G6 → монетизация пайплайна НОВАТЭКа («актив без монетизации» — K14 §6.1); G9 → СС-2 и юаневая экономика восточного разворота; G12 → НДПИ-качели Газпрома (изъятие→отмена-как-поддержка — полный цикл K08); G13 → мировые цены СПГ (для НОВАТЭКа — двойной канал: цена И конкуренция за покупателя без западного спота).

## 3. Сценарная карта
S1: Газпром — пис-бета №1 рынка (частичный возврат хоть какого-то ЕС-объёма = опцион на мёртвом активе, K22 §4-двуликость); НОВАТЭК — снятие проектных санкций монетизируется быстро (заводы готовы — K14 §8). S2: слабый плюс обоим (ставка), структурные проблемы не чинятся. S3: Газпром — фискально-инфраструктурный инструмент с отрицательным FCF-профилем эпизодами; НОВАТЭК — «одна линия с просачиванием в Китай». S4: оба минус; Газпром дополнительно — W7-хвост (последняя труба).

## 4. Мониторинг
СС-2: только финансовое закрытие/ценовая формула (меморандумы не считаются — T04-классификация); Arc7: каждое сданное/зашедшее судно — событие; календарь ЕС (СПГ-запрет 2027, промежуточные ограничения перевалки); НДПИ-строки бюджетных пакетов; отгрузки Арктик СПГ-2 (эпизодические китайские приёмки — темп просачивания); турецко-балканский транзит (единственность ТП).

## 5. Анти-паттерны / red-team флаги
Европейская ностальгия в терминальных допущениях (C06: труба в недружественном направлении = 0 в S3, опцион в S1 — двойная запись). СС-2 по европейской марже (вассальная цена — T08 §4). «Найдут танкеры» для Arc7 (физическая асимптота — K14 §6.4). Усреднение сегментного E1 НОВАТЭКа в один балл (Ямал работает ≠ Арктик-2 работает). Дивидендная модель Газпрома от FCF без бюджетной строки.

## 6. Дифференциация
Газпром — ставка на политическое воскрешение рынка (чистейший S1-инструмент с фискальным грузом в S3); НОВАТЭК — ставка на технологический обход/деэскалацию G6 (флот важнее переговоров): разные события двигают их врозь — парой торгуются только в грубых пис-волнах.

# Финансы и оценка — ПАО «Белон» (BLNG)

**Качество данных: НИЗКОЕ (data_quality=low).** Компания почти не раскрывает
отчётность; ключевой актив — доля 32,24% в ООО «ММК-Уголь» — отдельно не
раскрывается. Числа собраны из веба (MOEX ISS, Smart-Lab) и сверены с
`business_model.md`. Эмитент подтверждён: SECID **BLNG**, ISIN
**RU000A0J2QG8**, обыкновенные акции, 1 150 млн шт., котировка 9,70 ₽
(29.05.2026).

## 1. Динамика выручки и прибыли
Белон — **холдинговая «пустышка»**: собственной операционной деятельности
почти нет, выручка стабильно ~6 млн руб./год (после вывода добывающих активов
в 2016 г.). Поэтому строки P&L по выручке/EBITDA для оценки бесполезны.

Прибыль/убыток формируется **неденежной переоценкой доли в ММК-Уголь** по
методу долевого участия и движется вместе с ценами на коксующийся уголь:
- 2021: +3,43 млрд ₽, 2022: +1,11 млрд ₽ (пик цен на уголь);
- 2023: −2,04 млрд ₽, 2024: −7,57 млрд ₽, 2025: ≈ −7,5 млрд ₽ (*оценка по
  мостику капитала; на Smart-Lab дублируется значение 2024 — требует ручной
  проверки*).

Собственный капитал обвалился с 13,2 млрд ₽ (2023) до 3,57 млрд ₽ (2025) —
обесценение угольного актива на фоне кризиса в Кузбассе.

## 2. Здоровье баланса
**Долга нет, чистый долг = 0** во всех периодах (заимствования
консолидированы на уровне ММК-Уголь). Это плюс, но он не спасает от обесценения
актива. Денежная позиция на уровне холдинга не раскрыта.

## 3. Денежный поток
CFO / CapEx / FCF на уровне холдинга **не раскрыты** — в JSON null. Дивиденды

## 4. Оценка по мультипликаторам — АНОМАЛИЯ (anomaly_flag=true)
- **EV/EBITDA — не вычислим**: EBITDA ≈ 0 (нет операционной деятельности).
- **ND/EBITDA — не вычислим** по той же причине (числитель = 0, знаменатель ≈ 0).
- **P/E отрицателен** (убытки 2023–2025).
- **P/S ≈ 1859** — аномально высок из-за мизерной выручки, не информативен.
- **P/B ≈ 3,1** (2025) — рынок оценивает компанию выше обвалившегося капитала,
  закладывая опцион на восстановление цен на уголь.

Единственный осмысленный ориентир — **NAV / балансовая стоимость доли**
(BVPS 2025 ≈ 3,10 ₽; 2024 ≈ 9,65 ₽).

# Корпоративное управление — ПАО «Ковровский механический завод» (KMEZ)

_На 25.06.2026. Качество данных: низкое. Общий флаг управления: **weak** (слабое).
Итоговый балл: **2,6** из 5 (норма сектора — 3,0)._

## Вердикт

**Слабое управление с точки зрения миноритария — но без сработавших «красных флагов».**
KMEZ — закрытое госпредприятие атомно-оборонного контура: ~94–97% капитала под
контролем «ТВЭЛ»/«Росатома», бумага 3-го уровня листинга только для квалинвесторов,
дивидендов по обыкновенным акциям нет. Управление выстроено под задачи группы, а не
под акционерную стоимость; у миноритария нет ни денежного потока, ни переговорной силы.

## Почему такой балл (2,6)

- _Факт._ Дивиденды по обыкновенным акциям не платились ни в один год 2021–2025, в
  том числе в прибыльные 2024–2025 (499 и 737 млн ₽); решения ОСА — «не выплачивать».
  Источник несостоявшихся выплат — прибыль, не долг (это удержание потока в группе, а
  не «дивиденды из долга»). → `dividend_stability` = 2.
- _Факт._ Свыше 95% выручки — продажи аффилированному собственнику-заказчику «ТВЭЛ»
  по внутригрупповым контрактным ценам без рыночного бенчмарка (related-party severity
  high). → `related_party_flows` = 2, `controller_extraction` = 2.
- _Факт._ 3-й уровень листинга, нет публичной МСФО, урезанный РСБУ, санкции OFAC;
  состав и реальная независимость совета не подтверждены. → `transparency_actions` = 2,
  `board_independence` = 2.
- _Противовес (оценка)._ Единый класс акций без разрыва голос/капитал
  (`vote_capital_gap` = 4) и отсутствие квазиказначейского «навеса»
  (`quasi_treasury` = 4) — это удерживает балл выше «дна» и не даёт скатиться к 1–2.

**Что это значит для миноритария (вывод).** Доходность возможна только как курсовой
рост неликвидной бумаги: канала возврата через дивиденды нет, а распределение
стоимости идёт внутри группы. Структурный риск трансфертных цен и потенциального
squeeze-out (доля группы близка к порогу 95%) присутствует, но доказанного факта
вывода стоимости в раскрытии нет — поэтому ни один из шести override-триггеров методички
не активен.

## Связь с оценкой (assessment)

Балл 2,6 лежит в «российском хвосте» (<3) → по выпуклому контуру маппинга управление
работает на повышение ставки дисконтирования и понижение справедливой цены. Конкретное
число премии к ставке считает финанс-слой по `config/governance_mapping.json` на основе
`overall_score` = 2,6 и пустого списка активных red_flags (override не срабатывает —
премия от взвешенного балла, а не от потолка). Качественно это согласуется с тем, что
рынок уже торгует бумагу ниже балансовой стоимости (P/B < 1): дисконт за управление и
неликвидность частично наблюдаем в самой цене.

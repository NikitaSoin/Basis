# Red-team: ГЕО-КОМПАНИЯ ELMT (geo.json + geo_summary.md, as_of 2026-07-13)

## Специальные проверки задания

**Дословность handoff — чисто.** Совпадает дословно.

**Честность «пересечений с ИНСТ нет» — в основном честно**, но формулировка «заполняю ровно исключённое» численно ложна для канала C — см. п.7.

**Главная проверка (техпотолок) — фактура ПОДТВЕРЖДАЕТСЯ.** «Микрон» в SDN с 31.03.2022, серийный потолок ~90 нм, отечественная литография 350/130нм-прототип отстаёт на поколения, EDA отозваны в 2022. Актуально и корректно названо структурной константой. Дефекты — в атрибуции инструментов и одном непроверенном кейсе.

## Замечания по чек-листу

1. **(п.9, существенное) E8=3,5 противоречит собственному описанию и якорю 5** («двойная уязвимость: рублёвая выручка + валютный капекс») — балл без объяснения дисконта от якоря.

2. **(п.9, существенное) E14=3,0 без ссылки на якорь и против якорной семантики** — якорь 3 = «высокая пис-бета», ELMT по центральному тезису отчёта НЕ является пис-бетой (взаимогасящиеся каналы).

3. **(п.9, существенное) E1=3,5 не согласован с E13=4,0** — если верен тезис E13 (правило 50%, бумага блокирована), E1 занижен относительно собственного вывода.

4. **(п.11, косметика) Неверный инструмент экспортконтроля как источник** — соглашение янв-2023 (китайский трек) процитировано вместо РФ-специфичных актов 2022.

5. **(п.11 + мандат сверки, БЛОКЕР) Блок-статус по 50%-правилу оставлен несверенным, хотя проверяется открытым реестром — самое дешёвое закрытие самого крупного флага.** SDN-статус Ростеха и Сбера публично проверяем; при подтверждении суммы долей 41,9+41,66=83,6%>50% блок-статус следует автоматически. Отчёт публикует «самый недооценённый гео-риск владения» в подвешенном состоянии.

6. **(п.2/8, существенное) Величина канала A (+1,5 п.п.) не выведена, а унаследована** — зеркалит чужое исключение ИНСТ без независимой проверки калибровки.

7. **(п.3, существенное) «Заполняю ровно исключённое» — численно неверно для канала C** — ИНСТ исключил 4 п.п., ГЕО ставит 5-8% (включая 2-3 п.п. за E15-непрозрачность, которой у ИНСТ не было вовсе). Не двойной счёт, но заявление о зеркальности ложное и маскирует нетто-новую надбавку.

8. **(п.8, существенное) Величины канала C не заякорены историческими спредами** — 3-5% и 2-3% экспертные без якоря (например, K04 Русал даёт событийное окно).

9. **(п.8, косметика) «Пис-корзина +30-40%» без источника и метода.**

10. **(п.1/10, БЛОКЕР) Канал B «сжатие терминальной выручки vs неограниченный аналог» — сравнение с несуществующей альтернативой + неопределённый носитель поправок.** Канал A (+1,5 п.п. к Ke) не имеет носителя при DCF insufficient_data — риск задвоения на стыке при механической сборке карточки.

11. **(п.5/8, существенное) C08-декомпозиция смешивает четыре разных исхода в одном наборе процентов** (выручка/EBITDA/убыток/акция) без указания базы.

12. **(п.13/11, существенное) Цена трёхмесячной давности в шаге имплайд-рынка** (21.04.2026 при отчёте 13.07.2026, за это время IMOEX на многолетних минимумах).

13. **(п.9, косметика) E11 присвоен числовой балл вопреки правилу «только флаг».**

14. **(п.11, косметика) Мелкая эпистемика** — доля экспорта помечена «факт» при нераскрытии; кейс завода 65нм без источника; конфликт с institutions.json по срокам не разрешён.

**Чисто:** п.2 (макро-ставка), п.4 (имплайд-рынок, с оговоркой п.12), п.5/6 (сверхатрибуция/эндогенность образцовые), п.7 (survivorship), п.12 (нет данных ≠ нет проблемы), п.13 (свежесть), п.14 (пандитство), п.15 (ретроспектива).

# План Фазы 3 — перевод страниц на примитивы

Принцип: ПО ОДНОЙ странице, каждая — отдельный коммит + проверка на ЖИВОМ в обеих
темах ДО следующей. Страницы — независимые компоненты, поэтому регресс локален.
Примитивы/токены во время миграции НЕ меняем (заморожены). styles.css-классы НЕ
удаляем по ходу — чистку делаем В КОНЦЕ, когда все потребители переведены.

Метод проверки каждой страницы (Definition of Done): push → дождаться смены хеша
бандла на inbasis.ru → владелец открывает ИМЕННО эту страницу в светлой и тёмной
теме → подтверждает паритет (ничего не потеряно) + улучшения → только потом
следующая.

Обоснование: сначала самая простая изолированная страница — отлаживаем плейбук
(как менять рукописные классы на tw-примитивы без регресса). Глобальный сайдбар
СОЗНАТЕЛЬНО сдвинут к середине (после 3-4 локальных страниц): он виден везде,
поэтому берёмся за него только с отлаженным плейбуком. Карточка — в конце,
максимальный риск, отдельными заходами.

## Карточка компании — разбивка на безопасные заходы (НЕ один присест)

Карточка = общий «каркас» (шапка: логотип/тикер/название/чип сектора/цена+дельта,
сетка KPI, таб-бар) + 6-7 рендер-функций по табам. Заходы:

- **7a. Каркас + таб-бар** — шапка на KpiTile/Badge/Chip, таб-навигация на примитив
  `Tabs` (ARIA), bento-иерархия шапки. НЕ трогаем содержимое табов. Проверка: все
  табы по-прежнему открываются, контент внутри не изменился. (Закрывает «группировка
  7 табов» + «bento» для карточки.)
- **7b. Таб «Финансы»** (renderFinancials) — P&L/баланс/ОДДС на примитив `Table`
  (числа вправо, моно, дельты ▲/▼, hover), мультипликаторы на `KpiTile`, формат
  чисел через format.js. (Закрывает «реальные таблицы/hover-строки».)
- **7c. Таб «Управление»** (renderGovernance) — бар владения уже починен; дивиденды
  таблицей+графиком, риски/прецеденты на Card/Badge, флаг качества на Badge.
- **7d. Таб «Обзор»** (renderOverview) — 6 блоков на Card/KpiTile, аргументы за/против,
  риски, справедливая цена — bento-иерархия.
- **7e. Таб «Бизнес-модель»** (renderBusinessProfile) — секции на Card, единый ритм.
- **7f. Coming-soon табы** (Глубокий разбор/Консилиум/Стресс-тест) — единый компонент
  заглушки на примитивах; малый объём.

Каждый под-заход = отдельный коммит + проверка на живом на 1-2 тикерах (GAZP/ROSN):
открыть этот таб + бегло остальные (не разъехались ли), обе темы.

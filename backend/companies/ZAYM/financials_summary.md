**Расхождение методов >30%:** P/E (256) vs P/BV×ROE (150). Причина: P/E предполагает сохранение или рост мультипликатора к 5x (сейчас 3.4x — потенциальная переоценка); P/BV×ROE показывает, что при ROE ~28% ≈ Ke 27.2% P/B должен быть чуть выше 1.0 — т.е. акция практически на fair value сейчас. Ширина коридора отражает data_quality LOW.

**Ключевой катализатор:** снижение КС ЦБ РФ → снижение Ke → рост justified P/B и DDM-оценки. При Ke 22% (КС ~10%) и ROE 28% justified P/B → ~1.3x → fair value ~190 RUB.

**Главный риск:** дальнейшее регуляторное ужесточение (МПЛ, ПДН) → ROE < 24% → justified P/B < 1.0 → fair value < 145 RUB (ниже текущей цены).

## 1. Динамика выручки и прибыли

Выручка за 5 лет выросла с 118,8 до 179,2 млрд ₽ (+51%), CAGR ~11% — тарифная индексация + рост потребления ДФО.

EBITDA: 8,2 млрд (2022) → 14,8 млрд (2025), маржа EBITDA выросла с 6,5% до 8,3%.

Чистая прибыль — волатильна:
- 2023: 4,75 млрд (пик)
- **2024: 2,76 млрд (-42% при росте выручки +12% и EBITDA +25%)** — аномалия. По business_model, в дочерней ДГК прочие расходы по РСБУ выросли в 2,5 раза (убыток ~42,7 млрд). В МСФО эффект меньше, но без примечаний подтвердить нельзя.
- 2025: 5,29 млрд (+91%) — восстановление.

**Нормализация:** adjusted = reported для всех лет (мост пустой). Без примечаний МСФО корректировать 2024 — домысел. Аномалия помечена флагом.

Рентабельность по ЧП (ROS) крайне низкая: 1,4–3,0% — типично для вертикально интегрированного энергосбыта с регулируемыми тарифами.

Капитал: 20,2 → 32,0 млрд ₽. P/B текущий = 0,95x (ниже 1). Гудвил/НМА не раскрыты; tangible = балансовый.

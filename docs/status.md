  (IMOEX/RTSI/MCFTR); дивиденды + total return; `company_metrics`; волатильность,
  гибрид-бета (MOEX + расчёт с поправкой Диммсона), корреляции, R², VaR,
  downside-волатильность; ОФЗ G-curve; Шарп/альфа/Сортино/CAPM; графики, секторная
  диаграмма, концентрация; индекс качества + субиндексы + интерпретация корреляций.
  Вкладки: Состав, Агрегирующая, Доходность/оценка, Риск, Корреляции, ИИ-Диагноз
  (заглушка), Стресс-тест (заглушка).
- **Облигации**: полный охват ~3100 (борды TQOB/TQCB/TQOY/TQOD/TQRD); двойной
  рейтинг (рынок по спреду + агентский smart-lab) + расхождение; типы купонов
  (фикс/флоатер/линкер/структурная); муни; дефолтные; связка выпуск→эмитент +
  блок долговой нагрузки «сможет ли расплатиться»; глубокие разборы-якоря
  (ТГК-14/Сегежа/ЕвроТранс). docs/bonds-methodology.md.
- **Фьючерсы**: срез FORTS ~560 (плечо/ГО/экспирация/срочная структура); контекст
  базового актива (рубль/нефть). docs/futures-methodology.md.
- **Фонды**: БПИФ/ETF 102 (тип/TER-в-деньги/ликвидность); разбор среза. Состав

## Бизнес-контекст (неизменное)
Независимый аналитический слой и «второе мнение» для самостоятельного частного
инвестора: цельная картина (компания, рынок, макро, гео, портфель) вместо тумана.
НЕ брокер, не сделки, не «купить/продать». Снижает неопределённость, не хайп.
Ценность — доверие, логика, полнота рассуждений.

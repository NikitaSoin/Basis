### 🟠 Вердикт — риск недоплачен / на грани преддефолтного

Бумага платит ~15 п.п. сверх ОФЗ — широкий спред, но он НЕ компенсирует комбинацию «ICR ~1,2 + 100% краткосрочного долга + нет кэша + хронические налоговые недоимки» при дюрации 2,7 года. По стоп-правилу (ICR<1,5 + стена погашений из всего долга) вердикт не лучше оранжевого *(суждение)*.

> Эмитент — ООО «Идель Нефтемаш» (Татарстан/Башкирия), нефтепромысловое оборудование. Входит в одну группу с «Л-Старт». *(факт)*

### Природа доходности

YTM 30,60%, цена 100,00%, дюрация ~2,71 г, купон фикс 27%, оферты нет, погашение 02.04.2031 *(факт)*. Оферты нет → YTM корректна (near-offer артефакта нет). Размещён в апреле 2026, только для квалинвесторов, объём 250 млн — малоликвиден. Спред = кредитная премия (основное) + премия за неликвид; 30,6% — ценник на риск рефинансирования эмитента с почти нулевым кэшем *(суждение)*.

### Арифметика (3 шага)

Фактический G-spread ≈ 30,6% − ОФЗ ~15,5% ≈ +15 п.п. *(оценка)*. Требуемый: медиана B / нефтегаз-оборудование / 2–3 г ~9–11 п.п. + неликвид (+1) + стена погашений 100% краткосрочного (+1,5) + длинная дюрация для B (+0,5) ≈ 12–14 п.п. *(оценка)*. Премия ≈ +1…+2 п.п. формально — но перекрывается стоп-правилом.

### Аргументы (факт → механизм → вывод)

1. **ICR ~1,24 (снизился с 1,55); ЧП 30 млн при выручке 2,31 млрд (рентабельность 1,3%)** *(факт, РСБУ-2025)* → почти весь операционный результат уходит на проценты, запаса нет → любой рост ставки рефинансирования или провал по контракту делает обслуживание невозможным. Преддефолтный профиль.
2. **100% долга краткосрочный (1,076 млрд), денежные средства 101 тыс. ₽ (−99,5%), кредитных линий нет** *(факт)* → кэш покрывает короткий долг на ~0,01%; компания живёт на постоянном перекредитовании → классическая стена рефинансирования, где и происходят дефолты ВДО.
3. **Налоговые недоимки 39,8 млн непрерывно с августа 2024** *(факт)* → компания кредитуется у ФНС (выше держателей облигаций в очереди) → налоговый долг — ранний маркёр кассового разрыва; блокировок счетов ещё нет, но это вопрос времени.
4. **Нефтепромысловое оборудование; контракты с качественными контрагентами = 92% выручки-2025** *(факт, НКР)* → спрос есть, но при высокой ставке и капзатратной отрасли это не спасает баланс, перегруженный коротким долгом → бизнес жизнеспособен, финансовая конструкция — нет.

### Проверка ожидаемыми потерями

Для B PD ~5–10%, LGD ~70% → минимально ~5,6 п.п. Бумага платит ~15 п.п. → формально покрыто. Но при ICR~1,2 + нулевом кэше реальная PD ближе к верхней границе/выше группы, и за 2,7 года кумулятивная PD кратно растёт → запас тоньше, чем кажется по году *(оценка/суждение)*.

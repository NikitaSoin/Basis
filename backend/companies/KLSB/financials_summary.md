# KLSB — Калужская сбытовая компания: финансы и оценка

Данные: МСФО 2021-2022 (PDF, высокая достоверность) + агрегатор smart-lab 2023-2025 (medium). Баланс и ОДДС за 2023-2025 недоступны. Все числа — расчётные, источники раскрыты в financials.json.

---

## 1. Динамика выручки и прибыли

Выручка растёт стабильно: 24.6 млрд (2021) → 25.3 (2022) → 27.2 (2023) → 30.0 (2024) → 35.0 млрд руб (2025). Среднегодовой рост ~9%. Рост транслирует инфляцию тарифов и рост потребления, а не операционные улучшения: компания — «пропускной клапан» оптового рынка.

Чистая прибыль тонкая, но положительная: 86 млн (2021) → 96 (2022) → 113 (2023) → 124 (2024) → 84 млн руб (2025). Снижение 2025 (-32%) объясняется ростом процентных расходов при пике ключевой ставки ЦБ (16%+). Нормализованная прибыль = отчётной (разовых факторов не выявлено; для 2023-2025 проверка невозможна без первичной отчётности — оговорка зафиксирована в bridge).

EBITDA: скачок с 680 млн (2023) до 1400 млн (2024) — удвоение по данным агрегатора. Аномалия не верифицирована первичной отчётностью; anomaly_flag=true.

## 2. Баланс

Данные только за 2021-2022. Структура типична для энергосбыта с кассовым разрывом:

- Высокая дебиторка: 2.2-2.3 млрд руб (~9% выручки) — компания платит на ОРЭМ вперёд, собирает деньги с задержкой
- Высокий долг: чистый долг 3.1-3.2 млрд руб (2021-2022); оценочно ~3.2 млрд на конец 2024. Net debt/EBITDA = 4.0-4.7x (2021-2023)
- Капитал: 1.1-1.2 млрд руб (2021-2022); ощутимый капитал практически равен балансовому (НМА незначительные, гудвила нет)
- Current ratio < 1 (0.75-0.88) — текущие обязательства превышают оборотные активы; нормально для сбыта с кредитным финансированием кассового разрыва

## 3. Качество денежного потока

FCF данные только 2021-2022: 348 млн (2021) и 82 млн (2022). Высокая волатильность: capex вырос с 165 до 255 млн при стабильном CFO. Capex/выручка 0.7-1% — минимальный для сбыта без сетей. За 2023-2025 данных по CFO нет.

## 4. Мультипликаторы

Текущий P/E = 21.1x — выше 5y_mean=18.0x; бумага оценена чуть дороже исторического среднего. P/B доступен только за 2021-2022: 1.45-1.58x, среднее 1.51x. EV/EBITDA 2024 = 3.6x аномально низкий (скачок EBITDA в знаменателе).

## 5. Прогноз

Брокерского покрытия нет (листинг 3-го уровня). Механический прогноз: выручка 2026 ~38.2 млрд руб (+9%); ЧП 2026 ~92 млн руб (+10% за счёт снижения ставки ЦБ). Погрешность высокая.

## 6. Справедливая стоимость

Маршрутизация: регулируемый энергосбыт. Применены: P/E (adj), P/B, EV/EBITDA, CAPM. DCF не применим (EV=857 млн < net_debt=3200 млн при Ke=20%). Дивидендная не применима (DPS=0 с 2007 г.).

| Метод | Результат |
|---|---:|
| Исторический P/E (17.98x * EPS_fwd 1.01 руб) | 18.16 руб |
| Исторический P/B (1.51x * BVPS_2022 13.34 руб) | 20.20 руб |
| EV/EBITDA историческое (5.94x * EBITDA_fwd 1077 млн - долг 3200 млн) | 34.94 руб |
| DCF | не применим |
| CAPM (Ke=20%, DPS=0, P/E=1/Ke=5x) | 5.05 руб |

Расхождение методов 592%. Причина: CAPM отражает требуемую доходность при высокой ставке ЦБ; P/E/P/B кластер совпадает с рыночной ценой; EV/EBITDA зависит от неопределённого долга.

Коридор справедливой стоимости (дисконт 20% к P/E/P/B за governance и ликвидность):
- Консервативный: 15.3 руб
- Базовый: 19.2 руб
- Оптимистичный: 34.9 руб (условно, при подтверждении EBITDA и долга первичной отчётностью)

Оценка ненадёжна: data_quality=medium, долг оценочный, EBITDA 2024 под вопросом. Бумага малоликвидна, дивидендов нет 18+ лет — для миноритария единственный способ возврата капитала — рост котировки неликвидной акции.

# K10. Эмбарго ЕС + потолок цены: жизненный цикл дисконта

**Статус:** якорный кейс, полный формат.

## 1. Паспорт
Тип: экспортные ограничения нового поколения (эмбарго покупателя + ценовой потолок через услуги). Период: 05.12.2022 → настоящее время (живой ряд). Оси: G5 (якорный ряд), G11, C05 (декомпозиция дисконта), T09 (фазы адаптации). Роли: калибровка «пилы» G5; бэктест-эпизод (out-of-sample резерв V01).

## 2. Хронология [ФАКТ, если не отмечено]
05.12.2022 — эмбарго ЕС на морскую российскую нефть + потолок G7 $60 (западные страховка/фрахт/финансирование — только при цене ≤ потолка). 05.02.2023 — то же для нефтепродуктов (потолки $100/$45). Январь 2023 — пик боли: средняя Urals ~$49–50 при Brent в районе $83–84 → дисконт ~$34 (публикация Минфина). 2023 — двойная адаптация: (а) физическая — теневой флот (закупки сотен возрастных танкеров, инвестиции порядка миллиардов долларов [ОЦЕНКА]), разворот на Индию (с околонулевой доли до трети+ морского экспорта за год) и Китай; (б) фискальная — правительство ступенчато сжало «налоговый дисконт» (учитываемый в НДПИ/пошлинах максимум: с $34 весной до $20 к осени 2023), переложив часть боли с бюджета на компании. Итог 2024: дисконт ~$10–13 — фаза равновесия. Энфорсмент-волны против равновесия: первые SDN за нарушение потолка (танкеры, октябрь 2023) → K19 (01.2025, ~180 судов) → снижение потолка ЕС/UK до $47,6 (динамический, с 03.09.2025; США остались на $60 — расщепление коалиции) → K23 (SDN мейджоров, 10.2025) → каждое — временное расширение дисконта; на дату сборки — снова повышенная зона ~$15–20+ [ОЦЕНКА], спор о заморозке потолка на ~$44 в 21-м пакете [поиск].

## 3. Механика по каналам
Учебная демонстрация C05 §1: пик-2023 = логистическая компонента (флота нет) + комплаенс-премия (покупатели боятся) при работающей потолочной; сжатие-2024 = построенный флот убрал (а), привычка покупателей сузила (б); волны 2025–26 = энфорсмент бьёт по обеим. Дизайн потолка — редкий случай «санкции против санкционера тоже»: G7 сознательно сохранял объёмы на рынке (страх нефтяного шока), изымая маржу — гибрид истощения и самозащиты (T09 §2, функция важнее лозунга).

## 4. Рыночная динамика
Парадокс акций: худший для дисконта период (начало 2023) рынок нефтяных бумаг прошёл без катастрофы — слабый рубль и фискальные демпферы маскировали; лучший (2024, дисконт $10) не дал пропорционального ралли — изъятия и ставка съедали. Урок: **дисконт транслируется в акции через фискально-курсовой фильтр**, прямое чтение «дисконт↔котировки» проваливается (C08-дисциплина: канал G5 всегда в связке с G12/C04).

## 5. Развязка
Нет — ряд живой; структурное состояние: двухконтурный рынок нефти (белый ≤ потолка / теневой с трением) как постоянная архитектура, «цена трения» — устойчивая статья экономики экспорта.

## 6. Уроки для шкал
1. **Якорная линейка G5 — из этого ряда:** ~$1–2 (до 2022) / $10–13 (равновесие-2024) / $15–20 (энфорсмент-волны) / $30+ (шок без флота) — деления 1/3/4/5 калиброваны фактами.
2. **Полураспад санкции без энфорсмента ~6–12 мес** — прогнозная константа для любой новой экспортной меры (T09 §1.3 подтверждён числом).
3. **Пила, не ступень:** энфорсмент-календарь (пакеты, листинги) = календарь расширений дисконта → G5 прогнозируется событийно, не трендом.
4. **Фискальный перехват:** государство управляет распределением боли между бюджетом и компаниями («налоговый дисконт») → для эмитентов сжатие рыночного дисконта ≠ рост маржи автоматически (совместная зона с ИНСТ, правило одного входа).
5. **Адаптация капитализируется:** флот = капекс, создавший постоянный актив обхода; каждая следующая санкция стартует против УЖЕ построенной инфраструктуры (десенсибилизация реальная, не только психологическая).

## 7. Ex-ante тест
Консенсус декабря 2022 — «дисконт $30 навсегда / экспорт рухнет». Система с T09-фазами обязана была выдать против консенсуса: «сжатие к $10–15 в течение 12 месяцев при отсутствии энфорсмента; риски — волны листингов» — проверяемо по ряду (реализовалось). Симметричный тест на 2025: после K19/K23 не экстраполировать расширение навсегда (та же фазовая логика в обе стороны).

## 8. Перенос
Шаблон для любых будущих экспортных мер (металлы, удобрения, СПГ-2027): оценивай тройкой «эластичность покупателей × наличие обходной логистики × энфорсмент-ресурс санкционера»; и для гипотетических российских контр-эмбарго — та же механика против РФ работает в зеркале (покупательская адаптация).

## 9. Источники и метки
Минфин РФ (среднемесячная Urals — первичный ряд), тексты 6-го/8-го/18-го пакетов ЕС и решений G7 по потолку, релизы OFAC по листингам. Все компонентные разложения дисконта — [ОЦЕНКА] по методике C05 §1.

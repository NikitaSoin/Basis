# T12. Технологические цепочки (Миллер): поколенческий лаг и медленная деградация

**Слой:** знания/методология. Опора: Миллер «Chip War», литература экспортного контроля. Роль: содержательный движок субиндекса G6 и оценок E5/E6; лечит бинарное мышление «отрезали/не отрезали» заменой на **скорость деградации и поколенческий лаг**.

## 1. Суть

Полупроводники — самая концентрированная производственная цепочка в истории: передовая литография — фактическая монополия одной компании (Нидерланды), проектный софт (EDA) — олигополия США, передовое контрактное производство — сверхконцентрация на Тайване; поэтому чипы — идеальный chokepoint (T08) и главный предмет техновойны США–КНР. Три механики, переносимые на любую технозависимость: (1) **экспортный контроль — оружие медленного действия**: эффект не в остановке, а в запрете на СЛЕДУЮЩЕЕ поколение → отставание накапливается со временем; (2) **зрелые узлы ≠ передовые**: ~90% применений (промышленность, авто, энергетика, оружие массовых серий) живут на зрелых техпроцессах, которые Китай масштабирует и которые почти неблокируемы → для большинства российских отраслевых нужд канал КНР закрывает номенклатуру (с наценкой и без гарантий); критичен разрыв только там, где нужен передовой узел или сертифицированный оригинал; (3) **деградация парков**: у установленного западного оборудования свой темп умирания без сервиса/запчастей/обновлений ПО — каннибализация, серый сервис, падение выхода годного; это медленная кривая, а не событие.

## 2. Применение

**G6 как «скорость деградации», не «есть/нет».** Мониторинг: цены и сроки серого импорта, кейсы отказов, поведение посредников под вторичными санкциями (2024–26: удары по хабам Гонконг/ОАЭ/Турция → трение растёт [ФАКТ-паттерн]). **Отраслевая раскладка для E5/E6:** гражданская авиация — эталон медленной деградации (каннибализация бортов, серые запчасти, сдвиги вправо импортозамещённых программ → растущая цена лётной годности: профиль Аэрофлота); нефтесервис — средняя зависимость (гидроразрыв/каротаж высокого класса — уходившие мейджоры частично замещены, потолок сложности снизился → влияет на трудноизвлекаемые запасы, долгосрочный фактор добычи); телеком — санкционно-чувствительный капекс (базовые станции: запасы → китайское замещение с лагом); станкостроение/ЧПУ — критическая западная прослойка, замещение КНР частичное; СПГ-технологии — крупнотоннажное сжижение и суда Arc7 (K14: перемножение с географией по T10 §4). **Военная экономика:** массовое оружие живёт на зрелых узлах и трофейной/серой номенклатуре → санкции НЕ останавливают производство, но капают в цену и качество (вход в G12-издержки).

## 3. Алгоритм

Шаг 1. Для отрасли/эмитента инвентаризуй технозависимости по трём корзинам: передовой узел (незаместимо) / зрелый узел (КНР-заместимо с наценкой) / сервис-и-обновления установленного парка. Шаг 2. По каждой корзине — поколенческий лаг (на сколько лет/поколений отстаёт доступное) и скорость деградации (что произойдёт за 12–24 мес при текущем режиме). Шаг 3. Экономический перевод: наценка серого канала [ОЦЕНКА-диапазон по прецедентам], рост простоев/ремонтов, сдвиг капекс-программ вправо → мост в EBITDA/капекс для C06. Шаг 4. Проверка на анти-хрупкость: где деградация уже вынудила работающее замещение (обратный сигнал — снижение E5). Шаг 5. Выход: E5/E6-баллы с траекторией («4, ухудшение»), вклад в G6.

## 4. Сквозной мини-пример

Арктик СПГ-2 (K14) по трём корзинам: передовая корзина — крупнотоннажные линии сжижения (западные лицензии ушли → достройка очередей под вопросом/своими силами с лагом и потерей КПД), сервис-корзина — работоспособность построенного, критическая связка — суда Arc7 (санкции на верфь-исполнителя и поставщиков) → итог: «актив построен — монетизация парализована», балл-перемножение по T10 §4. Прогнозная ценность рамки ex ante: уже 02.11.2023 корзиночный разбор давал «паралич вероятнее обхода», против консенсуса «найдут танкеры» — проверяемо в V01 при развороте кейса.

## 5. Анти-паттерны

Бинарность «отрезали/не отрезали». Недооценка зрелых узлов (алармизм «останется без чипов») и переоценка передовых («всё купят в Китае»). Импортозамещение по анонсам (считать по серийным поставкам и сертификации). Игнор софтовой компоненты (железо без обновлений и лицензий = деградирующее железо). Статические наценки (трение серого канала растёт волнами энфорсмента — T09 §1.3, пересматривать). Перенос чиповой драмы на отрасли, где её нет (ритейл, недвижимость — их технориск ничтожен: не изобретать экспозицию).

## 6. Шаблон вывода

«Объект: … Корзины: передовое …, зрелое …, парк-сервис … Поколенческий лаг: …, скорость деградации (12–24 мес): … Экономический мост: наценка … [ОЦЕНКА], капекс-сдвиг …, простои … E5/E6 = …/… (траектория: …). Вклад в G6: … Маркеры: …»

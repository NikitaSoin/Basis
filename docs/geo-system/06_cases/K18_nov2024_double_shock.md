# K18. Ноябрь-2024: Газпромбанк × «Орешник» — двойной шок одной недели

**Статус:** развёрнутый кейс.

## 1. Паспорт
Тип: кластер финансового и военного шоков с валютной кульминацией. Период: 19–27.11.2024. Оси: G7 (ступень 4→4,5), G1/T04 (сигнальный обмен), C04 §3 (механика рублёвого шока). Роли: подтверждение кластеризации (K06 §6.4); якорь «нет неприкасаемых»; T04-эталон чтения дорогих сигналов (разобран там же).

## 2. Хронология [ФАКТ]
19.11 — 1000-й день войны: первое применение ATACMS по территории РФ (Брянская обл.) после снятия ограничений; в тот же день — указ об обновлённых Основах ядерной доктрины (расширение оснований). 21.11 — двойной удар дня: OFAC вносит в SDN **Газпромбанк** (+ ~50 банков) — последний крупный неблокированный канал расчётов за энергоносители (через него шли платежи за газ по «рублёвой схеме»); вечером — удар «Орешником» по Днепру с последующим обращением. 21–27.11 — валютная кульминация: рубль слабеет к ~110–114/$ (внебиржевые пики), спреды слоёв курса разъезжаются (C04 §1в), ЦБ приостанавливает покупки валюты по бюджетному правилу до конца года; IMOEX продавливается к годовым минимумам (дно цикла — 17.12, ~2 370). Параллельный фискальный аккорд той же осени — налог 40% Транснефти (K30) — добавил ощущение «сезона изъятий».

## 3. Механика по каналам
Финансовый канал: ГПБ-санкция ударила не по банку как эмитенту, а по РАСЧЁТНОЙ функции — экспортная выручка «зависла» в перестройке маршрутов → лаги возврата = временный провал предложения валюты при неизменном торговом балансе → курсовой шок (образцовая демонстрация C04 §3.1: платёжная механика важнее баланса). Консенсусная ошибка кануна: «ГПБ не тронут — Европе самой нужен газовый канал» — санкционер пожертвовал чужим удобством ради давления: правило «нет неприкасаемых» для G7-шкалы. Военный канал: обмен ATACMS↔доктрина↔«Орешник» — управляемая эскалационная коммуникация (полный разбор — T04 §4): дорогие сигналы без необратимых шагов → рыночная паника недели против сигнального чтения = сверхреакция, откупленная к началу 2025.

## 4. Рыночная динамика
Совмещение каналов дало нелинейность: каждый шок порознь — переживаем, вместе — валютный мини-кризис; ЦБ-пауза покупок (административный стабилизатор из меню C04 §2) погасила спираль за дни. Для декомпозиции постфактум: курсовая компонента ~ГПБ-механика, акционная — микс страха эскалации и ставочных ожиданий (декабрьский разворот K20 начался ровно из этой ямы).

## 5. Развязка
Расчётные маршруты за энергию перестроены за недели-месяцы (посредники, валюты третьих стран) — очередная фаза T09-адаптации; ядерный обмен остался эпизодом сигнальной лестницы; рынок — в дно 17.12 и разворот (K20). Кластер стал последней страницей «года ужесточения».

## 6. Уроки для шкал
1. **Кластеризация — правило, не совпадение:** политические шоки приходят сериями (юбилейные даты, пакетные решения, ответные обмены) → стресс-тесты и событийные прогнозы строятся на кластер, не на одиночное событие (наследует K06 §6.4, теперь с двумя якорями).
2. **«Нет неприкасаемых»:** аргумент «эту цель не тронут, она нужна самому Западу» — систематически ненадёжен; в E1-скоринге такие «щиты» дают скидку ≤0,5 балла, не категорию (КТК-щит B04 §2 — единственное калиброванное исключение, и то частичное).
3. **Расчётная функция ≠ эмитент:** санкция на банк скорится по его роли в ПОТОКАХ страны (доля энергорасчётов), а не по его балансу — G7-методика.
4. **Сигнальное чтение окупается:** T04-классификация давала «сверхреакция» в реальном времени — эпизод в out-of-sample резерве V01 именно для проверки этого навыка.

## 7. Ex-ante тест
T=18.11: корректное состояние — «юбилейная неделя + ATACMS-решение = окно кластера; ГПБ — первый кандидат оставшегося списка (E1=4,5 с осени)»; конкретная связка «доктрина+Орешник» — непрогнозируема, но её КЛАСС (дорогой ответный сигнал без перехода S4) — прогнозируем из T16-карты порогов. Проверка: не экстраполировал ли агент панику в «начало S4».

## 8. Перенос
Шаблон «двойного шока» — для любых юбилейно-пакетных окон (мониторить календарь символических дат + санкционных дедлайнов совместно); ГПБ-механика — на оставшиеся расчётные каналы (юаневый контур: триггер №24). Сигнальная часть — прямой прецедент для чтения будущих ядерных демонстраций.

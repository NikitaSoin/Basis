### (1b) ИСПРАВЛЕНИЕ FLAT-RATE АРТЕФАКТА (2026-06-17) — метод оценки префа типа-2
- **БЫЛО (артефакт):** дивидендный Гордон по плоской ставке **23,9%** (CAPM 20,9% + governance +3 пп) → справедливая цена 0,52–0,61 ₽, фейковый даунсайд **−55%** при цене префа 1,242 ₽ и реальной дивдоходности ~10,7%. - **СТАЛО:** RTSBP — **преф типа-2** (маленький дивидендный класс, ценится по СВОЕМУ дивиденду, НЕ pro-rata к обычке; интринсик обычки не применяется — иной ценовой масштаб). Справедливая цена = ожидаемый норм. DPS / РАЗУМНАЯ требуемая доходность. Требуемая доходность: **ОФЗ-10л 14,6% + ~2 пп премия энергосбыт/неликвид → коридор 14,6–18%, база 16,5%** (НЕ 24%). - **Новый `fair_value_range` = 0,70 / 0,89 / 1,05 ₽** (cons r=18%/g=1%/норм.DPS 0,1189; база r=16,5%/g=1,5%/трейлинг DPS 0,1328; opt r=14,6%/g=2%). Апсайд к цене префа 1,242 ₽: база **−28,5%**, opt −15,2% → умеренный даунсайд, не фейковый −55%.
- **Доп-сигнал (факт→суждение):** дивдоходность ~10,7% (факт: DPS 0,1328 / 1,242) — в ВЕРХНЕЙ части собственной ист. нормы → преф скорее дёшев к своему дивиденду при условии устойчивости выплаты. Это сигнал, не подгонка под рынок.

### (4) Синтез-вердикт по ПРЕФУ
RTSBP — **малоликвидный преф дочернего энергосбыта под ~96%-ным контролем ГК «ТНС энерго».** Главный фактор для префа — **дивиденд**: он возобновлён и растёт (0,048 → 0,105 → 0,133 ₽), давая высокую трейлинг-доходность **~10,7%** — заметно выше обычки (3,3%).

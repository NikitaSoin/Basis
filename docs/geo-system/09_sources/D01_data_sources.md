# D01. Источники данных: реестр, деградация после 2022, прокси

**Слой:** знания (обновляемый). Консолидированный реестр к B03: что откуда берётся, какого класса, что сломалось после 2022 и чем заменено. Правила классов: **A** — первичные документы и официальная статистика (двигают баллы); **B** — качественные агентства/издания (двигают баллы при отсутствии A); **C** — телеграм, инсайды, анонимные утечки (только гипотезы в watchlist, баллы не двигают — B03 §2).

## 1. Санкционно-правовой контур (класс A)
OFAC: SDN Search, пресс-релизы, FAQ, General Licenses (телеграф разрядки/ужесточения — C02 §2); дедлайны GL — календарный watchlist. EU: Official Journal (тексты пакетов — единственный точный источник против пересказов), консолидированная санкционная карта Еврокомиссии. UK OFSI, швейцарский SECO (следует ЕС с лагом — сам лаг информативен). BIS (экспортный контроль: Entity List, FDPR-обновления). Судебные базы (делистинговые иски в судах ЕС — ранние сигналы пересмотров). Российский зеркальный контур: официальное опубликование указов (81, 520, 95, контрсанкционные), решения правкомиссии — публикуются фрагментарно → частично класс B по прессе [деградация: полнота решений правкомиссии невосстановима из открытых источников — фиксировать как лакуну, не домысливать].

## 2. Российский макро-фискальный контур
Класс A: ЦБ (решения/резюме по ставке, ОНЕГДКП, обзоры рисков финрынков, статистика валютной структуры расчётов — G9/G7-входы); Минфин (среднемесячная Urals ~5-го числа — опорный ряд G5/C05; месячное исполнение бюджета и НГД — гайдаровская четвёрка T11); Росстат (недельная инфляция); аукционы ОФЗ (доступ к внутреннему долгу). **Деградация после 2022:** таможенная статистика закрыта (2022, частично восстановлена агрегатами), детальная отчётность банков ограничена, эмитентам разрешено скрытие чувствительных данных (пост. 1102-линия) → правило: закрытая строка = выше E15 и ниже confidence, никогда не «нет данных = нет проблемы». Прокси вместо умерших рядов: экспорт — зеркальная статистика партнёров (таможня КНР — месячно, класс A; индийская торговая статистика; Eurostat по остаткам), денежные агрегаты и валютные потоки — публикации ЦБ вместо детальных форм.

## 3. Товарно-логистический контур
Платные первоисточники (Argus, Platts — дисконты; Kpler, Vortexa — танкер-трекинг) — в базовой конфигурации недоступны → **санкционированные прокси класса B:** регулярные сводки Reuters/Bloomberg с их данными (цитаты Argus/Kpler), Минфин-ряд как якорь Urals, публикуемые фрахтовые индикаторы. Правило прокси: помечать «по данным X через Y», расхождения прокси между собой — фиксировать диапазоном. Мировые ряды (класс A/B): EIA/IEA (балансы нефти), заседания ОПЕК+, LME (металлы), FAO/биржи (зерно, удобрения — C09-каскады).

## 4. Военно-политический контур
Карта и динамика фронта: ISW (ежедневно, класс B, прозападная оптика — учитывать), DeepState (B, украинская), сводки МО РФ/Генштаба ВСУ (обе — C: сторона конфликта; правило двух источников из B03 §2: факт признаётся при подтверждении противником либо нейтральным B). Удары по НПЗ/энергетике: Reuters/Ъ/РБК + региональные (B). Переговорные треки: стенограммы и официальные заявления (A), дипломатические сливы Axios/Reuters/FT (B — исторически информативны, но инструментальны: каждая утечка кому-то служит — T05 reverberation-поправка). Медиапанель T07: федеральные эфиры/ленты — источник класса «данные о намерениях» (не фактов).

## 5. Рыночный контур
Биржевые ряды MOEX (индексы, объёмы, RGBI), раскрытие эмитентов (e-disclosure, МСФО/РСБУ), ЦБ-курсы и внебиржевые индикаторы, спред «биржевой юань ↔ внебиржевой доллар» (ежедневный стресс-индикатор G7 — B03). Внешний контур: котировки замещающих облигаций, исторические DR-ряды (для K-кейсов). Консенсусы аналитиков — класс B и только как объект (что прайсит рынок), не как истина.

## 6. Правила пользования (сквозные)
1. **Датируй каждый факт** датой публикации источника (не события) — фундамент freeze-протокола V01. 2. **Один факт — один класс:** апгрейд C→B/A только новым источником, не повторением того же инсайда десятью каналами. 3. **Мёртвые ссылки:** квартальная ревизия (B03 §1); умерший ряд не выбрасывается из методики молча — заменяется прокси с записью в описи (V03 §2). 4. **Конфликт источников** равного класса = диапазон + явная фиксация конфликта (Принцип 5), не выбор удобного. 5. **Пропаганда обеих сторон** — данные о намерениях и подготовке аудиторий (T07), никогда — о фактах поля. 6. **Платное недоступно ≠ слепота:** каждый платный ряд имеет прокси-цепочку выше; качество ниже — confidence ниже, это честная цена конфигурации.

## 7. Карта лакун (честный слой — обновлять при ревизиях)
Невосстановимое из открытых источников на дату сборки: полные решения правкомиссии по сделкам; детальная структура теневого флота и его бенефициаров (только оценки B-класса); реальные условия китайских расчётных схем (агентские комиссии — вилки по прессе); закрытые строки отчётности санкционных эмитентов; фактические объёмы «С»-счетов в разрезе бумаг. Для каждой лакуны действует связка: E15↑ / confidence↓ / явная оговорка в отчёте.

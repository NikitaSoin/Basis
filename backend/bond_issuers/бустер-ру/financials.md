## Финансы эмитента: сможет ли Бустер.Ру расплатиться

Главный вопрос ВДО — «расплатится ли». По эмитенту картина двойственная *(факт, РСБУ-2025)*: выручка +154% до 452 млн руб., чистая прибыль 88 млн руб. (+12%), маржа по EBITDA ~66%, операционный поток (за вычетом покупки новых цессий) положительный ~90 млн руб. Доходность бизнеса высокая.

Но **долговая нагрузка значительная** *(оценка)*: Долг/EBITDA ~4,3x, Net Debt/Equity ~4,7x, покрытие процентов ICR ~1,9–2,3x. Для коллектора это повышенный, но не критичный уровень при условии продолжения роста сборов. Без поручительства, по оценке рынка, это «уровень C» — собственная кредитоспособность эмитента слабая *(суждение аналитиков)*.

**Вся надёжность держится на поручителе — МФК «Джой Мани»** *(факт)*: рейтинг **ruBB- со стабильным прогнозом, Эксперт РА, подтверждён 24.06.2025**. У поручителя: чистая прибыль 2025 >400 млн руб., капитал ~1,5 млрд руб.

**Важно держателю:** рейтинг ruBB- относится к **поручителю, не к эмитенту** — это спекулятивная категория, дефолтный риск реальный; высокий купон ~23,5% есть плата за этот риск *(суждение)*. Качество поручительства — ключевой фактор. Выпуск только для квалинвесторов; есть put-оферта 27.03.2028 и амортизация — YTM корректно считать к оферте.

*Источники: cbonds; smart-lab; Эксперт РА (Джой Мани, 24.06.2025); Т-Банк. По состоянию на 10.06.2026. Профиль эмитента — общий для всех его облигационных выпусков. Метрики долга — РСБУ/оценки аналитиков.*

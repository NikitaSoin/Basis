## ВЕРДИКТ {light:gray}
Светофор-премию построить нельзя: у выпуска нет подтверждённых рыночных параметров (купон/оферта/цена) и нет кредитного рейтинга ни одного из четырёх агентств — фактический и требуемый спред не считаются. Качественно эмитент — крупный, прибыльный дистрибьютор электротехники, но с тонкой маржой и оборотным рычагом; решение по доходности невозможно до раскрытия параметров и присвоения рейтинга.

## АРИФМЕТИКА
- Фактический спред: **не определён** — выпуск «РС ООО-БО-01» на момент проверки в статусе «готовится к торгам», подтверждённых цены/YTM/оферты в открытых источниках нет {факт}
- Требуемый спред: **не определён** — нет ни агентского рейтинга, ни достаточных данных по долгу (Net Debt/EBITDA, ICR) для подразумеваемого рейтинга Basis; группа сравнения не строится {оценка}
- Премия: **неприменима** (gray) — арифметика светофора требует обоих чисел, ни одного нет {суждение}

## ГЛАВНЫЕ АРГУМЕНТЫ
- **Нет кредитного рейтинга ни одного аккредитованного агентства (АКРА/Эксперт РА/НКР/НРА) на 14.06.2026** {факт} — «звание лидера отрасли» по Всероссийскому бизнес-рейтингу к кредитному качеству отношения не имеет — механизм: без рейтинга нет якоря группы сравнения и матрицы PD → требуемый спред не калибруется, по методике надбавка «нет рейтинга» +1 п.п. поверх несуществующей базы
- **Параметры выпуска не подтверждены (купон, объём, срок, оферта)** {факт} — механизм: без YTM/YTP и без дюрации не считается фактический спред → светофор-премия неприменима
- **Рентабельность по чистой прибыли ~2,4% при выручке ~151 млрд ₽ (2025)** {факт} — механизм: дистрибуция живёт на тонкой марже и большом обороте; малая ошибка в дебиторке/запасах съедает прибыль и давит на обслуживание долга → блок A чувствителен к стоимости фондирования и КС
- **Нет публичной долговой картины: Net Debt, EBITDA, Долг/EBITDA, ICR не раскрыты** {факт} — механизм: ядро оценки (блок A) держится на оценочных суждениях, а не на цифрах → достоверность всей оценки ограничена

## ПРОВЕРКА ПОТЕРЯМИ
Расчёт PD×LGD невозможен: без рейтинга нет PD по группе, без параметров выпуска нет спреда для сравнения. Якорь ожидаемых потерь не строится. Косвенно: для крепкого дистрибьютора-лидера PD скорее в зоне A/BBB (~1–2%/год), но это суждение без подтверждения рейтингом; при LGD ~70% минимально нужный спред ориентировочно ~0,7–1,4 п.п. — но сравнивать не с чем. {суждение}

# Группа Черкизово ПАО (GCHE) — флоатеры, доходность за риск

_as_of 2026-06-13. Режим A (флоатеры, привязка к КС). Выпуски: БО-002Р-01 (RU000A10B4V0, фикс/AA(RU)) и 002Р-02 (RU000A10C5F9, флоатер КС+2 п.п., погашение 12.07.2027). Разбор ниже — про флоатер 002Р-02._

## ВЕРДИКТ {light:amber}
Флоатер КС+2 п.п. эмитента AA(RU): спред к риску честный, но узкий — за крепкое имя платят справедливо, без запаса; кредитное качество хорошее, процентный риск держателя снят.

## АРИФМЕТИКА
- Формула купона: **КС + 2,0 п.п.** {факт}
- Прогноз ЦБ по средней КС на 2026: **14,0–14,5%** (базовый сценарий, ОНЕГДКП / решение 24.04) → ожидаемый купон ~16–16,5% {оценка}
- Оценочная YTM (от прогноза ЦБ, не факт): **~16–16,5%** при цене у номинала {оценка}
- Фактический спред = спред купона к базе: **+2,0 п.п. к КС**; G-spread к ОФЗ дюрации ~1,1 г при оценочной YTM ~16,3% и ОФЗ ~14,3% → **~+2,0 п.п.** {оценка}
- Требуемый спред: группа AA(RU)/потребсектор/<2 г → медиана ~1,5–2,5 п.п. {оценка}
- Премия: **~0…+0,5 п.п.** — справедливо, без запаса 🟡

## ГЛАВНЫЕ АРГУМЕНТЫ
- **AA(RU), АКРА, подтверждён 01.08.2025, прогноз стабильный** {факт} — высокая кредитоспособность, прогноз АКРА FCF положительный 2025–2027 → низкая PD, основа жёлтого/зелёного.
- **ND/EBITDA ~1,5–1,8x, ICR (EBITDA/проценты) 3,1x в 2025** (по МСФО: EBITDA 67,2 млрд, фин.расходы 21,5 млрд) {оценка} — умеренный леверидж, проценты покрываются с запасом даже при выросшей КС.
- **Спред КС+2 п.п. — тонкий** {суждение} — для имени AA(RU) это рыночно-справедливо, но не даёт премии; держатель получает плату ровно за риск, апсайда по спреду нет.
- **Процентный риск снят (флоатер)** {факт} — купон плавает с КС, тело не проседает при росте ставки; главный риск — кредитный + траектория КС вниз (купон падает вместе со ставкой).

## ПРОВЕРКА ПОТЕРЯМИ
PD для AA(RU) ~0,3–0,5%/год × LGD ~70% ≈ 0,2–0,35 п.п. минимально. Спред +2 п.п. с большим запасом покрывает ожидаемые потери → по EL картина зелёная; узость премии — вопрос относительной ценности, не покрытия потерь. {оценка}

## ТРИГГЕРЫ
- Снижение КС к 13–14% по плану ЦБ → купон падает, реальная доходность держателя снижается (это механика флоатера, не дефолт-риск) → 🟡.
- Рост долга / новая инвестпрограмма сверх плана, ICR < 2,5x, отмена льготы по налогу АПК → 🟠.
- Понижение рейтинга ниже AA-(RU) → пересмотр требуемого спреда вверх → 🟠.

## БЛОК A — Платёжеспособность {score:2}
- **A1 Долговая нагрузка (динамика).** ND/EBITDA ~1,5–1,8x (2025, оценка от подразумеваемого долга ~110–120 млрд при фин.расходах 21,5 млрд и ставке ~18–20%); EBITDA 67,2 млрд (2025, ↑ с 46,4 в 2024) {оценка} — нагрузка умеренная для сектора, обслуживаемая.
- **A2 Покрытие процентов (ICR).** EBITDA/проценты 3,1x (2025), 3,7x (2024) {оценка} — снизился из-за роста фин.расходов (21,5 vs 12,4 млрд) при росте КС, но остаётся комфортным. При КС+2 п.п. сценарий: проценты ~+10–12%, ICR ~2,7–2,8x — всё ещё >2,5x. {оценка}
- **A3 Ликвидность и стена погашений.** Флоатер 002Р-02 гасится 12.07.2027; capex резко снижен (2025 capex/выручка 8,4% против 14,2% в 2024), FCF вышел в плюс (7,5 млрд 2025) {факт} — инвестпик пройден, давление на ликвидность снижается.
- **A4 Качество денежного потока.** CFO стабилен ~31,7 млрд (2025), CFO/capex 1,31x {факт} — операционный поток покрывает инвестиции, не пирамида.
- **A5 Анти-рисование.** Отчётность МСФО, биоактивы по IAS 41 дают неденежные переоценки P&L (изменение справедливой стоимости −0,9 млрд в 2024) — учтены, не искажают EBITDA-кэш существенно; ETR низкий (1–9%) из-за льготы АПК — риск отмены помечен. {суждение}

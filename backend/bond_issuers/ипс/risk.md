### 🟠 Вердикт — риск недоплачен: купон 30,5% это ценник дистресса, не премия

Купон 30,5% и спред ~24,6 п.п. выглядят как огромная премия, но микровыпуск (~57 млн ₽) без рейтинга у эмитента с прибылью 3 млн ₽ при выручке 884 млн физически не обслуживает долг из операционной прибыли. Так дорого занимает тот, кому больше негде. По стоп-правилам №3/№5 вердикт не лучше оранжевого *(суждение)*.

> ⚠️ Эмитент — ООО «Интернешнл Петролеум Сервис» (ИПС), управляет ~22 АЗС в Москве/МО.

### Природа доходности

YTM 37,6% при цене 96,4% и купоне 30,5% — **не** near-offer артефакт (call далеко, погашение 2030) и не бумажная доходность: бумага платит высокий купон по факту. Доходность ≈ почти весь купон + небольшой подтяг к номиналу. Высокий купон сам по себе — сигнал стоимости риска *(оценка)*.

### Аргументы (факт → механизм → вывод)

1. **Прибыль 3 млн ₽ при выручке 884 млн (рентабельность 0,34%), долг/капитал ~14:1** *(факт, РСБУ 2024)* → операционная прибыль не покрывает даже проценты, обслуживание идёт за счёт новых займов → зависимость от рефинансирования, ICR около/ниже 1 (блок A = 4–5).
2. **Выпуск ~57–60 млн ₽, без рейтинга, для квалов** *(факт, cbonds)* → нет ликвидности вторички и внешнего кредитного контроля; выйти при плохих новостях почти невозможно → «бумажный» рынок, премия за неликвид обязательна.
3. **Бизнес — 22 АЗС в Москве/МО, тонкая маржа топливной розницы под госрегулированием цен и демпфером** *(факт)* → при высокой ставке маржа не покрывает стоимость долга → источник денег структурно слабый (блок B = 4).
4. **Цель займа — модернизация 3 АЗС** *(факт)* → капзатраты без быстрой окупаемости при отрицательном запасе прочности → долг растёт быстрее способности гасить, FCF под давлением.

### Арифметика (3 шага)

Фактический спред ~24,6 п.п. *(оценка)*. Группы по рейтингу нет → скоринг Basis B/B-CCC: медиана ~12–16 п.п. + надбавки (нет рейтинга +1, неликвид +1–1,5, тонкая платёжеспособность +1–2) → требуемый ~16–20 п.п. *(оценка)*. Формальная премия ~+6 п.п. → но по стоп-правилу №3 широкий спред у эмитента без рейтинга с ICR на грани = «рынок видит риск дефолта» → **🟠**.

### Проверка ожидаемыми потерями

Для B PD ~5–10%, LGD ~70% → минимально ~3,5–7 п.п. Фактические ~24,6 п.п. покрывают, но запас — плата за высокую вероятность именно этого сценария + неликвид. Оба якоря (группа и EL) показывают высокий риск → консервативный 🟠 *(оценка)*.

### Рейтинг

Рейтинга от АКРА/Эксперт РА/НКР/НРА **НЕТ** (проверено 10.06.2026). Скоринг ЛИСП «rlBBB-» — алгоритмическая модель, не агентский рейтинг; опираться нельзя. Нет рейтинга → +1 п.п. и потолок прозрачности.

Отчётность 2025/1п2026: прибыль/ICR не выросли → к 🔴; рост маржи → к 🟡; присвоение настоящего рейтинга снимет надбавку; техдефолт/задержка → стоп-правило №4 (recovery); реализация call после 18.09.2026 (право эмитента) — при удешевлении денег эмитент отзовёт, держатель теряет высокий купон.

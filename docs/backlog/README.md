## 🔴 Активные (в работе / ближайшие)
- **Финансист закрывает дыры (В РАБОТЕ).** Пустоты по годам (P&L/баланс/ОДДС/мультипликаторы),
  единицы (где-то трлн подписано как млрд), какие показатели показывать в блоке Финансы.
  Модель Sonnet. См. `docs/backlog/finance_data_fix.md`.
- **Ревизия 19 ранних флагманов** (после финансиста). Темпоральный якорь (атрибуция от средней
  ставки года, не спота) + банковская нелинейность (SBER/VTBR/T — под-каналы). Список:
  ROSN/SBER/GAZP/GMKN/MTSS/MGNT/SNGS/PIKK/LKOH/NVTK/MOEX/TATN/SIBN/T/AFKS/AFLT/MTLR/RTKM/TRNFP.
  Критик теперь ловит это сам. PHOR уже исправлен как образец.

## 🟡 Субагенты — доработка (по опыту раскаток)
- **Критик макро (macro-critic) — детально прописать** по накопленному опыту раскаток: типовые
  ловушки по секторам (демпфер/эскроу/NIM-lag/CoR-сегменты/кубышка/дезинфляция/ВВП-прокси для
  дискреционного), проверка нелинейности, стыки между вкладками, темпоральный якорь, негодные
  прокси. Собрать чек-лист из реальных находок сессии (см. `docs/macro_reflection.md` кейсы).
- **Критики для ДРУГИХ вкладок** — по образцу macro-critic сделать критиков для: бизнес-модель,
  финансы, governance, гео, рынки, облигации. Каждый — второе мнение по своей вкладке (фактчек,
  полнота, стыки, ложная точность). Сейчас критик есть только у макро; он доказал ценность.
- **Механизм «анализ→критик→правка» для других вкладок** — тот же цикл, что показал качество в макро.

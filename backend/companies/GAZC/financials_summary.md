## 5. Оценка справедливой стоимости

Маршрутизация методов:
- DCF → not_applicable (нет предсказуемых FCFF, пассивный холдинг)
- Исторический P/E → not_applicable (нет надёжного ряда; P/E малоинформативен для холдинга)
- NAV / P/B → основной метод (якорь для инвестхолдинга)
- CAPM → применён для диагностики
- Дивидендная → not_applicable (нет регулярных DPS)

### CAPM (диагностика)
Ke = 14,6% + 1,1 × 9,0% + 3,0% (illiq) = 27,5%
Справедливый P/NAV = (ROE − g) / (Ke − g) = (3,08% − 3,0%) / (27,5% − 3,0%) = 0,003
FV по Гордону ≈ 1,06 ₽. Интерпретация: при ROE 3% и Ke 27,5% бизнес разрушает стоимость. Результат методологически корректен: показывает, что при текущей доходности на капитал бумага фундаментально не должна стоить дороже NAV.

## 6. Риски и оговорки

Критические:
- Цена 674 ₽ vs NAV 354 ₽ — премия ~90% без фундаментального основания. Бумага либо спекулятивна, либо отражает ожидания реализации пакета Газпромбанка.
- Нулевой free-float: выход из позиции по текущей цене практически невозможен без проскальзывания.
- Нет регулярных дивидендов: денежный возврат миноритарию отсутствует.
- ROE 3% << Ke 27,5% → хроническое разрушение стоимости при текущей структуре.

Структурные:
- 46% капитала — нереализованная переоценка FVOCI (Газпромбанк под санкциями ЕС/UK).
- Закрытое раскрытие, внутригрупповой контроль через УК «Лидер» / «Газфонд».
- OCR данные 2020–2023 частичны → исторический анализ ограничен.

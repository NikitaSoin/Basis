# Арбитраж карточки BISVP (fact-check v3 — связность)

**Дата:** 2026-06-16
**Источник истины по числам:** companies/BISVP/financials.json
**Роль:** арбитр связности — свожу все вкладки к единому стандарту, масштабу,
числу акций и дивидендной формуле. Триггер пересборки — аудит 4.5/10 (вкладки в
двух несовместимых стандартах; неверное число акций; нераскрытая дивформула префа).

## Сводка арбитража

- **Стандарт сведён:** ✓ — МСФО (консолидация) во всех вкладках; РСБУ только в дивформуле.
- **Масштаб сведён:** ✓ — выручка ~25 млрд (МСФО группа) против ~6 млрд (РСБУ родитель) — разъяснён как разные периметры, не противоречие.
- **Число акций верное:** ✓ — 985,189,110 (949,02 млн обыкн. + 36,17 млн преф) во всех мультипликаторах.
- **Дивформула раскрыта и проверена:** ✓ — DPS = 10% ЧП РСБУ / 246,30 млн; проверка даёт 0,67 ₽ = факт.
- **Синтез-вердикт:** ✓ — единый вывод «около справедливой, не дорого; downside защищён чистым кэшем».

## 1. Единый стандарт и масштаб (главный дефект аудита)

| Что было (аудит) | Что стало |
|---|---|
| БМ/макро — РСБУ (выручка 4,6 млрд, «1% ЧП»); финансы — МСФО (25,1 млрд, «10% ЧП») | Единый стандарт карточки = **МСФО (консолидация)**. БМ переведена на МСФО-масштаб (выручка 32→25,9→25,1 млрд по годам). РСБУ родителя (~6 млрд) оставлен ЯВНО справочно — только под дивформулу |
| «10% ЧП» vs «1% ЧП» выглядели противоречием | Развязано: 10% — ставка формулы на акцию; ~1,47% — совокупная доля класса префов (36,17 из 246,30 млн базы). Оба числа верны, описывают разное |

**Почему 25 млрд (МСФО) ≠ 6 млрд (РСБУ).** МСФО консолидирует дочерние активы
группы; РСБУ — отчётность головного юрлица «Башинформсвязь». Это один эмитент в двух
периметрах, а не ошибка. Подтверждено: smart-lab МСФО (выручка 21,6/25,7/32,0/25,9/25,1),
tadviser/cbonds РСБУ (~6,1/5,6 млрд). Дивиденд по уставу привязан к РСБУ ЧП → РСБУ
правомерен только там.

## 3. Дивидендная формула префа (дефект аудита)

Раскрыта из устава и проверена на факте во всех трёх вкладках (governance, БМ, финансы):

```
DPS = 10% × ЧП_РСБУ / (25% × 985,19 млн) = 10% × ЧП_РСБУ / 246,30 млн
ЧП РСБУ 2023 ≈ 1,65 млрд → DPS = 0,67 ₽  ✓ (= фактически выплачено за 2023)
Доля класса префов = 10% × 36,17/246,30 = 1,47% ЧП РСБУ
```

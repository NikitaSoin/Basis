### 🟠 Вердикт — премия за риск спорная, кредитное качество слабое

Это **флоатер** (купон КС+3 п.п.), светофор по YTM 31,69% строить нельзя — это бумажная величина. Реальная плата = фиксированный спред купона +3 п.п. к КС плюс дисконт цены до 88% номинала. Сам +3 п.п. для BB узковат; рынок добирает премию дисконтом цены. С учётом дисконта плата умеренная, но капитал на грани критического + понижение рейтинга держат вердикт на оранжевом: риск оплачен в обрез *(суждение)*.

### Природа доходности (почему не YTM)

- Купон **КС + 3 п.п.**, ежемесячно *(факт)*; при КС 14,5% (с 24.04.2026) текущий 17,5% — совпадает с БД.
- YTM 31,69% и G-spread для флоатера методически некорректны (шаг 1.1): доходность механически раздувается дисконтом цены.
- Цена **88% номинала** для флоатера — кредитный дисконт: рынок считает +3 п.п. недостаточным для BB и компенсирует падением цены. Эффективный спред к КС с учётом дисконта при дюрации ~1 год ~+6…+8 п.п. *(оценка)*.

### Аргументы (факт → механизм → вывод)

1. **Достаточность капитала ~4,3%, в зоне критических значений** *(факт, АКРА окт.2025)* → тонкий буфер на покрытие убытков по портфелю; рост просрочки быстро съедает капитал → главный кредитный риск, причина понижения рейтинга.
2. **АКРА понизило до BB, прогноз «развивающийся»** *(факт)* → допускает движение в обе стороны в зависимости от обещанной докапитализации → вердикт условен: исполнят — снизится риск, нет — давление вниз.
3. **Мелкий лизинг для МСП (60-е место, портфель спецтехника/авто/оборудование)** *(факт)* → цикличная отрасль, первой страдающая от высокой ставки: реже берут технику, растут изъятия и просрочка → денежный поток уязвим.
4. **Снижение КС (14,5%, тренд вниз)** *(факт)* → двойной эффект для флоатера: купон держателя снижается (минус доходу), но дешевеет рефинансирование эмитента и оживает спрос на лизинг (плюс платёжеспособности) → среда улучшается, заложено в отскок цены.

### Проверка ожидаемыми потерями

Для BB PD ~3–4%, LGD ~70% → минимально ~2,5 п.п. Голый купонный спред +3 п.п. едва покрывает EL — премии за нервы/неликвид почти нет. С учётом дисконта цены (эффективный ~6–8 п.п.) покрытие появляется. Премия существует только благодаря дисконту 88%, а не условиям выпуска *(суждение)*.

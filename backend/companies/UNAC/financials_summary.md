# ОАК (UNAC) — Финансы и оценка: интерпретация

Данные: МСФО, smart-lab + деловые СМИ (PDF первичной отчётности недоступен). Качество: medium.

---

## 1. Динамика выручки и прибыли

Выручка восстановилась после провала 2022 (362 млрд) и к 2025 достигла 759 млрд (+62% к 2022).
Рост — гособоронзаказ и постепенный запуск гражданских программ.

Прибыль — хроническая убыточность 2021–2024: −17, −26, −36, −15 млрд руб.
adjusted = reported: PDF недоступен, разовые из первичного источника не подтверждены.
Аномалия 2022 (gross_margin 5.8% против 30% в 2021) — вероятно списания/убыточные контракты,
без первичной отчётности не верифицируется.

2025: по smart-lab (прогноз) — первый год прибыли +9.4 млрд. Требует подтверждения.
EBITDA-маржа 2025: 13.9% (разворот). Finance_costs ~95 млрд/год делают прибыль хрупкой.

## 2. Баланс: долг, ликвидность, капитал

Чистый долг ~595 млрд (2025) — превышает капитализацию 401 млрд. Обязательства 2024: 2 209 млрд
при активах 2 289 млрд → капитал 80 млрд (2024) / 139 млрд (2025). Леверидж 28x — экстремальный.

Детали баланса не раскрыты (ГОЗ + SDN). CFO отрицателен 2022–2024 (-3, -74, -34 млрд).
FCF = null (capex не раскрыт). Компания держится на госфинансировании.

## 4. Справедливая стоимость

- **DCF**: insufficient_data. При FCF₁=20 млрд: EV=80 млрд < долга 595 млрд → equity < 0.
- **P/E**: not_applicable. Убытки 2021–2024; 42.8x (2025) — 1 точка, не серия.
- **P/B (основной)**: BVPS 2025 = 0.1377₽. Cons 0.0689₽ / Base 0.1377₽ / Opt 0.2754₽.
- **EV/Sales-relative**: break-even EV/Sales = 0.78x; реалистичный 0.1–0.2x → equity = 0.
- **CAPM 12m**: Ke=28.1% (бета 1.5); EPS 0.0093₽ → таргет 0.033₽.
- **Дивидендная**: not_applicable.

**Коридор: 0.033–0.2754₽**. Текущая цена 0.3962₽ — выше верхней границы по всем методам.
Расхождение P/B base vs CAPM = 4x → коридор намеренно широкий.

## Ключевые риски

1. Долг 595 млрд > капитализации — любое ухудшение уничтожает equity
2. 2025 данные неаудированы; без подтверждения прибыли метрики меняются
3. SDN ограничивает раскрытие — детали баланса, ОС, запасы неизвестны
4. Допэмиссии размывают миноритариев
5. Бета 1.5 — суждение; реальная выше при ликвидности UNAC

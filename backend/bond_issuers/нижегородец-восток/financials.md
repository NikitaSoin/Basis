## Финансы эмитента: сможет ли Нижегородец Восток расплатиться

Главный вопрос — «сможет ли расплатиться» — здесь напряжённый. Это глубокое ВДО: высокий купон (19% годовых) есть прямая плата за повышенный риск, а не «выгода» *(суждение)*.

**Рейтинг *(факт)*:** Эксперт РА присвоил ruB «стабильный» 21.02.2024 — уже низкая, спекулятивная ступень. **Критично: 17.02.2025 агентство ОТОЗВАЛО рейтинг без подтверждения** «в связи с окончанием срока действия договора и отсутствием достаточной информации». То есть на 10.06.2026 действующего публичного рейтинга НЕТ — негативный сигнал прозрачности *(суждение)*.

**Долг и покрытие *(факт, отчётность 2023–1кв.2024)*:** Net Debt/EBITDA ~**9,1x** (8,5x без внутригрупповых займов) — чрезвычайно высоко. Покрытие процентов по EBIT ниже 1x (~0,75x в среднем за 3 года), по EBITDA ~1,01x — прибыли едва хватает на обслуживание долга. Операционный денежный поток отрицателен 2019–2023 гг. Финансовый долг ~6 млрд ₽, из них свыше 65% краткосрочный — высокая потребность в постоянном рефинансировании *(факт)*. Ликвидность: текущая ~1,24x, быстрая ~0,76x — без запаса *(факт)*.

**Флаги риска *(суждение)*:** (1) отозванный рейтинг + неконсолидированная отчётность РСБУ; (2) покрытие процентов на грани 1x; (3) **put-оферта 19.06.2026** — близкая контрольная точка, после которой будущие купоны не определены; (4) цена ~84% номинала и неполное размещение (320 тыс. из 1 млн бумаг) указывают на слабый спрос.

*Источники: Эксперт РА (ruB 21.02.2024; отзыв 17.02.2025); Boomin; MOEX. По состоянию на 10.06.2026. Профиль эмитента — общий для всех его облигационных выпусков.*

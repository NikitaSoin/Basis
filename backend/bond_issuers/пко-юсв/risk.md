## ВЕРДИКТ {light:gray}
Это флоатер (купон КС+6%), а не фикс — системные YTM 21,6% и G-спред 865 б.п. артефактны, светофор-премия к ОФЗ-кривой неприменима; кредитное качество эмитента — спекулятивно-уязвимое (BBB-.ru, экстремальный леверидж, анонимный поручитель, отрасль под ужесточением 230-ФЗ). По состоянию на 11.06.2026.

## АРИФМЕТИКА
- Тип купона: **плавающий, КС ЦБ + 6,0 п.п.** (первый купон 24% при КС 18%; при КС 14,5% текущий купон ≈ 20,5%) {факт} — для флоатера фактическая плата за риск = **спред купона к КС = +6,0 п.п.**, а не G-спред к ОФЗ.
- Системный «фактический спред» **+8,65 п.п. при YTM 21,61%** {оценка} — артефакт расчёта YTM фикс-методом по флоатеру; некорректен, не используется в вердикте.
- Требуемый спред: для группы BBB- / коллекторы (МФО-аналог) ориентир по премии флоатера к КС **~5–7 п.п.**, N в группе мал (флоатеры BBB- коллекторов единичны) → деградация до «рейтинг + сектор-аналог», точность ограничена. {оценка}
- Премия: формально спред купона +6,0 п.п. лежит в верхней части ориентира группы, но сравнение по флоатерам ненадёжно → опираемся на проверку потерями и качественные блоки. Светофор-премия не строится.

## ГЛАВНЫЕ АРГУМЕНТЫ
- **Коэффициент автономии 0,05; долг/капитал 12,71 (2024, по РСБУ)** {факт} — бизнес финансируется на 95% заёмными, собственный капитал почти не покрывает обязательства → при любом ухудшении сбора долгов или росте КС эмитент мгновенно неплатёжеспособен (блок A=5).
- **Выручка +341% до 624,8 млн ₽, чистая прибыль лишь +29% до 25,1 млн ₽ (2024)** {факт} — взрывной рост портфеля не конвертируется в прибыль, маржа тонкая → денежного буфера на обслуживание долга почти нет.
- **Поручитель не раскрыт (анонимный, рейтинг BBB-)** {факт} — держатель кредитует «кота в мешке»: качество обеспечения непроверяемо, при дефолте recovery неопределим (блок E риск повышен).
- **Ужесточение 230-ФЗ и расширение полномочий ФССП с 01.09.2026** {факт} — должник вправе полностью запретить взаимодействие; ФССП получает контроль над частотой контактов → прямой удар по эффективности взыскания, то есть по самому денежному потоку эмитента (блок D).

## ПРОВЕРКА ПОТЕРЯМИ
PD для группы BBB-/спекулятивный профиль с ICR на грани ~3–5% в год × LGD ~70% (необеспеченный по сути, поручитель анонимен) ≈ **2,1–3,5 п.п.** минимально необходимого спреда. Спред купона +6 п.п. к КС формально покрывает ожидаемые потери и даёт запас на неликвид/нервы. НО: PD по матрице для дебютанта 2021 г.р. с D/E 12,7 и отраслевым регуляторным шоком, вероятно, занижена — консервативно якорь ближе к границе покрытия. {оценка}

## ВЕРДИКТ {light:orange}
Флоатер КС+3,5% с потолком 18%: оценочная YTM ~14,5–15,5% (от прогноза ЦБ по ставке) даёт спред к ОФЗ всего ~+2…+3 п.п. — для эмитента группы B (рейтинг отозван по инициативе самого эмитента, ICR 1,72x) этого мало: риск недоплачен. По состоянию на 13.06.2026.

## АРИФМЕТИКА
- **Тип купона: плавающий (флоатер), КС + 3,5%, но не более 18%.** При текущей КС 14,5% формула даёт 18,0% — **потолок активирован**; при росте КС спред к КС сжимается, при снижении КС купон падает вместе со ставкой. {факт}
- **Режим A (оценочная YTM от прогноза ЦБ).** Среднесрочный прогноз ЦБ (24.04.2026): средняя КС **2026 ~14,0–14,5%**, **2027 ~8–10%**. Подстановка в формулу: купон 2026 ≈ min(14,25+3,5; 18) = **17,75%** (под потолком), 2027 ≈ min(9+3,5; 18) = **12,5%**. Блендированный ожидаемый купон по жизни бумаги ~14–15%. {оценка}
- **Оценочная YTM при цене ~101,1% и дюрации ~1,28 г ≈ 14,5–15,5%** (judgement, не факт: будущие купоны зависят от траектории КС; при более высокой КС в 2026–2027 — ближе к верху диапазона, при быстром снижении — ниже). {оценка}
- **Фактический спред к ОФЗ** сопоставимой дюрации (~1,3 г, ОФЗ ~12,5–13%): **+2…+3 п.п.** Системные YTM 19,53% и спред +665 б.п. посчитаны как для ФИКСА — **методически неверны для флоатера**, не использую. {оценка}
- **Требуемый спред:** группа B / производство продуктов питания / ВДО без действующего рейтинга. Деградация группы → «только рейтинг B» (малая выборка по отрасли), ориентир по фиксам B-сегмента **~7–11 п.п. к ОФЗ**, +1 п.п. за отсутствие рейтинга, +1–1,5 п.п. за неликвид → **~9–13 п.п.** {оценка}
- **Премия = +2,5 − ~10 ≈ −7,5 п.п.** → риск существенно недоплачен. Дополнительно: купонный спред к КС всего +3,5 п.п. с потолком — узкий для группы B. {суждение}

## ГЛАВНЫЕ АРГУМЕНТЫ
- **Спред оценочной YTM к ОФЗ ~+2…+3 п.п. против требуемых ~9–13 п.п. для группы B** {оценка} — флоатер с потолком 18% не транслирует кредитную премию ВДО → рынок (на тонкой ликвидности) платит как за качество выше реального → риск недоплачен (блок F).
- **Рейтинг B(RU) отозван АКРА 25.06.2025 по инициативе самого эмитента** (приостановка оплаты услуг), без подтверждения {факт} — эмитент группы B добровольно ушёл из-под надзора → потеря независимого мониторинга, ранний сигнал нежелания раскрываться (блок C/F).
- **ICR (EBITDA/проценты) = 1,72x за 9м2025** {факт} — операционная прибыль покрывает проценты с тонким запасом; при сохранении высокой КС банковский долг дорожает (купон облигаций упёрся в потолок, но прочий долг — нет) → запас тает (блок A).
- **FCF отрицательный −107 млн руб за 9м2025 при долге 399 млн** {факт} — бизнес обслуживается за счёт долга → зависимость от рефинансирования, БО-02 гасится ~август 2026 (блок A).

Оба якоря (группа и ожидаемые потери) сходятся: риск недоплачен 🟠. {оценка}

## 4. Оценка справедливой стоимости

**Метод NAV_PB:** NAV 344.78 руб./акцию. Холдинговый дисконт 35-60%.

- Базовый (50%): 172 руб.

**Метод DDM (g=0):** DPS 14.33 руб.
- Ke = Rf = 14.6% (квазиоблигация): 98 руб.
- Ke = 23.9% (CAPM + 3% illiquidity): 60 руб.

**Итоговый коридор:**
- conservative: 98 руб. (DDM по Rf)
- base: 172 руб. (P/NAV 0.50x)
- optimistic: 224 руб. (P/NAV 0.65x)

Текущая цена 899.5 руб. → P/NAV=2.61x → значительно выше всех оценочных уровней.

## 5. Ключевой вывод

Рыночная цена GAZS не имеет фундаментального обоснования. Холдинг-оболочка без операций с единственным непрозрачным активом (пакет санкционного банка) должна торговаться с дисконтом к NAV, а не с 161%-й премией. Вероятные причины: крайняя неликвидность 3-го эшелона, нерыночные сделки внутри аффилированных структур.

Достоверность оценки: низкая. Первичная отчётность недоступна, NAV = оценка Газфонда, дивидендная история ненадёжна (только 1 год из 5). Коридор широкий (98-224 руб.) — честное отражение неопределённости.

*Применены: NAV_PB (базовый), DDM (дополнительный). Не применимы: DCF, historical_PE, relative_peers, CAPM.*

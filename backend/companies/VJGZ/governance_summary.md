# Корпоративное управление — ПАО «ННК-Варьеганнефтегаз» (VJGZ)

*as_of: 2026-06-01 · data_quality: low · контроль: частный мажоритарий*

## 1. Кто контролирует

**Факт.** Компанию контролирует **ООО «ННК-Холдинг» с долей 93,89%** (с 29.12.2020).
Конечный бенефициар — **Эдуард Худайнатов** (экс-глава «Роснефти»). Актив передан
«Роснефтью» структурам ННК в обмен на Пайяхское месторождение: «Роснефть» вышла из
зрелых, обводнённых и дорогих в добыче активов ХМАО.

> ВАЖНО: это НЕ «Роснефть». Многие справочники по инерции показывают старую
> структуру («РН Холдинг 85,34% + Novy Investments 8,55%») — это данные ДО декабря
> 2020 года. Сегодня контроль частный, не государственный.

**Капитал — два класса акций:**
- **Обыкновенные (VJGZ)** — 18 091 110 шт., 1 голос, ~88,6% капитала. Дивиденды по
  ним фактически не платятся.
- **Привилегированные (VJGZP)** — ~2,32 млн шт. (оценочно), ~11,4% капитала.
  Дивиденд по уставу формульный: **10% чистой прибыли / число префов**, с правилом
  выравнивания (если по обыкновенной заплатят больше — преф подтягивают до неё).

**Вывод для миноритария.** Free float ~6%, мажоритарий держит 93,89% — миноритарий
без рычагов влияния. Бумага малоликвидна, 3-й эшелон. При доле >93% структурно
высок риск принудительного выкупа (squeeze-out при 95%).

## 2. Дивиденды

**Факт.** Предсказуемой дивполитики по обыкновенным акциям нет — выплат фактически
нет. По префу формула есть, но практика разрежена: подтверждённая выплата —
**2019 г., 3,02 ₽** на преф (доходность ~0,9%), затем многолетняя пауза; по итогам
2025 СД рекомендовал по обыкновенным — ноль, по префу — символические **0,5 ₽**.

**Оценка.** Формульный дивиденд по префу привязан к прибыли, которая искажена
трансфертными ценами группы (РСБУ показывает прибыль, МСФО — убыток). Поэтому база
дивиденда занижена и непредсказуема.

**Чего ждать.** Слабая и нерегулярная дивидендная отдача; по обыкновенным акциям —
рассчитывать на дивиденд не стоит.

## 3. Отношение к миноритариям

**Прецеденты.** (1) Смена контроля 2020 — пакет ушёл «Роснефти»→ННК без публичной
сопоставимой оферты миноритариям обыкновенных акций. (2) Ключевой канал ущерба —
**продажи нефти связанным сторонам группы ННК по трансфертным ценам**: маржа и
прибыль выводятся внутри группы, что бьёт по базе дивиденда префа и по стоимости
для миноритария.

**Тренд.** Скорее ухудшение защиты: из госструктуры «Роснефти» актив перешёл к
частному холдингу с сильной внутригрупповой зависимостью и минимальным free float.

## 4. Управление и прозрачность — вывод

**Сильные стороны:** бенефициар раскрыт (цепочка ННК-Холдинг → Худайнатов понятна);
по уставу есть формульный дивиденд по префу.

**Главные риски:** вывод прибыли через связанные стороны (high); сверхконцентрация
контроля + риск squeeze-out (high); слабая дивидендная защита (medium); низкая
ликвидность и информационная асимметрия — МСФО за ряд лет не сводится, число
префов не подтверждено первоисточником (medium).

**Оценки (1–5):** прозрачность владения **4**, защита миноритария **2**,
дивидендная стабильность **2**, независимость совета **2** (состав не раскрыт,
низкая уверенность), раскрытие **2**. **Overall — 2.**

**Общий вывод.** Структура владения прозрачна, но по сути это малоликвидная дочка
частного холдинга с внутригрупповой зависимостью: маржа выводится через связанные
стороны, дивидендов по обыкновенным нет, по префу — редкие и заниженные. Для
частного инвестора — актив с высокой управленческой асимметрией и слабой защитой
миноритария.

*Где не уверен: число префов и точная формульная сумма дивиденда по префу (2019
плохо сводится с формулой); состав совета директоров не раскрыт — оценки board
поставлены по факту концентрации контроля.*

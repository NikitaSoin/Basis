# Управление — Русагро (AGRO / RAGR)

**Вердикт: weak (слабое).** Итоговый governance-балл **2,9** из 5 (норма сектора — 3,0), с активным «красным флагом» деприватизации, который сам по себе тянет премию к ставке к потолку. Управление перешло из режима «контролируемый основателем агрохолдинг» в режим реализованного госриска.

**Почему такой балл.** *Факт:* в мае 2026 контрольный пакет (~65,4%) основателя В. Мошковича обращён в доход государства по антикоррупционному иску за 5 дней; основатель и экс-гендиректор под стражей; дивиденды не платятся с 2021 (рекомендация за 2024 не исполнена после ареста, за 2025 — отказ при наличии прибыли); обязательной оферты миноритариям не ожидается; заявлен дополнительный иск >550 млрд ₽ потенциально к самой компании. Эти факты тянут вниз пять факторов из восьми (защита миноритариев, стабильность дивидендов, контролирующий акционер, независимость совета, прозрачность/корпдействия — все по 2). *Оценка:* балл удержан на 2,9, а не ниже, тремя «чистыми» факторами — нет разрыва голос/капитал по классам акций (4), нет квазиказначейских пакетов (5), нет доказанного вывода FCF/tunneling (3, дивиденды исторически из прибыли). *Вывод для миноритария:* главный нерешённый вопрос — будет ли госсобственник делиться прибылью; выхода по справедливой цене (оферты) нет, возврат капитала полностью зависит от воли государства, конечная структура контроля не определена.

**Связь с оценкой.** *Суждение:* при балле в зоне «российского хвоста» (2,9, у нижней границы нормы) и активном override-триггере деприватизации governance-премия к ставке должна стоять у верхней границы диапазона — финанс-слой считает её по `config/governance_mapping.json` от `overall_score` + активных red_flags (число в `governance_discount.premium_to_wacc_pp` субагентом не зашивается). Практический эффект: повышенная ставка дисконтирования → ниже справедливая цена; дивидендный метод оценки неприменим (выплат нет). Дешевизна по мультипликаторам — это плата за реализованный риск владения, а не недооценка.

## Замечания по чек-листу

1. **(п. 1) Юаневый контур в двух каналах** (B — поток, A — хвост) без явной декларации не-пересечения в double_count_check. **Критичность: существенное.**

2. **(п. 4) Декомпозиция рыночного дисконта не сходится арифметически** — 0,43×(1−0,24−0,055)≈0,30 ≠ 0,18 фактическая цена, остаток ~28пп не назван остатком. **Критичность: существенное.**

3. **(п. 5) Сверхатрибуция — весь убыток-2022 (613 млрд) приписан гео-каналу**, хотя валютная переоценка и шок ставки/резервов (К2) тоже участвовали; C08 применён только к 2024→2025, не к 2022. **Критичность: существенное.**

4. **(п. 8) БЛОКЕР — главное число отчёта (~5-6пп дисконта, единственная материальная величина ГЕО-контура) не имеет собственного якоря**, целиком унаследовано от экспертной прикидки ИНСТ. Ни одного исторического спреда VTBR-к-аналогам не приведено. **Критичность: блокер.**

5. **(п. 9) Несоответствие якорям — систематический паттерн «скоринг по статусу против скоринга по экспозиции»** (E1=5 без выполнения клаузулы якоря 5, E4/E10 взвешены по худшему каналу, а не по материальности, в отличие от E2). **Критичность: существенное.**

6. **(п. 11) Доли causal_attribution без тегов факт/оценка/confidence/метода.** **Критичность: косметика.**

7. **(п. 14) Событийное окно SPO −6% без контроля рынка** (не вычтено движение IMOEX в тот день). **Критичность: косметика.**

**Чисто:** п.2 (макро-ставка образцово), п.3 (границы с ИНСТ), п.6 (обратная причинность), п.7 (survivorship с оговоркой), п.10 (nirvana), п.12 (нет данных ≠ нет проблемы), п.13 (свежесть барометра), п.15 (ретроспектива).

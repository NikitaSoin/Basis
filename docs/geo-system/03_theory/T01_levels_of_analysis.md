# T01. Уровни анализа (Уолтц): дисциплина атрибуции причин

**Слой:** знания/методология. Опора: Уолтц «Человек, государство и война» (три образа), «Теория международной политики» (структурный реализм), Сингер (level-of-analysis problem). Роль: теоретический фундамент шага 3 процедуры ГЕО-МАКРО; главный предохранитель от смешения причин разных этажей.

## 1. Суть

Причины международных событий живут на трёх уровнях: **ЛПР** (убеждения, психология, горизонт конкретных людей), **государство** (режим, коалиции, внутренняя экономика), **система** (анархия + распределение силы: структура вознаграждает одни стратегии и наказывает другие независимо от намерений). Уолтц-теоретик добавляет главное ограничение: структурная теория предсказывает **повторяющиеся паттерны** (балансирование, подражание успешным), но НЕ конкретные решения конкретных государств — для них нужны нижние уровни. Отсюда рабочее правило: уровень причины определяет её **инерционность**: системные причины переживают смену лиц и режимов; государственные — смену лиц; личностные — волатильны и завязаны на W1.

## 2. Применение

Война 2022 в трёх атрибуциях: системная (сдвиг баланса и расширение западного порядка на постсоветское пространство — реалистская трактовка; инерционна: не исчезнет со сменой персоналий), государственная (динамика режима, потребности правящей коалиции — T06/T07; для внутрироссийского слоя оценку даёт ИНСТ-система), личностная (убеждения и горизонт ЛПР — исторические тексты, окружение). Честный вывод — многоуровневый с весами [ОЦЕНКА], и веса — это прогноз: чем больше вес системного уровня, тем меньше событие W1 (транзит) меняет траекторию конфронтации; чем больше личностного — тем больше. Тот же метод для контрагентов: политика США по РФ — сколько в ней системного (сдерживание ревизиониста, транзит силы с КНР — T02), сколько государственного (T17), сколько личностного (стиль конкретной администрации).

## 3. Алгоритм

Шаг 1. Для каждого крупного события/тренда — три строки атрибуции (система/государство/ЛПР) с весами и confidence. Шаг 2. Проверка на подмену: не объясняется ли системное давление «личностью» (дешёвое объяснение) и не маскируется ли решение конкретных людей «объективной необходимостью». Шаг 3. Из весов — вывод об инерционности: что из драйверов переживёт смену лиц/выборы/транзит. Шаг 4. Прогноз чувствительности: перечисли, какие события какого уровня способны изменить траекторию (выборы — государственный; W1 — личностный; сдвиг сил США-КНР — системный).

## 4. Сквозной мини-пример

Жёсткая ДКП 2024–2026. Системный уровень: война и санкционная изоляция → фискальный импульс + шоки предложения → инфляция (инерционно: пока идёт S3). Государственный: экстрактивная консолидация, монополизация, огосударствление → структурная инфляция (зона ИНСТ-системы, их макроконтур — импортируется, не дублируется). Личностный: консерватизм руководства ЦБ (волатильно: кадровое решение меняет только эту компоненту). Следствие для прогноза ставки: смена главы ЦБ [гипотетическая] сдвинула бы траекторию слабее, чем ждал бы «личностный» аналитик, — доминируют верхние уровни. Веса [ОЦЕНКА]: система ~50%, государство ~35%, ЛПР ~15%.

## 5. Анти-паттерны

Редукция к одному уровню («всё решает один человек» / «всё предопределено структурой» — обе позиции убивают прогноз). «Системность» как отговорка от анализа акторов. Смена уровня внутри одного рассуждения без маркировки. Использование личностного уровня без наблюдаемых индикаторов (превращается в психологизацию — T13 §5). Забыть, что уровни взаимодействуют: структура задаёт меню, выбирают из меню люди.

## 6. Шаблон вывода

«Событие/тренд: … Атрибуция: система …% (…), государство …% (…), ЛПР …% (…) [ОЦЕНКА, conf.]. Инерционность драйвера: … Чувствительность траектории: к выборам …, к транзиту …, к системным сдвигам … Следствие для G…/S…: …»

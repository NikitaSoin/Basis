# K13. СПБ Биржа (02.11.2023): смерть бизнес-модели за один акт

**Статус:** развёрнутый кейс. Антипод K16 по готовности (T15-пара).

## 1. Паспорт
Тип: SDN на инфраструктуру внешнего контура — реализация бинарного риска. Период: 02.11.2023 → хвосты (иски, заморозки) открыты. Оси: E1=5 (реализация), G7 (якорь-ступень), C03 (кастодиальные цепочки), T15 (отсутствие плана Б). Роли: бэктест-эпизод №4; якорь «бинарного профиля» C07.

## 2. Хронология [ФАКТ]
Бизнес-модель до удара: главный российский шлюз к ИНОСТРАННЫМ бумагам (акции США, с 2023 — разворот на Гонконг) — вся ценность = мост во «враждебную» инфраструктуру. 02.11.2023 — OFAC вносит СПБ Биржу в SDN. Торги иностранными бумагами остановлены немедленно; возобновления не произошло. Клиентские активы, сидевшие в цепочке «СПБ Банк → вышестоящие иностранные депозитарии», заморожены (масштаб — предмет оценок и исков, порядок — миллиарды долларов розничных денег [ОЦЕНКА]). Лицензия OFAC на разблокировку (до 31.01.2024) проблему клиентов не решила — вышестоящие звенья перестраховались (overcompliance, C02 §2). 2024–2025 — акция теряет кратно от доударных уровней ступенями [ОЦЕНКА]; попытки новой модели: площадка для российских размещений, цифровые активы — замещение на порядок меньшей экономики. Пис-опцион: в эпизодах мирных ожиданий (K20/K24) бумага — самый волатильный «лотерейный билет» рынка (разблокировка = воскрешение модели).

## 3. Механика по каналам
Двухэтажный удар: (этаж 1) эмитент — модель, на 100% состоящая из моста в юрисдикцию санкционера, имеет **бинарный** профиль: не «дисконт за риск», а «работает/мертво»; (этаж 2) клиенты — заморозка активов через кастодиальную цепочку: риск ДЕРЖАНИЯ бумаги оказался отделён от риска самой бумаги (иностранные акции никуда не делись — доступ к ним умер). Контраст готовности с K16 (Мосбиржа два года строила юаневую альтернативу): у СПБ плана Б не существовало — «гонконгский разворот» 2023 сменил витрину, не юрисдикционную механику (те же западные звенья в цепочке) → T15-альфа готовности заякорена этой парой.

## 4. Рыночная динамика
Каскадное, а не одномоментное падение: рынок неделями переоценивал глубину (сначала «переживут», потом масштаб клиентских заморозок) — при бинарных исходах цена сходится к правде ступенями отрицания. Ретейл-составляющая (сотни тысяч клиентов) добавила социально-регуляторное измерение (реакция ЦБ, иски) — прецедент для оценки политической цены будущих инфраструктурных ударов.

## 5. Развязка
Модель-1 мертва; компания существует в поиске модели-2; клиентские заморозки — многолетний юридический процесс. Терминальная ценность = пис-опцион + остаточная лицензионная инфраструктура.

## 6. Уроки для шкал
1. **Бинарные профили не усредняются** (C07 анти-паттерн №1 — родом отсюда): экспозиция «всё на одном мосте» скорится сценарной парой (жив/мёртв × вероятность), а не средним баллом.
2. **Кастодиальная цепочка — отдельная строка риска** (закреплено в C03 §5): «оценил бумагу — оцени владение бумагой»; для любых квазииностранных инструментов в российском контуре — обязательный вопрос red-team.
3. **Витрина ≠ юрисдикция:** смена ассортимента (Гонконг) без смены инфраструктурных звеньев риск не снижает — проверять маршрут, не меню.
4. **Пис-опционная классификация:** бумаги-«лотереи» (стоимость = вероятность разблокировки) выделяются в E14 отдельным классом с запретом DCF-ложной точности — только сценарная оценка.

## 7. Ex-ante тест
T=01.11.2023: корректное состояние — E1=4,5 (риск публично обсуждался с 2022; мост в юрисдикцию санкционера = первоочередная мишень), флаг бинарности, флаг клиентской цепочки, T15-минус за отсутствие плана Б → «держание = ставка на неудар, не инвестиция». Дата — дискреция санкционера (честно непрогнозируема). Проверка агента: выдал ли бинарную структуру вместо среднего балла.

## 8. Перенос
На всё, чья ценность = мост во внешнюю инфраструктуру (криптошлюзы, депозитарные сервисы, платёжные мосты): бинарная разметка + цепочка + план Б. Зеркально в S1: этот же класс — максимальная пис-бета с механикой мгновенного воскрешения (пара к K29 §6.5).

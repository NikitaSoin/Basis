# ВТБ Факторинг — доходность и риск (флоатеры)

_as_of 2026-06-13. Разбор по методике v1.3, Режим A (флоатер). Эмитент: ООО «ВТБ Факторинг» — факторинговая дочка группы ВТБ (госбанк). Анализируемый выпуск: 001P-01 (RU000A10BVY2), флоатер КС+1,43%, погашение 20.08.2026._

## ВЕРДИКТ {light:green}
Короткий флоатер квазигоса (AA(RU), группа ВТБ) с погашением через ~2 месяца: спред КС+1,43% при цене 101,3% адекватен высокому кредитному качеству, риск оплачен.

## АРИФМЕТИКА
- Тип купона: **флоатер**, ставка купона = КС ЦБ + **1,43 п.п.**, ежемесячные выплаты {факт}
- Базовая ставка (прогноз ЦБ 2026): средняя КС **~13,3–14,5%** (web 04–05.2026) {оценка}
- Оценочная YTM: при цене 101,3% и погашении 20.08.2026 рыночная YTM ≈ **8,3%** к погашению — артефакт near-maturity (премия к номиналу «съедается» за ~2 мес.), для оценки риска нерелевантна {оценка}
- Фактический спред (флоатер, верно): **спред купона к КС = +1,43 п.п.** минус цена выше номинала → эффективный спред к КС ≈ **+1,0…+1,2 п.п.** на остаточном сроке {оценка}
- Требуемый спред для AA(RU) / финансовый институт / <1 г: **~0,8–1,5 п.п.** к базовой ставке (группа AA, квазигос) {оценка}
- Премия: **~0** — справедливо/в рамках группы; запас даёт господдержка {суждение}

## ГЛАВНЫЕ АРГУМЕНТЫ
- **AA(RU), прогноз «стабильный» (АКРА, 25.03.2025)** {факт} — рейтинг опирается на очень высокую вероятность поддержки со стороны группы ВТБ (госбанк, системная значимость) → кредитное качество близко к материнскому, риск дефолта на горизонте бумаги минимален → блок A=2.
- **Остаточный срок ~2 месяца до погашения 20.08.2026** {факт} — дюрация ~0,2 г, чувствительность к КС и кредитный горизонт минимальны → даже сценарий резкого роста ставки почти не затрагивает держателя.
- **Флоатер КС+1,43%** {факт} — процентный риск переложен на эмитента: купон автоматически следует за КС → для держателя нет риска переоценки тела при движении ставки (главное преимущество флоатера в фазе неопределённости по ДКП).
- **Принадлежность к группе ВТБ** {суждение} — факторинговый бизнес фондируется и контролируется материнским госбанком; вероятность того, что группа допустит дефолт публичного выпуска дочки, крайне низка (репутационный риск для госбанка).

## ПРОВЕРКА ПОТЕРЯМИ
PD для группы AA ~0,1–0,3%/год × LGD ~70% ≈ **0,1–0,2 п.п.** минимум. Спред купона +1,43 п.п. кратно перекрывает ожидаемые потери. С учётом остаточного срока ~2 мес. накопленная PD ничтожна. Якоря (группа + EL) согласованы → 🟢. {оценка}

## ТРИГГЕРЫ
- Прохождение погашения 20.08.2026 по номиналу → закрытие позиции без событий (базовый сценарий) → 🟢

## БЛОК A — Платёжеспособность {score:2}
- **A1 Долговая нагрузка.** Факторинговая компания: «долг» — это фондирование портфеля уступленных требований; структура баланса принципиально иная, чем у промкорпората. Финансирование преимущественно от группы ВТБ. Достаточность капитала и качество портфеля — ключ. Детальной публичной МСФО-отчётности эмитента в открытом доступе не найдено — оценка опирается на рейтинговое суждение АКРА. {оценка}
- **A2 Покрытие процентов / маржа.** Факторинговая маржа = ставка клиенту − стоимость фондирования; в флоатерной модели обе ноги привязаны к КС, процентный риск хеджирован структурно. {суждение}
- **A3 Ликвидность и стена погашений.** Доступ к фондированию группы ВТБ; для анализируемого выпуска — погашение через ~2 мес., риск рефинансирования держателя не касается (деньги возвращаются по номиналу). {факт}
- **A4 Качество портфеля.** Факторинговый портфель — краткосрочная дебиторка под уступку; риск концентрации на дебиторах. Конкретных данных по просрочке нет — нет данных. {оценка}
- **A5 Анти-рисование.** Не РСБУ-кейс; капитал и поддержка верифицированы рейтингом АКРА. Признаков «рисования» не выявлено. {суждение}

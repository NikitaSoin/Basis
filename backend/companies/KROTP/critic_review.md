# Критик-ревью карточки KROTP (ПАО «Красный Октябрь», преф)

## 1. ВЕРДИКТ ЦЕЛОГО
**5/10 — качественные вкладки сильные и связные, но финансовый блок развалился на две несовместимые версии: текст-витрина и JSON противоречат друг другу по цене, балансу и справедливой стоимости. Пока это не сведено, цифровое ядро карточки нельзя показывать.**

## 2. КАРТОЧКА КАК ЦЕЛОЕ

**Связность (в целом хорошая).** Бизнес-модель → Рынки → Макро → Гео → Управление образуют по-настоящему сквозную нить и держат один и тот же тезис: это не история операционного роста, а «актив на балансе» (московская недвижимость на Берсеневской) при низкомаржинальном сырьезависимом производстве. Какао как главный драйвер себестоимости честно проведён через четыре вкладки (бизнес-модель → рынок → макро → гео — каждая под своим углом, без дублирования). Дивиденд-как-формальность из управления согласован с «ценность в активах, а не в потоке». Это сильная сторона.

**КРИТИЧЕСКОЕ противоречие — финансы рассинхронизированы сами с собой.** `financials_summary.md` (витрина, текст) и `financials.json` (то, что считалось при v2-перепрогоне 13.06) описывают РАЗНУЮ компанию по числам:

| Параметр | financials_summary.md | financials.json |
|---|---:|---:|
| Текущая цена | **448 ₽** | **1143 ₽** |
| Балансовая стоимость (BVPS) | 1086,6 ₽ | 1160,8 ₽ |
| P/B текущий | 0,41× («торгуется 0,41 BV») | 0,98× |
| P/E как метод | `not_meaningful` | `ok`, fair 1551 ₽ |
| Коридор справедливой цены | 652 / 869 ₽ | 1102 / 1310 / 1551 ₽ |
| Вывод | апсайд +94% от цены 448 | коридор 1102–1310 от цены 1143 |

Это не нюанс — это два взаимоисключающих инвест-вывода на одной вкладке. Витрина говорит «преф торгуется за 0,41 баланса, апсайд почти двойной»; JSON говорит «цена 1143, P/B ~1,0, апсайд скромный». Summary устарел (датирован 01.06, JSON пересчитан 13.06 по financial-analyst v2), но именно summary рендерится во фронт. **До переписывания summary под актуальный JSON финансовая вкладка вводит инвестора в заблуждение.**

**Дыра в логике префа.** В summary дисконт префа к обычке (KROTP/KROT ≈ 0,40) был ядром тезиса «большой апсайд». В новом JSON BVPS и P/B считаются уже на уровне эмитента (9 217 583 акций — это, судя по всему, общий счёт, не префовый), и явного префового дисконта в коридоре 1102/1310/1551 не видно. Нужно решить и явно показать: применяется ли вообще дисконт за преф/миноритарный/ликвидностный статус, и какова цена именно KROTP против общего BVPS. Сейчас это потеряно при перепрогоне.

**Ценность вкладок.** Лишних нет. Гео при минимальной экспозиции честно сведена к одному каналу (какао из Западной Африки) — оправдана. Управление несёт главную предупредительную ценность (контроль холдинга, дивиденд ~0%, риск squeeze-out) и хорошо связано с «не покупай ради потока». Слабое звено по наполнению — финансы (см. выше).

## 3. ПО ВКЛАДКАМ

**Бизнес-модель — сильно.** Честная разметка «оценка/факт», корректная оговорка про фингод (30.04), реалистичная структура расходов. Слабость: мини-P&L здесь даёт EBITDA −0,48 млрд и ЧП 0,004 млрд за ФГ2024/2025, тогда как financials_summary даёт по тому же году EBITDA +161 млн, ЧП +93 млн, а JSON — ЧП 101 млн. Три блока — три разных числа прибыли последнего года. Нужна одна цифра.

**Финансы — критично (см. раздел 2).** Отдельно: ND/EBITDA 18,4 как «артефакт» помечен честно — хорошо. P/E 117 / EV/EBITDA 86 названы неинформативными — правильно. Но сам факт, что JSON затем строит fair value по historical_pe (1551 ₽) от «нормализованных 2022–2023», прямо конфликтует с текстом summary, где P/E объявлен `not_meaningful`. Методологическая несогласованность внутри одной вкладки.

**Рынки — сильно.** Размер рынка, позиция «challenger» (а не лидер — важная честная поправка против соблазна записать бренды в лидерство), отдельный «второй рынок» недвижимости. Оговорки прозрачны.

**Макро — сильно.** Формат «факт / как доходит / суждение» выдержан. Разворот какао увязан с маржой; слабый спрос как ограничитель — корректный противовес. Не дублирует рынок (там — структура рынка, здесь — режим/ставка/курс).

**Гео — адекватно объёму.** Не раздуто; главная уязвимость (поставки какао, один регион) названа прямо.

**Управление — сильно.** Самый ценный предупредительный блок: контроль «Гуты»/«Объединённых кондитеров», дивиденд-формальность, ограниченное раскрытие, риск для миноритария. Хорошо стыкуется с asset-кейсом из финансов.

## 4. ФАКТИЧЕСКИЕ ОШИБКИ / ПРОТИВОРЕЧИЯ
1. **Цена бумаги: 448 ₽ (summary) vs 1143 ₽ (JSON).** Расхождение в 2,5 раза — критично.
2. **Коридор справедливой цены: 652/869 (summary) vs 1102/1310/1551 (JSON).** Несовместимы.
3. **P/B: 0,41× (summary) vs 0,98× (JSON).** Меняет весь тезис «дёшево к балансу».
4. **P/E-метод: `not_meaningful` (summary) vs `ok` 1551 ₽ (JSON).**
5. **Прибыль последнего фингода: −0,004 / +0,161 / +0,093 / +0,101 млрд** в business_model vs summary vs JSON — четыре несогласованных значения.
6. **Префовый дисконт** (ядро старого тезиса) не виден в новом коридоре — потерян при перепрогоне.

## 5. ЧЕГО НЕ ХВАТАЕТ ДО ЭКСПЕРТНОСТИ
- **Свести финансы в одну версию:** переписать `financials_summary.md` строго под актуальный `financials.json` (цена 1143, BVPS 1161, коридор 1102–1551), убрать устаревшие 448/652/869/0,41.
- **Восстановить и явно показать префовую логику:** цена KROTP против общего BVPS, применяемый дисконт за преф/миноритарность/ликвидность — иначе непонятно, почему оценивается общий капитал, а покупается преф.
- **Унифицировать прибыль последнего фингода** во всех трёх блоках.
- **Численно зацепить главный актив:** во всех вкладках недвижимость «может недооцениваться балансом», но нигде нет даже грубого диапазона рыночной стоимости. Хотя бы порядок (площадь × ставка/м²) превратил бы asset-кейс из лозунга в оценку — сейчас сквозной тезis карточки количественно не подкреплён.

## Финансы эмитента: сможет ли РВК-Инвест расплатиться

Вопрос «сможет ли расплатиться» для регулируемого водоканала упирается в стабильность тарифной выручки и умеренный долг — у группы оба фактора исторически в норме. По данным Эксперт РА (на 31.12.2020): выручка 27,6 млрд ₽, EBITDA-маржа ~23%, чистая прибыль 2,87 млрд ₽, активы 36,6 млрд ₽. Долговая нагрузка низкая: **Долг/EBITDA 1,3х, покрытие процентов 7,8х** *(факт/оценка)*. Для эмитента с регулируемым денежным потоком это комфортный профиль.

Свежие данные по юрлицу скромнее: чистая прибыль ООО «РВК-Инвест» за 2024 г. — 542,8 млн ₽ (−5,91% г/г) — это показатель отдельного юрлица-держателя, не консолидированной группы *(факт)*. Полной свежей отчётности публично нет *(данных нет)*.

**Рейтинг** *(важный нюанс)*: кредитный рейтинг самой компании **отозван 25.07.2024** без подтверждения; при этом **рейтинг выпуска облигаций 001Р-01 действует — ruA+, подтверждён 17.07.2025** *(факт)*. Отзыв корпоративного рейтинга при живом рейтинге выпуска — флаг, снижающий прозрачность *(суждение)*.

**Особенность выпуска:** флоатер (КС + 2,75%), есть put-оферта (август 2026) — процентный риск низкий, доходность считается к оферте. Основной риск — регуляторный (тарифы) и инфраструктурный (износ сетей), а не острый дефолтный.

*Источники: Эксперт РА (ruA+ по выпуску, 17.07.2025; корп. рейтинг отозван 25.07.2024); cbonds; MOEX. По состоянию на 10.06.2026. Профиль эмитента — общий для всех его облигационных выпусков.*

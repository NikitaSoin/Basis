# T16. Лестницы эскалации и деэскалации: пороги, фокальные точки, off-ramps

**Слой:** знания/методология. Опора: Герман Кан (концепт лестницы эскалации), Шеллинг (фокальные точки, saliences), современная литература об управлении эскалацией и «порогах» (thresholds). Роль: теоретический каркас для прикладной лестницы B04 и для чтения любой конфликтной динамики (включая удары вглубь, морские инциденты, ядерную сигнализацию). Пара к T04: там — язык сигналов, здесь — карта местности, по которой сигналят.

## 1. Суть

Эскалация — не рубильник «мир/война», а **лестница дискретных ступеней**, разделённых порогами. Пороги держатся не физикой, а **фокальными точками** (Шеллинг): взаимно очевидными рубежами, которые обе стороны без переговоров опознают как «здесь проходит черта» — государственная граница НАТО, тип вооружения (конвенциональное/ядерное), география (своя территория/чужая), статус цели (военная/гражданская, энергетика/командование). Три следствия. (1) **Пороги пересекаются осознанно**: случайный дрейф через фокальную точку редок; пересечение — это решение, то есть информация (T04: дорогой сигнал). (2) **Условность вероятностей**: P(следующей ступени) резко меняется после каждого пересечения — каждый пройденный порог удешевляет следующий психологически и политически; считать ступени независимыми — грубая ошибка. (3) **Эскалационное доминирование**: сторона, у которой на следующей ступени преимущество, может эскалировать «в свою пользу»; но обладание доминированием ≠ намерение — путать запрещено.

Эскалация бывает **вертикальной** (интенсивность/тип средств на том же театре) и **горизонтальной** (география, новые участники, новые домены — море, космос, кибер, инфраструктура). Горизонтальная часто выбирается именно чтобы НЕ пересекать вертикальную фокальную точку (удары по теневому флоту вместо ударов по территории; кабели вместо ракет).

**Деэскалация — тоже лестница**, и несимметричная: спуск требует последовательности «прекращение огня → развод/буфер → верификация → политическая рамка», и каждая ступень спуска без следующей нестабильна (перемирие без верификации — T03 commitment-блокер; эмпирия — K25). Критичны **off-ramps**: заранее существующие способы отступить без потери лица; требование-принуждение без off-ramp толкает вверх, а не вниз (T04 §3.4).

## 2. Применение

**Лестница ударов вглубь (вертикальная, внутри текущей войны).** Наблюдаемая градация целей 2022–2026: военные объекты → энергетика прифронтовая → НПЗ и топливная система (кампания 2025–26, K26) → энергетика столиц → (не пройдено) командные центры/центры принятия решений. Каждая ступень имела свою «премьеру», после которой удары этого класса стали рутиной — иллюстрация условности вероятностей: рутинизация пройденной ступени + удешевление следующей. Для рынка: рутинизированная ступень перестаёт двигать цены (десенсибилизация, K19-паттерн), новая — двигает сильно.

**Морская ветка (горизонтальная).** Инциденты с кабелями и теневым флотом на Балтике 2024–2026 — классическая горизонтальная эскалация ниже порога: deniable, без фокальной точки «territory». Следующие пороги этой ветки размечены в B04 (L4–L5).

**Ядерная ветка.** Ступени сигнализации из T04 §2 — это верхняя часть общей лестницы; фокальная точка «применение» настолько сильна, что вся сигнализация живёт в подступах к ней.

## 3. Алгоритм

Шаг 1. Для каждого активного конфликта веди явную карту: текущая ступень; ближайшие пороги вверх и вниз; фокальные точки, на которых они держатся. Шаг 2. Каждое событие размечай: рутина текущей ступени / проба порога / пересечение. Шаг 3. После пересечения — пересчитай условные вероятности всей верхней части лестницы (не только следующей ступени) и обнови B02/B04. Шаг 4. Для деэскалационных эпизодов проверяй комплектность спуска (огонь→развод→верификация→рамка): некомплектный спуск получает явную вероятность relapse. Шаг 5. Ищи off-ramps: если их нет — вероятность движения вверх при том же давлении выше; появление off-ramp (посредник, формула, размен) — самостоятельный маркер вниз.

## 4. Сквозной мини-пример

Дроны в Польше, сентябрь 2025 [ФАКТ]: событие на фокальной точке «территория НАТО». Разметка: проба порога (deniable, без жертв). Ответ НАТО — ниже худших ожиданий (консультации, ПВО-усиление, без военного ответа) → информация: порог реален, но его «стоимость» для пробующего ограничена → условная P(повторных проб) вверх, P(немедленной вертикальной эскалации) вниз, ступень L2 лестницы B04 «обжита» [интерпретация — ОЦЕНКА, помечена]. Рыночное следствие: короткая риск-премия без смены сценария — что и наблюдалось. Контрпример для дисциплины: та же логика обязана была бы РЕЗКО поменять выводы при жертвах или официальном признании — граница между «пробой» и «пересечением» формализована заранее, не задним числом.

## 5. Анти-паттерны

Считать пороги стабильными (карта порогов — живой документ; каждое пересечение её перерисовывает). Строить одну лестницу (нужен веер: вертикальная, морская, ядерная, кибер — у каждой свои пороги). Путать эскалационное доминирование с намерением эскалировать. Пропускать горизонтальные ветки, глядя только на фронт. Симметричная слепота вниз: не замечать комплектной деэскалации, потому что «они всё равно врут», — модель мира из T04 должна применяться и к спуску. Ретроспективное «это было очевидно»: пробы порогов очевидны только после разметки — потому разметка обязана существовать ДО события (watchlist B03 §5).

# S04. Чёрная металлургия и уголь: секторная гео-карта

**Периметр:** Северсталь, НЛМК, ММК, Мечел, Распадская, ЮГК-класс исключён (граница K31).

## 1. Гео-профиль сектора
E2 — ключевая развилка судьбы 2022+: доля потерянного премиального Запада определила траекторию (Северсталь — мгновенная потеря ЕС/санкции-2022 → внутренний разворот; НЛМК — дольше всех держал европейские активы и слябовые квоты ЕС — исключения как «политика трещин»; ММК — исходно внутренний → наименьший гео-удар). E3 — восточное плечо упирается в Восточный полигон (T10 «сухопутный пролив»: уголь — главная жертва тарифно-провозной борьбы). G13 — китайский стальной цикл давит цены глобально (декомпозиция K29-методом: спред к мировым бенчмаркам = гео-компонента). Внутренний спрос = стройка → G12-ставка (K27) вторым каналом.

## 2. Трансмиссия
G3/G11 → рынки и активы (у НЛМК — судьба европейских заводов/квот: календарь исключений ЕС); G12 → внутренний спрос (девелоперский цикл) + фискальные качели (акциз на сталь/НДПИ-надстройки — «где толсто» посещало сектор в 2022–23); G5-аналог для угля: дисконты энергетического угля + логистическая маржа съедает FOB-цену (Мечел/Распадская: тариф РЖД × очередь полигона = операционный потолок).

## 3. Сценарная карта
S1: возврат премиальных рынков — медленный (гистерезис + защитные пошлины ЕС останутся: рынок ЕС structurally протекционистский к стали) → пис-бета сектора умеренная, ниже нарратива; главный плюс — через внутренний цикл (ставка→стройка). S2: ставочный плюс (лучший сценарий: внутренний спрос без геополитики). S3: боковик под китайским циклом и полигонными квотами; уголь — хронически слабое звено. S4: минус через спрос и логистику; рубль-компенсация частична (внутренние цены рублевые).

## 4. Мониторинг
Китайские стальные PMI/экспортные цены (G13-строка), тарифные решения РЖД и квоты полигона (уголь!), календарь ЕС-исключений (НЛМК-слябы), внутренние цены на прокат vs себестоимость, девелоперские продажи (спрос-прокси), фискальные строки по сектору в бюджетных пакетах.

## 5. Анти-паттерны / red-team флаги
Приписывание китайского цикла санкциям (обязателен спред к мировым пирам — K29 §6.2). Единый «металлургический» балл при противоположных E2-профилях. Игнор логистического потолка у угля («цена выросла» ≠ «маржа выросла» при съедающем тарифе). Пис-ралли сектора по общей корзине (умеренная бета — не Газпром). Забытая фискальная история 2022–23 (сектор в списке «толстых» при следующем цикле изъятий).

## 6. Дифференциация
ММК — внутренний чемпион: минимальный гео, максимальная ставочная бета (K27-зеркало); Северсталь — завершённый разворот: гео-шок в базе, торгуется внутренним циклом + дивидендной дисциплиной; НЛМК — остаточная европейская экспозиция = событийный риск/опцион (единственный в секторе с живой G11-строкой); Мечел — K28-профиль (долг × логистика: форма исхода определяется балансом, не сталью); Распадская — уголь+периметр (материнский Evraz-узел — E9-хвост).

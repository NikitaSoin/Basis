# НЕФАЗ (NFAZ) — Финансы и оценка (МСФО, 2020-2024)

## 1. Динамика выручки и прибыли

Выручка за 2020-2024 росла неравномерно: 27.2 → 33.1 → 24.4 → 35.2 → 42.3 млрд ₽. Провал 2022 (−26%) — переориентация госзаказа; рост 2023-2024 — восстановление автобусных и электробусных программ. Структурно выручка зависит от госконтрактов Мосгортранса и региональных программ (~70%).

**Отчётная чистая прибыль** резко падала: 767 → 554 → 378 → 105 → 33 млн ₽. Бизнес фактически на грани безубыточности.

**Нормализованная ЧП** (adjusted) после налоговой нормализации: 767 → 554 → 378 → **126** → **66** млн ₽. Основные корректировки — аномальный ETR 2023 (37.5%) и 2024 (62.0%), нормализованные к устойчивым 25%. Причины аномалий ETR в первичном отчёте не раскрыты — корректировки с достоверностью «judgement».

Мост 2023: reported 104.7 млн + 20.9 млн (избыточный налог) = adjusted 125.7 млн.
Мост 2024: reported 33.3 млн + 32.5 млн = adjusted 65.8 млн.

## 2. Маржи и структура затрат

Валовая маржа снижается: 7.3% → 5.6% → 6.4% → 5.3% → **4.3%** — сжатие из-за роста себестоимости (металл, комплектующие, труд) при жёстких ценах госконтрактов.

EBITDA-маржа: 4.8% → 3.2% → 4.4% → 1.4% → **1.7%** — ультратонкая. Любое отклонение по материалам или загрузке переводит в убыток (2025: убыток −505.5 млн, подтверждён из business_model.md).

ROS (adj): 2.8% → 1.7% → 1.6% → 0.4% → **0.16%** — устойчивый тренд к нулю.

## 3. Баланс: долг, ликвидность, капитал

Данные частично отсутствуют (баланс только 2022-2023 полный). Ключевые наблюдения:
- 2022: долг 3.65 млрд ₽, Net debt/EBITDA = 3.1x — высокая нагрузка
- 2023: долг полностью погашен (CFF −3.84 млрд), net_debt = −0.5 млрд (кэш > долга)
- Дебиторка 2023: 2.64 млрд (vs 3.44 в 2022 — инкассация); кредиторка выросла 3.1 → 4.4 млрд

Капитал рос за счёт нераспределённой прибыли: 1.56 → 2.00 → 2.10 → ~2.16 млрд ₽ (2024 оценочно). ROE adj падал: 49% → 19% → 6% → 3% — следствие роста базы при коллапсе прибыли.

По business_model.md (2025): дебиторка 16.8 млрд, кредиторка 19.7 млрд, чистые активы 1.4 млрд — оборотный капитал взлетел кратно.

## 4. Качество денежного потока

CFO нестабилен: 2023 = +4.6 млрд при EBITDA 0.49 млрд (аномалия: инкассация + рост кредиторки); 2022 и 2024 — отрицательный CFO. FCF_normalized (NOPAT + DA − capex_sust) отрицателен в 2023-2024 — компания не генерирует устойчивый свободный денежный поток.

## 6. Прогноз

Консенсус брокеров недоступен (3-й уровень, малоликвидная бумага). 2025 факт: выручка 31.4 млрд (−25.6%), убыток −505.5 млн. Прогнозный EPS отрицателен — P/E не применим. При нормализации госзаказа возможен возврат к выручке 35-40 млрд и символической прибыли, но неопределённость высока.

## 7. Справедливая стоимость

Маршрутизация: машиностроение → EV/EBITDA (относительная) + P/B + CAPM; DCF — insufficient_data (FCF нормализованный < 0); P/E — not_applicable.

| Метод | Результат | Статус |
|---|---|---|
| DCF | ~114 ₽ (справочно) | insufficient_data — implied EV/EBITDA 0.6x = артефакт |
| Исторический P/B | **291 ₽** | ok — якорь |
| CAPM (12м) | **325 ₽** | ok |
| Относительная EV/EBITDA | **548 ₽** | ok — оптимистично |
| Дивидендная | — | not_applicable (выплат нет) |

**Коридор: 291 — 308 — 548 ₽** (conservative — base — optimistic).
- Conservative = P/B 1.08x × BVPS 269 ₽ = 291 ₽ (исторический якорь)
- Base = среднее P/B + CAPM = (291 + 325) / 2 = 308 ₽
- Optimistic = Relative EV/EBITDA при переоценке к peers 5.5x = 548 ₽

Текущая цена 258.8 ₽ — на ~11% ниже conservative. Оценка условная: data_quality medium, 2025 год убыточен, peers = judgement.

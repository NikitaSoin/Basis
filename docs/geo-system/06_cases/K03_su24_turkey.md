# K03. Су-24/Турция (11.2015–2016): точечный конфликт и «помидорная дипломатия»

**Статус:** развёрнутый кейс.

## 1. Паспорт
Тип: двусторонний военно-политический инцидент между прагматичными игроками с быстрой нормализацией. Период: 24.11.2015 – лето 2016 (хвосты до 2017–18). Оси: G13 (региональный слой), российские контрсанкции как инструмент (зеркало C02), T04 (принуждение с off-ramp). Роли: эталон ОБРАТИМОСТИ; шаблон конфликтов РФ с «дружественными/нейтральными».

## 2. Хронология [ФАКТ]
24.11.2015 — Турция сбивает российский Су-24 у сирийской границы. Ответ РФ (указ 28.11): продовольственное эмбарго (томаты — символ и главная статья), запрет чартеров и продажи туров, заморозка «Турецкого потока» и АЭС «Аккую», ограничения для турецких подрядчиков и рабочей силы. Июнь 2016 — письмо Эрдогана с выражением сожаления (согласованная face-saving формула) → поэтапная нормализация: туризм и стройка — быстро, «Турецкий поток» перезапущен (соглашение окт-2016), томаты — дольше всех (до 2017–18, под давлением российского агролобби — импортозамещение успело окупиться).

## 3. Механика по каналам
Учебная демонстрация T04: российские меры — **принуждение** (требование извинений), калиброванное по болевым точкам асимметричной зависимости (туризм и агроэкспорт — чувствительные статьи Турции) при сохранении стратегических связок (энергетика заморожена, не разорвана — мосты для деэскалации). Ключ развязки — заранее существующий **off-ramp**: формула «сожаления» позволила обеим сторонам объявить победу. Внутрироссийский слой: контрсанкции создали своих бенефициаров (тепличный бизнес), которые затем ЛОББИРОВАЛИ сохранение эмбарго — санкции рождают группы интересов, тормозящие их отмену (зеркальный урок к западному гистерезису T09 §1.5: механика универсальна).

## 4. Рыночная динамика
Точечная: туроператоры и авиаперевозка — удар; продуктовый ритейл/агро — выигрыш замещения; широкий рынок — малочувствителен. Полезный контраст масштаба: конфликт с региональной державой без санкционной коалиции ≠ системный шок.

## 5. Развязка
Полная нормализация за ~год по политическому слою; Турция затем стала ключевым «нейтральным» хабом эпохи 2022+ (параллельный импорт, туризм, зерновая сделка) — быстрота разворота 2016 предсказала транзакционность турецкой позиции.

## 6. Уроки для шкал
1. **Обратимость точечных конфликтов** между прагматиками — при наличии off-ramp; шаблон деэскалации: болевые меры + сохранённые мосты + face-saving формула (вход в T16 §1 деэскалационную ветку).
2. **Контрсанкции создают внутренних бенефициаров-лоббистов** отмены/сохранения — учитывать при прогнозе снятия ЛЮБЫХ мер (в обе стороны).
3. **Асимметричные зависимости — карта болевых точек** (T08-метод на двустороннем масштабе).
4. **Транзакционность партнёра** — измерима историей его разворотов; турецкий профиль калиброван этим кейсом.

## 7. Ex-ante тест
После 24.11 система обязана была: классифицировать по T04 (принуждение, ограниченные цели), спрогнозировать «меры по асимметричным точкам + сохранение энергомостов + нормализация при формуле извинения в горизонте ~года» — против алармистского консенсуса «разрыв с Турцией». Реализовалось по всем пунктам.

## 8. Перенос
На эпизоды трения РФ с «дружественными» (Азербайджан, Казахстан, ОАЭ-комплаенс): читать через ту же тройку «болевые точки / мосты / off-ramp»; сигнал системности вместо точечности — если бьют по мостам (энергетика, стратегические проекты), а не по помидорам.

## 9. Источники и метки
Указ 583 (28.11.2015) и постановления правительства, хроника нормализации 2016. Отраслевые эффекты — [ОЦЕНКА] по отчётности туроператоров/ритейла.

# S07. Транспорт и логистика: секторная гео-карта

**Периметр:** Совкомфлот, НМТП, ДВМП (FESCO), Аэрофлот, Транснефть (как инфраструктура), Глобалтранс-класс (ж/д операторы).

## 1. Гео-профиль сектора
Сектор = физическое воплощение трансмиссии: его выручка — это чужие E3. **Морская ветка:** Совкомфлот — K29 (гранулярные SDN, спред-термометр); НМТП — черноморский узел (Босфор/КТК-соседство, B04 §2, дроновые риски портов); ДВМП — бенефициар восточного разворота (контейнерный дефицит полигона = ценовая власть). **Воздушная:** Аэрофлот — эталон T12-деградации (каннибализация парка, лизинговые «национализации»-2022 как E9-прецедент, закрытое небо ЕС/США = маршрутная география war-режима) и одновременно топ пис-корзины (открытие неба = мгновенная переоценка). **Трубопроводная:** Транснефть — K30-фискальный якорь + K26-операционный след (удары по ПНС). **Ж/д:** операторы = производная Восточного полигона (T10: дефицит ниток = ставки аренды вагонов).

## 2. Трансмиссия
G5/C05 → фрахтовая экономика (санкционная премия↔оковы — двусторонность K29 §6.3); G1 → физические риски узлов (порты Черноморья, ПНС — некомпенсируемый военный капекс K26 §6.4); B04 L4–L5 → экзистенциальный хвост балтийско-черноморского плеча; G3 → лизинг/страхование/классификация судов и бортов; G13 → мировые циклы фрахта (контрольная группа обязательна).

## 3. Сценарная карта
S1: сектор-чемпион пис-корзины — Аэрофлот (небо), Совкомфлот (делистинг судов = механическое воскрешение, K29 §6.5), НМТП (страховая нормализация); встречный ветер: конец санкционных премий и «дефицитных» ставок (ДВМП-класс теряет ценовую власть — внутрисекторная РОТАЦИЯ, не общий рост!). S2: умеренный плюс без структурных починок. S3: жизнь на трении (кто-то его платит — кто-то собирает); Транснефть — тарифно-фискальные качели. S4: L4+-хвосты доминируют; Аэрофлот — мобилизационные риски парка/неба.

## 4. Мониторинг
Поимённые листинги судов (доля дедвейта — C05 §4), спреды ставок к мировым (K29-термометр), Восточный полигон (провозные решения, тарифы), страховые war-risk премии Черноморья/Балтики (первый индикатор B04 L4), лизинговые иски/урегулирования (Аэрофлот-класс), физические инциденты узлов (K26/W7-строки).

## 5. Анти-паттерны / red-team флаги
«Всё из-за санкций» без мирового цикла (K29-провал). Пис-ралли сектора единым блоком (ротация внутри: бенефициары трения ПРОИГРЫВАЮТ от мира — знак у ДВМП-класса противоположен Аэрофлоту!). Игнор двусторонности премии (сегодняшний сборщик трения — завтрашняя мишень листинга). Транснефть как «нефтяная бета» (она — регулируемо-фискальная история K30). Оценка Аэрофлота без T12-строки (деградация парка = растущий скрытый капекс).

## 6. Дифференциация
Совкомфлот — двухфакторная машина (санкции×цикл): вечный полигон C08; Аэрофлот — максимальная механическая пис-бета при худшем S3-профиле (идеальный сценарный инструмент); НМТП — рентный узел с военно-страховым хвостом; ДВМП — анти-пис-бета логистического трения (уникальный знак в секторе); Транснефть — фискально-тарифный регулируемый профиль (S10-родство); ж/д операторы — производная дефицита ниток: инфраструктурные решения государства = их P&L.

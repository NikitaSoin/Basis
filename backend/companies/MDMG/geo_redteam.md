## Замечания

1. **(п.2/4, БЛОКЕР) Канал A: +1,5 п.п. системный пол поверх рыночной ставки, при собственном выводе «компанейская дельта ≈ 0».** C06 §2 ограничивает канал A компанейской дельтой, запрещает системную «премию за войну» поверх рыночной ставки. Если Rf финслоя — наблюдаемая рублёвая (ОФЗ), системный гео-риск уже в ней — двойной счёт с макро-ставкой. «Консервативность» — не аргумент. Тест: выяснить конструкцию ставки финслоя.

2. **(п.1/3, существенное) Фактор «риск роста налога на прибыль после выборов 09.2026» не попал ни в одну таблицу трансляции ни одной системы** — назван в E12 и в S3 (базовый сценарий), но не оценён владельцем нигде. Обратная сторона двойного счёта — нулевой счёт.

3. **(п.4, существенное) Канал C: «−3-5%» не заякорено историческим спредом**, ИНСТ оставил развилку «либо уже в общерыночном мультипликаторе, либо ГЕО» — ГЕО взял себе, не разрешив развилку.

4. **(п.9, существенное) E14 без обязательной пары чисел** (качественные слова вместо Δ), плюс внутреннее противоречие — описание соответствует якорю 1, выставлен балл 2.

5. **(п.5/8, существенное) causal_attribution: разложение без метода по каналу, арифметика не показана** — при отрицательном К5 сумма должна превышать 100%, остаток не назван. Тест К1 формально пропущен.

6. **(п.11, существенное) Ключевые числа сорсятся на блог-агрегатор (wtfinvest.ru), не на МСФО/IR** — «инфляция медуслуг ~18%» (главный драйвер, 50% исхода) вообще без источника.

7. **(п.8, существенное) Канал B «−0,5% FCF» подозрительно точно равен арифметическому остатку хейркат ИНСТ**, а не независимо оценённой величине.

8. **(п.9, существенное) E8=2 при описании, соответствующем якорям 3-5** («рублёвая выручка при частично импортном капексе, натурального хеджа нет») — активная фаза capex (Лапино-3, Домодедово, >10 млрд/год) требует явной защиты через долю FX в капексе, которой нет.

9. **(п.9, косметика) E6 rationale дословно якорь 3, выставлено 2,5; E11 получил числовой score вопреки правилу «только флаг».**

10. **(спецпроверка, косметика) Из цитаты handoff опущены sector_flags и watchlist_30d.**

11. **(п.7, косметика) Абсорбция навеса выведена на единственном позитивном кейсе (X5/K12)** без базовой ставки/контрпримера.

12. **(п.15, косметика) Миграция legacy без Δ-строки выводов.**

13. **(вне чек-листа, косметика) Синонимия «Инерция ИНСТ = S3 ГЕО» заявлена как факт**, хотя разные сценарные сетки с разными вероятностями не согласованы.

**Чисто:** эндогенности нет, nirvana fallacy нет, отсутствие данных корректно деградировано, сценарии с триггерами.

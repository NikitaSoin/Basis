# K24. «28 пунктов» → Женева/Абу-Даби (11.2025–02.2026): вторая волна и её затухание

**Статус:** развёрнутый кейс.

## 1. Паспорт
Тип: вторая большая волна мирных ожиданий — с документом на столе. Период: 11.2025 – 02.2026. Оси: G2, T05 (ратификационная геометрия — разобрана там на этом кейсе), E14 (точка №2 ряда затухания). Роли: калибровка коэффициента доверия пис-беты; демонстрация «документ ≠ вероятность».

## 2. Хронология [ФАКТ/поиск]
Ноябрь 2025 — утечка «28-пунктного плана» (американо-российская рабочая заготовка: территориально-гарантийно-санкционный пакет с активами G8 в размене); шок союзников → экстренные женевские консультации США–Украина–европейцы, украинско-европейская контрверсия (сокращённый список пунктов). Декабрь — челночная фаза (переговорщики в Москве/столицах). 23–24.01.2026 — Абу-Даби: раунды с участием всех треков; территориальные вопросы впервые обсуждаются предметно на бумаге. 17–18.02.2026 — Женева: сближение по части «технических» пунктов при тупике в ядре (территории де-юре, гарантии, последовательность снятия санкций). Далее — пауза, перекрытая ближневосточной эскалацией (W3 съедает дипломатическую полосу пропускания США [ОЦЕНКА-интерпретация]); частичный мост к K25 (пасхальная инициатива). Рынок: ноябрьский импульс заметно слабее февральского-2025 (амплитуда пис-корзины и индекса — доли от K20 [ОЦЕНКА по рядам]), выдыхается к концу февраля без выраженного обвала — низкая посадка волны.

## 3. Механика по каналам
Отличие от K20 — наличие ДОКУМЕНТА: рынок получил не звонок, а структуру сделки — и всё равно оценил скромнее. Причина корректна по T05: документ без ратификаторов — черновик; геометрия win-set'ов (киевский, брюссельский, московский) не покрывала ядро пакета, и это было видно из публичных реакций недели утечки. Дополнительный слой — **усталость выборки**: после цикла-2025 рынок требует фактов ступенью выше (не «план», а «подписанный режим»; не «раунд», а «рабочая группа по санкциям») — порог доказательности сместился, что и есть механизм затухания пис-беты.

## 4. Рыночная динамика — ряд затухания
Точки ряда (амплитуда мирного импульса, нормированная): K20 (02.2025) = 1,0 → K24 (11.2025–02.2026) ≈ 0,3–0,5 → K25 (04.2026) ≈ ещё ниже [ОЦЕНКА-паттерн, точные коэффициенты — по рядам при бэктесте]. Ряд задаёт **динамический коэффициент доверия** E14: прогнозная амплитуда следующей волны = базовая бета × коэффициент, пока не случится качественно новый факт (K20 §8-правило).

## 5. Развязка
Волна осела без разрыва трека: наработки Женевы легли в основу пасхальной инициативы (K25) — процессная непрерывность при рыночном скепсисе. Абу-Даби/Женева остаются якорем деления G2=3 в B01 («территории в повестке предметно»).

## 6. Уроки для шкал
1. **Документ ≠ вероятность:** структурированный план без движения ратификаторов добавляет к P(S1) меньше, чем кажется — вес события определяется T05-геометрией, не детальностью бумаги.
2. **Порог доказательности мигрирует вверх** с каждым несбывшимся циклом — имплайд-протокол C06 §5 обязан использовать ТЕКУЩИЙ порог (маркеры, которые рынок ещё способен купить), а не февральский-2025.
3. **Ряд затухания — рабочий инструмент:** коэффициент K20→K24→K25 применяется к прогнозам всех следующих волн до появления факта нового класса.
4. **Конкурирующие кризисы съедают треки:** дипломатическая полоса пропускания великих держав конечна (W3 против украинского трека) — межтрековая связка входит в оценку темпа переговоров (стык T02 §2б-перегрузка).

## 7. Ex-ante тест
T=день утечки: корректный выход — «импульс будет и будет слабее K20 (порог доказательности + ратификационный дефицит, наблюдаемый по первым реакциям Киева/ЕС); маркеры повышения: [рабочая группа по санкциям, медиаподготовка, вето-подвижки ЕС]; их отсутствие 30–45 дней = затухание» — проверяемо целиком. Тест на дисциплину: не поднял ли агент P(S1) на сам факт существования детального плана.

## 8. Перенос
Лекало для каждой следующей «большой бумаги» (а они будут): детальность ≠ вероятность; считать ратификаторов; применять коэффициент затухания; помнить о конкурирующих кризисах. И встречное правило: первый факт нового класса (реальный акт снятия / подписанный верифицируемый режим) ОБНУЛЯЕТ ряд затухания — тогда недооценка станет главным риском (симметрия ошибок).

## Замечания

1. **(п.3, п.1) БЛОКЕР — отсутствует сборочная таблица A02 §6, конфликт «тонкий рынок» отмечен, но не разрешён.** ГЕО-канал C содержит «навес → тоньше рынок», ИНСТ-канал C — «эрозия франшизы → меньше IPO → тоньше рынок» — разные причины, одна переменная исхода. Отчёт перекладывает разрешение на «сборщика» вместо самостоятельной таблицы «фактор→владелец→канал→величина». **Критичность: блокер.**

2. **(п.1) Неполная таблица «эффект→канал»** — «рыночный навес» без строки в valuation_translation; потенциальный пересчёт хвоста S4 (мета-риск в канале C поверх уже взвешенных сценарных FCF). **Критичность: существенное.**

3. **(п.6) Эндогенность несущего аргумента** — «умеренная реакция акции на SDN → удар мал» частично объясняется тем, что маргинальные продавцы (нерезиденты) были заморожены с 2022, то есть ценовой сигнал сам подавлен санкционной инфраструктурой. На этом висят E1=3.5, канал A≈0, «справедливо в цене». **Критичность: существенное.**

4. **(п.9) E1=3.5 противоречит букве якоря 3** («без US SDN» при полном US SDN у MOEX) — балл в делении, чьё определение прямо противоречит фактам; нужен явный флаг дыры шкалы, не молчаливая адвокатура. **Критичность: существенное.**

5. **(п.2, п.8) Внутреннее противоречие Δ-оценок E14** — дельты декларированы «сверх макро-эффекта ставки», но внутри учтён противоход ЧПД (который и есть эффект ставки) — риск двойного счёта при сборке с финслоем. **Критичность: существенное.**

6. **(п.11) Гигиена фактов** — источник датирован будущим (08.2026 вместо 08.2025, видимо опечатка); ВЭБ 9,4% (гео) vs 8,4% (ИНСТ) — конфликт не размечен; «~180 млрд EUR» НРД без источника; имя акционера расходится между файлами (Fundament/Fundamental Capital). **Критичность: существенное (первые два), косметика (остальное).**

7. **(п.4, п.8) Канал C (3-5%) без количественного якоря** — качественная отсылка вместо спреда мультипликатора вокруг события-якоря. **Критичность: существенное (нижняя граница).**

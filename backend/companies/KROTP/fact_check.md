### (4) Синтез-вердикт по ПРЕФУ
KROTP — **актив-кейс без денежного потока к держателю префа**. Дивидендный метод (основной для префа) = not_applicable: дивдоходность ~0.03%, уставной 10%-механизм де-факто не работает (governance-вывод прибыли в холдинг). Апсайд только событийный (монетизация/девелопмент недвижимости, выкуп/squeeze-out по справедливой цене) с низкой вероятностью; базовый сценарий — длительный «спящий» актив.

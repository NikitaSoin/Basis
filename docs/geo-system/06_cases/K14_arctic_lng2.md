# K14. Арктик СПГ-2 (2023→): актив построен — монетизация парализована

**Статус:** развёрнутый кейс.

## 1. Паспорт
Тип: санкции на ПРОЕКТ (не на материнскую компанию) с технологическо-логистическим удушением. Период: 02.11.2023 → продолжается. Оси: G6×E5 (технологии), E3 (флот), T10 §4/T12 §4 (перемножение), рост-премия НОВАТЭКа. Роли: якорь режима «актив без монетизации» для C06; демонстрация точечности нового поколения санкций.

## 2. Хронология [ФАКТ]
02.11.2023 — OFAC вносит «Арктик СПГ-2» (проектную компанию) в SDN — материнский НОВАТЭК не блокирован: хирургическая точность (удар по будущим объёмам при сохранении действующего «Ямал СПГ» — мировой рынок СПГ не шокирован, T09-калибровка боли). Декабрь 2023 — форс-мажор покупателям; иностранные акционеры (TotalEnergies, CNPC/CNOOC, японский консорциум) замораживают участие/офф-тейк. Т1 линия технически готова (конец 2023) — отгрузок нет 8+ месяцев: нет газовозов Arc7 (корейские верфи разорвали, «Звезда» — сдвиги вправо; T12-корзина «передовое»). Август–октябрь 2024 — попытка теневого экспорта: несколько отгрузок судами-«призраками» → суда и структуры немедленно листингуются, груз частично «плавает»/в хранение — покупателей нет. 2025 — просачивание: первые приёмки в китайских терминалах (авг-2025 и далее — эпизодические [паттерн]) — фаза адаптации с жёстким потолком. Т2/Т3 — сдвинуты/под вопросом. НОВАТЭК: переоценка роста — котировка теряет десятки процентов от пиков 2023 [ОЦЕНКА], рост-премия схлопывается.

## 3. Механика по каналам
Перемножение (T10 §4): свободная арктическая география (свой выход в океан!) × нулевая доступность ледовых газовозов = нулевой маршрут — географический и технологический баллы НЕ усредняются. Точечность: санкционер отделил «будущий рост» от «текущих потоков» — эталон калибровки T09 (боль без самоповреждения рынка). Теневая адаптация в СПГ структурно тяжелее нефтяной: судов Arc7 в мире — единицы против тысяч танкеров, каждый — на виду; потолок обхода задан физикой флота, не смелостью трейдеров (контраст с K10).

## 4. Рыночная динамика
Классика переоценки growth-историй: удар пришёлся не в текущий FCF (Ямал работает), а в терминальную стоимость и пайплайн — мультипликаторная компрессия при внешне стабильной отчётности; sell-side долго нёс «Т2 запустится» в моделях — санкционная ревизия допущений отставала от фактов на кварталы [ОЦЕНКА-паттерн; red-team-урок].

## 5. Развязка
Открыта; равновесие — «одна линия с эпизодическим китайским сбытом ниже мощности»; развилки: флот (каждый сданный Arc7 — событие), Т2-достройка своими силами, судьба долей иностранных акционеров (заморожены — E10-зеркало для НИХ).

## 6. Уроки для шкал
1. **Режим «актив без монетизации»** — отдельный класс в C06: DCF растущих проектов при G6≥4 обязан нести сценарий «построено ≠ продаётся» с вероятностью, а не сноской.
2. **Перемножение E5×E3** закреплено якорем: узкое место одной корзины обнуляет ценность остальных — агрегирование C07 помечает критические связки.
3. **Точечные проектные санкции** — растущий класс (дешевле для санкционера): E1 эмитента может оставаться умеренным при E1=5 его ключевого проекта → скоринг по сегментам, не только по юрлицу-эмитенту.
4. **Физический потолок адаптации:** где обход требует уникальных активов (Arc7, литография), фазовая модель T09 получает жёсткую асимптоту — не экстраполировать нефтяную гибкость на всё.

## 7. Ex-ante тест
T=03.11.2023: корректный выход — «паралич вероятнее обхода: судов нет и не будет в горизонте; Т1 без отгрузок квартал(ы); рост-премию НОВАТЭКа резать сейчас» — против консенсуса «найдут танкеры, как нашли для нефти». Корзиночный разбор T12 давал ответ механически; проверка V01 — воспроизводит ли агент асимптоту.

## 8. Перенос
На весь СПГ-пайплайн (Мурманский СПГ и др.) и любые технологически-узкие проекты роста (глубокая нефтехимия, авиапрограммы): сегментный E1 + корзинки T12 + вопрос «что физически повезёт/произведёт». Зеркально в S1: снятие проектных санкций монетизируется быстро (актив-то готов) → высокая пис-бета сегмента при низкой — текущего потока.

## 9. Источники и метки
Релизы OFAC (02.11.2023 и суда-2024), раскрытия/заявления НОВАТЭК и партнёров, отраслевые трекинги отгрузок [паттерн]. Котировочные величины — [ОЦЕНКА].

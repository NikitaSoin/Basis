### 2. Сигналы конвейера (✓ согласованы)
- ✓ governance: is_pref=true, контроль **государства 72.31%** (Росимущество с 29.08.2025, по суду — конфискация активов М. Каитова); `governance_quality_flag=weak`; дивиденды РФ-устав платятся ТОЛЬКО по префам (обычка DPS=0 за 2021–2025).
- ✓ related_party: исторически орбита активов МРСК СК/Каитова (основание конфискации); после перехода к государству характер связанных сделок может измениться — внутригрупповые потоки не раскрыты (data_flag), прямого вывода ликвидности не подтверждено.
- ✓ macro: overall_effect **supportive**, main_risk **маржа**; ключ ставка 14.5% в цикле снижения. ВАЖНО для префа: часть прибыли = процентный доход на чистой денежной позиции (−1.89 млрд) при высокой ставке → снижение ставки ужмёт этот компонент и, вероятно, дивиденд.
- ✓ geo: exposure **minimal**, overall_effect **neutral** — внутренний энергосбыт, без экспорта/валюты; гео транслируется только через прибыль и ликвидность. Согласовано с macro.
- ✓ market: монополист-гарантирующий поставщик Ставрополья (~55% объёма, ~90% территории), регулируемая сбытовая надбавка.

### 3. Противоречия
- (a) **Цена/число акций префа в meta были от обычки** — главный дефект, устранён (см. п.1).
- (b) **Блок valuation.methods (P/E, P/B, EV/EBITDA, CAPM) построен на обычке** (859 млн, 3.842 ₽, DPS=0). Для префа НЕ якорные — помечены справочными (оценка бизнеса в целом). Добавлен ведущий метод `dividend_yield_pref`.
- (c) **Дивиденд 2025 аномально высок** (+76% к 2024): рост прибыли РСБУ + процентный доход на кэше. Нормализован при расчёте FV (0.649 ₽) во избежание recency-смещения.

### 4. Синтез-вердикт по ПРЕФУ
Преф STSBP = квазиоблигация с плавающим купоном (10% ЧП РСБУ / 262.1 млн), главный довод владения — гарантированный уставом денежный поток, которого обычка лишена (DPS=0 при бесполезном голосе под госконтролем 72.31%).
- **Дивдоходность:** 9.0% (на 2024 div) → **15.9% (на объявленный 2025 div 0.828 ₽)** к цене 5.215 ₽.
- **Спред к обычке:** STSBP 5.215 ₽ vs STSB 5.12 ₽ → преф с **премией +1.9%** (нетипично для РФ-неликвида, но объяснимо: только преф даёт денежный поток).
- **Справедливая цена (по дивдоходности, req-yield 16% база):** **4.06 ₽** (коридор 3.61–5.71 ₽). Текущая цена 5.215 ₽ ВЫШЕ базы — рынок капитализирует разовый всплеск дивиденда 2025; при нормализации (снижение ставки → меньше процентного дохода → меньше ЧП РСБУ → меньше купон) поддержка цены ослабевает.
- **Governance_discount:** высокий (заложен в req-yield-спред +1.4пп к ОФЗ): госконтроль, непрозрачность, теоретический риск пересмотра уставной формулы префа голосованием мажоритария; related-party — без подтверждённого вывода ликвидности, но история конфискации повышает неопределённость.
- **Достоверность:** medium (нет ОДДС, повреждён первичный PDF, малоликвид). Пробелы помечены, не домыслены.

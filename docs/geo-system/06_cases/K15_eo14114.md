# K15. EO 14114 (22.12.2023): вторичный поворот — страх работает вместо ударов

**Статус:** развёрнутый кейс.

## 1. Паспорт
Тип: экстерриториальная угроза банкам третьих стран. Период: 22.12.2023 → действует; расширение 12.06.2024. Оси: G4 (якорь перехода 2–3 → 4), C02 (вторичный тип), T08 (panopticon+chokepoint на чужих банках). Роли: якорь принципа «поведение исполнителей > тексты актов»; предтеча K17.

## 2. Хронология [ФАКТ]
22.12.2023 — исполнительный указ: OFAC получает право вносить в SDN **иностранные финансовые организации** за проведение/содействие значимым транзакциям, связанным с российским ВПК (по определённому перечню секторов/товаров). Январь–март 2024 — волна де-рискинга без единого применённого наказания: турецкие банки массово закрывают счета российских компаний и «релокантов», эмиратские ужесточают онбординг, китайские крупные — стопорят платежи (разворачивается в K17). 12.06.2024 — тем же пакетом, что K16: определение «военно-промышленной базы РФ» расширено практически на всю подсанкционную экономику → периметр страха вырос без нового «удара». Далее — режим постоянного фонового давления: точечные listing'и посредников поддерживают температуру.

## 3. Механика по каналам
Инновация инструмента: санкционер переложил enforcement на КОМПЛАЕНС-ОТДЕЛЫ чужих банков — им не нужно применять меру, достаточно сделать её вероятной: банк третьей страны сравнивает маржу российского клиента с риском потери долларового клиринга (chokepoint T08) — асимметрия тотальна, решение предопределено. Overcompliance (C02 §2) из побочного эффекта стал ЦЕЛЬЮ дизайна. Юридическая техника = рычаг: ширина определения (июнь-2024) масштабирует страх росчерком — мониторить надо дефиниции, не только списки.

## 4. Рыночная динамика
Прямой рыночной точки нет — эффект размазан в издержках всех импортёров/экспортёров (E4-инфляция трения: комиссии, задержки, предоплаты) и виден в отчётностях с лагом кварталов; классический «налог без даты», который событийные модели пропускают, а маржинальные мосты C08 обязаны ловить.

## 5. Развязка
Действующий режим; равновесие — двухуровневая банковская архитектура расчётов (крупные банки третьих стран — закрыты, мелкие/региональные/агенты — рабочий канал с премией) — институционализировано в K17.

## 6. Уроки для шкал
1. **G4 скорится по поведению банков, а не по актам** (закреплено в B01/T18 §4): метрики — закрытия счетов, сроки прохождения, комиссии агентов; тексты указов — лишь опережающий индикатор поведения.
2. **Дефиниционный мониторинг:** расширение определений = удар без удара; в B03-скан добавлен слой «изменения формулировок OFAC/FAQ».
3. **Угроза эффективнее применения:** ни один системный банк не был наказан — и не потребовался; прогнозируя «применят ли вторичку к банкам КНР», правильный вопрос — «нужно ли им применять» (обычно нет — страх уже работает; применение = сигнал СБОЯ страха, отдельный маркер эскалации).
4. **Порог самоограничения санкционера:** удар по системным банкам КНР граничит с финансовой войной США–Китай (T17: сопротивление Уолл-стрит, риски трежерис) — этот потолок держит G4 ниже 5 при любом фоне; его пробой = событие класса «сценарная перевзвеска».

## 7. Ex-ante тест
T=23.12.2023: корректный прогноз — «де-рискинг банков дружественных юрисдикций начнётся в недели, без применений; главные жертвы — расчёты импортёров; смотреть Турцию/ОАЭ первыми (долларозависимость), Китай — с лагом и тише» — реализовалось посекторно. Проверка: вывел ли агент механику страха из структуры инструмента, не дожидаясь фактов.

## 8. Перенос
На любые будущие вторичные конструкции (тарифные версии: «500%-законопроекты»; страховые; портовые): та же тройка «чей комплаенс исполняет / какова асимметрия для исполнителя / где потолок самоограничения». И зеркально: российские контр-угрозы третьим странам слабы ровно потому, что асимметрии-рычага нет — трезвость в оценке «ответов».

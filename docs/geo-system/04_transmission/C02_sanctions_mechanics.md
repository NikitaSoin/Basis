## 2. Смягчающие механизмы — читать так же внимательно, как запреты

**General Licenses (GL):** OFAC-разрешения на классы операций (wind-down, энергетика, продовольствие/удобрения — устойчивые carve-out'ы, T17-агролобби). Правила чтения: (а) GL с дедлайном = управляемый процесс — календарь дедлайнов в watchlist обязателен (управление продажей активов Лукойла — K23); (б) выдача широкой GL = сигнал разрядки дешевле заявлений (T04-индекс); (в) непродление ожидавшейся GL = ужесточение без нового акта. **Специфические лицензии, исключения в пакетах ЕС** (изъятия для отдельных стран/проектов — картина реальной коалиционной политики). **Overcompliance:** банки/трейдеры режут шире буквы запрета (страх вторички) → фактический периметр всегда > юридического; оценивая E1, добавляй слой «поведенческий периметр» [ОЦЕНКА по прецедентам отказов].

## 3. Правила скоринга из типологии

E1: тип меры задаёт этаж балла (SSI ≈ 2–2,5; односторонний SDN ≈ 4; мульти-юрисдикционный SDN на ядро бизнеса = 5), поведенческий периметр и GL-карта двигают в пределах этажа. G3: агрегат по покрытию экономики блокирующими мерами (якоря в B01). G4: не по актам, а по **поведению исполнителей** третьих стран (T18 §4-логика). Скорость: при новом акте немедленно фиксируй тройку «правовая форма / дедлайны GL / кодифицировано ли» — она определяет и глубину удара, и гистерезис снятия для S1 (T09 §1.5).

## 4. Анти-паттерны

Смешение SSI и SDN (ошибка уровня шкалы). Игнор 50% rule и суммирования долей (структуры владения обходят/ловят именно здесь). Чтение только запретов без GL-карты (половина информации — в разрешениях). Оценка периметра по букве без overcompliance. Пропуск дедлайнов GL (самые предсказуемые события санкционного календаря). «ЕС ввёл — ЕС снимет»: снятие требует единогласия, вводившегося по частям, — асимметрия по построению.

# Финансы и оценка — RTGZ (ПАО «Газпром газораспределение Ростов-на-Дону»)

_Дата расчёта: 2026-06-13. Данные: МСФО 2021–2025, источник smart-lab (агрегатор). data_quality = medium. Первичный PDF отсутствует._

---

## 1. Динамика выручки и прибыли

Выручка росла уверенно: 8 720 млн (2021) → 13 100 млн (2025), CAGR +10,7%. Рост обусловлен регулируемыми тарифами ФАС с инфляционной индексацией и расширением газификации.

Чистая прибыль нестабильна и систематически ниже операционной — это тревожный флаг. Операционная прибыль 2023 = 3 580 млн, но ЧП = 291 млн (разрыв 3,3 млрд). В 2024 г. — убыток -677 млн по МСФО при операционной прибыли 1 460 млн. В 2025 г. картина повторяется: ОП = 1 730 млн, ЧП = 282 млн (разрыв 1,4 млрд). Природа разрыва (налог, амортизация, обесценения, прочее) без первичного МСФО-отчёта не верифицирована.

Нормализация: adjusted = reported по всем годам — корректировки без первичного PDF запрещены методикой. Мост нулевой. Это важное ограничение: нормализованная прибыль может быть выше (если 2024 содержит разовые списания) или соответствовать реальности (амортизация капиталоёмкой инфраструктуры + высокий налог).

## 2. Баланс: долг, ликвидность, капитал

Совокупные активы выросли с 26,8 млрд (2021) до 46,6 млрд (2025) — признак активной инвестпрограммы. Капитал вырос с 18,8 до 25,5 млрд. Обязательства росли быстрее: 8,0 → 21,1 млрд; доля в активах — с 30% до 45%.

Структура долга (долгосрочный/краткосрочный, кэш) не раскрыта. net_debt = null. Это ограничивает оценку через EV.

Ощутимый капитал принят равным балансовому (НМА/гудвил не детализированы). BVPS 2025 = 212 065 ₽/акц. Текущий P/B = 0,19 — акция торгуется с 81%-ным дисконтом к балансовой стоимости.

## 6. Справедливая стоимость

### Маршрутизация методов

Регулируемая инфраструктура (ГРО, дочка «Газпром»). Применены:
- P/B (Гордон ROE/Ke) — основной якорный метод
- CAPM — требуемая доходность, implied fair value
- DCF — insufficient_data (нет ОДДС)
- Исторический P/E — insufficient_data (нет исторических цен)
- Относительная — insufficient_data (нет peers.json)
- Дивидендная — not_applicable (DPS = 0 с 2012 г.)

| CAPM (12m) | ~41 508 | ok |
| DCF | — | insufficient_data |
| Hist. P/E | — | insufficient_data |

### Расхождение методов

Расхождение P/B Гордон vs CAPM составляет ~659% (>30% — обязательная нота). Причина структурная: нормализованный ROE = 3,4% при Ke = 20,0% — компания зарабатывает в 6 раз меньше стоимости капитала. Справедливый P/B = 0,026 → ~5 470 ₽. CAPM, напротив, показывает: при прогнозном EPS 8 302 ₽ текущая цена ~40 937 ₽ соответствует требуемой доходности 20%. Два метода отвечают на разные вопросы. При систематически низком ROE P/B по Гордону — более фундаментальный якорь.

### Где оценка ненадёжна

- Без первичного МСФО-отчёта неизвестна природа разрыва OP/NP. Если 2023–2024 содержат разовые списания, нормализованный ROE выше → P/B растёт значимо.
- Бета = 0,6 — judgement (листинг 3-го уровня, нет ликвидных данных).
- Малоликвидная бумага (для квалинвесторов) — цена не отражает фундаментальную стоимость эффективно.

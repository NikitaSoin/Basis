# K05. «Кремлёвский доклад» и апрель-2018: списки-сигналы и кандидатские пулы

**Статус:** развёрнутый кейс. Парный к K04 (там — удар; здесь — предвестие и выбор жертвы).

## 1. Паспорт
Тип: сигнальный документ без санкций → выборочная реализация через два месяца. Период: 30.01.2018 – апрель 2018. Оси: G3, E13 (механика кандидатского пула). Роли: якорь правила «списки поднимают E13 ДО удара».

## 2. Хронология [ФАКТ]
30.01.2018 — Минфин США публикует доклад по разд. 241 CAATSA: 210 имён (114 чиновников + 96 бизнесменов — фактически список Forbes). Рынок и комментаторы выдыхают: «телефонная книга», санкций нет — облегчение, бумаги фигурантов отскакивают. 06.04.2018 — из пула выбраны: 7 бизнесменов (Дерипаска — K04, Вексельберг, Керимов и др.), 17 чиновников, 12 компаний → SDN. 09–11.04 — распродажа: RTS −11% за день 09.04, рубль ~−10% за неделю [уровни дня — ФАКТ, недельные — ОЦЕНКА]; Ренова Вексельберга теряет контроль в западных активах (Sulzer — экстренное разбавление ниже 50% за дни: первое боевое применение «лечения по 50% rule», предвосхитившее план Баркера).

## 3. Механика по каналам
Доклад = **создание кандидатского пула**: юридически пустой, информационно — карта будущих целей. Ошибка рынка января («формальность») — классическая недооценка дешёвого-на-вид сигнала, за которым стоит дорогая законодательная машина (CAATSA обязывала — T17: Конгресс уже связал руки администрации, вопрос был «кого», не «будет ли»). Выбор конкретных жертв из 96 — дискреция санкционера: НЕ прогнозируем поимённо (найтова компонента, честно), но пул как класс обязан был нести повышенный E13.

## 4. Рыночная динамика
Двухтактная: ложное облегчение (январь–март) → концентрированный шок (апрель) — сумма движений больше, чем при честном прайсинге с января: рынок заплатил за категориальную ошибку «нет санкций = нет риска».

## 5. Развязка
Частично в K04 (Русал-ветка); Вексельберг-ветка — потеря западного контура при сохранении российского; Керимов и др. — персональный слой без обрушения публичных компаний. Дисперсия исходов внутри одного пакета — сама по себе урок (E13 бьёт по-разному в зависимости от структуры владения и географии активов — прообраз полей E9/E10).

## 6. Уроки для шкал
1. **Правило кандидатского пула (закреплено в C02/B03):** публичный список-сигнал → +0,5–1,0 к E13 всех фигурантов немедленно, без ожидания реализации; «формальный доклад» — запрещённая формулировка.
2. **Дискреция выбора ≠ равновероятность:** внутри пула ранжирование по уязвимости (доля ≥50%, долларовый контур, западные активы) — Дерипаска и Вексельберг были верхом такого ранжирования ex ante.
3. **Двухтактный паттерн** «облегчение → удар» — поведенческая ловушка, детектируемая: облегчение без юридической развязки = ложное.
4. **50% rule работает в реальном времени** (Sulzer за дни) — скорость лечения тоже якорится здесь.

## 7. Ex-ante тест
На 31.01.2018: корректный выход — «пул создан; CAATSA делает реализацию вопросом времени и выбора; E13 фигурантов ↑; верх ранжирования: [критерии]; тайминг и имена — найтовы». Проверка агента: не поддался ли «телефонной книге».

## 8. Перенос
Прямой — на все последующие списки-сигналы (законопроектные перечни, «теневые» списки Конгресса 2025–26, украинские санкционные предложения как опережающие индикаторы западных): каждый обрабатывается как пул с ранжированием. Верхний этаж переноса: сегодняшние «кандидатские» дискуссии по ещё не заблокированным экспортёрам — та же механика (K23 §8).

# K25. Пасхальное перемирие-2026 и срыв продления: relapse-эталон

**Статус:** развёрнутый кейс с объявленными лакунами (событие свежее; поля для дозаполнения помечены).

## 1. Паспорт
Тип: ограниченный режим прекращения огня без верификации → предсказуемый распад. Период: апрель 2026. Оси: G1/G2 (движение внутри делений), T03 (commitment-блокер), T16 (комплектность спуска), точка №3 ряда затухания E14. Роли: якорь S2→S3 relapse (B02 §5); тест медиапанели T07.

## 2. Хронология [ФАКТ/поиск 11.07.2026; детализация — лакуна]
Апрель 2026 — на пасхальный период объявляется перемирие (инициатива в развитие женевско-абударбийских наработок K24); соблюдение — частичное, взаимные обвинения в нарушениях с первых суток; по истечении заявленного срока продление не согласовано — боевые действия возобновляются в полном объёме; к маю трек возвращается в паузу на фоне ближневосточной доминанты (W3) и риторики «ожидания 2027» (FT-линия [поиск]). **Лакуны, подлежащие заполнению при ревизии (V03):** точные даты объявления/истечения; параметры режима (география, классы вооружений); рыночная реакция по дням (ряды); медиапанель T07 за две недели до и после.

## 3. Механика по каналам
Хрестоматийный T03: перемирие без механизма верификации и гарантов = нерешённый commitment-блокер → нестабильность по построению; и T16: спуск без комплектности (огонь без развода сил, без мониторинга, без политической рамки) несёт высокую P(relapse) ex ante. Прогностическая ценность кейса — в его ПРЕДСКАЗУЕМОСТИ: система, размечавшая комплектность спуска, обязана была нести «продление маловероятно без появления верификационного компонента» ещё при объявлении.

## 4. Рыночная динамика
[Лакуна по рядам.] Ожидаемый паттерн по ряду затухания (K24 §4): импульс ниже K24, распад быстрее; заполнить фактическими амплитудами и включить точку №3 в коэффициент доверия E14. Пре-регистрированная гипотеза для проверки (важно зафиксировать ДО изучения рядов — дисциплина против ретроспективной подгонки): нормированная амплитуда ≤0,2 от K20.

## 5. Развязка
Relapse состоялся; след в шкалах: G2 удерживается в делении 4 («перемирия точечные и не продлеваются» — формулировка якоря B01 писана этим кейсом), G1 — без устойчивого улучшения. Процессный след: даже сорванное перемирие оставляет прецедент режима — техническая база следующей попытки (деэскалационные ступени T16 накапливаются, как и эскалационные).

## 6. Уроки для шкал
1. **Комплектность спуска — предиктор №1:** чек «огонь→развод→верификация→рамка» применяется к КАЖДОМУ будущему режиму в момент объявления; некомплектный получает явную P(relapse) в отчёте (закреплено в B02 §5, T16 §3.4).
2. **Дисконт доверия после relapse:** каждое следующее перемирие прайсится с понижающим коэффициентом — теперь с тремя точками ряда, а не декларативно.
3. **Пре-регистрация гипотез** для свежих кейсов: поля-лакуны + ожидаемые значения фиксируются до сбора данных — методологическая новация этого кейса, распространяемая на все будущие «горячие» разборы (анти-ретроспектива, Принцип 6/V01 §3).
4. **Сорванные режимы информативны о поле торга:** ГДЕ нарушалось и ЧТО не согласовали в продлении — карта реальных резервных точек сторон (вход в T03 §3.1), ценнее деклараций.

## 7. Ex-ante тест
T=день объявления: корректный выход — «режим некомплектен → P(продления) низкая; рыночный импульс ≤ K24; маркеры апгрейда: появление мониторингового механизма/третьей стороны»; T07-подтест: сопровождалось ли объявление внутренней медиаподготовкой (если нет — одноразовость почти гарантирована). Заполнение лакун покажет, где рамка сработала, где нет.

## 8. Перенос
На все последующие ограниченные режимы (энергетические перемирия, локальные прекращения огня, «жесты доброй воли»): чек комплектности + дисконт доверия + пре-регистрация. Встречное правило K24 §8 действует: первый КОМПЛЕКТНЫЙ режим обнуляет ряд затухания.

## 9. Источники и метки
[Поиск 11.07.2026] — событийная канва; первичные заявления сторон апреля 2026 — собрать при ревизии; биржевые ряды апреля–мая 2026 — собрать. Кейс помечен «частично заполненный» в K00 до закрытия лакун — честный статус вместо имитации полноты.

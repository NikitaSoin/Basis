## ⚠️ Где данные несвежие (вниманию владельца)
1. **Аналитика карточек** (бизнес-модель/финансы-ряды/governance/geo/macro тексты) — **статична** с момента последнего прогона субагентами. Обновление = ручной перепрогон. Это by design (числа в БД / текст в файлах), но «свежесть» = дата прогона.
2. **rates.csv** — старый снимок MOEX ISS. Цена отвязана (живёт в quotes), но число акций и будущие дивотсечки берутся отсюда → при старой выгрузке возможны пропуски свежих отсечек / неточное число акций у недавних доп-эмиссий. Лечится перевыгрузкой rates.csv.
3. **Прогноз ЦБ — альт-сценарии**: базовый тянется из комментария ЦБ; проинфляционный/дезинфляционный/рисковый публикуются в годовом ОНЕГДКП — извлекутся, только если есть в обрабатываемом тексте (иначе только базовый).

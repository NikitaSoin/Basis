# C08. Протокол причинной атрибуции (предохранитель №1)

## 1. Зачем

Профдеформация геополитического аналитика — видеть геополитику везде: любой убыток объясняется санкциями, любой рост — военным спросом. Уроки ИНСТ-проекта, перенесённые сюда как прецеденты ошибок: убытки М.Видео были в основном процентными расходами по долгу при КС 16–21% (опосредованный макроканал), а не доменными издержками; результаты Совкомфлота 2023–2024 двигала и глобальная нормализация фрахта (мировой цикл), а не только санкции. Сверхатрибуция даёт ложные дисконты, двойной счёт и непроходимый бэктест. Протокол обязателен: (а) в отчёте КОМПАНИИ — для интерпретации исторических результатов эмитента (шаг 4); (б) в отчёте МАКРО — для интерпретации рыночных движений («рынок упал из-за эскалации» требует проверки против ставки/нефти/ликвидности).

## 2. Алгоритм

**Вход:** наблюдаемый исход (изменение прибыли/FCF/котировки/дисконта) за период.
1. **Сформулируй исход количественно** [ФАКТ]: что, на сколько, за какой период.
2. **Составь полный список каналов-кандидатов** (минимум пять, всегда все пять групп):
   - К1 глобальный отраслевой цикл (мировые цены, фрахт, спрос) — «что было бы с идентичной нероссийской компанией?»;
   - К2 внутренний макрофинансовый (ставка → процентные расходы и спрос; курс; инфляция издержек) — опосредованный канал войны, но НЕ прямой гео;
   - К3 конкуренция и отраслевая динамика внутри РФ;
   - К4 менеджмент и идиосинкразия (M&A, авария, учётные эффекты, разовые статьи);
   - К5 прямой геополитический (санкции на эмитента/рынок/логистику, военный спрос/потери).
3. **Для каждого канала — тест величины:** оценка вклада в исход (число или порядок) с методом: сравнение с нероссийским аналогом (для К1), чувствительность к ставке × изменение ставки (К2), разложение по сегментам отчётности (К3–К5), событийные окна вокруг дат санкций (К5).
4. **Ранжируй вклады, просуммируй санити-чеком** (сумма вкладов ≈ исход? необъяснённый остаток — назвать остатком, не приписывать удобному каналу).
5. **Вывод:** «доля прямого гео-вклада в исходе ≈ X% [ОЦЕНКА, confidence, метод]». Только эта доля имеет право влиять на GRE-баллы и гео-дисконты.

## 3. Мини-пример (схема)

Исход: EBITDA экспортёра −25% г/г. К1: мировая цена товара −15% (нероссийские аналоги: EBITDA −18%) → вклад ~−12…−15 п.п. К2: КС выросла, но долг низкий → ~0; крепкий рубль периода → −4 п.п. К5: расширение дисконта после санкций на трейдера-посредника → −5 п.п. (событийное окно + спред котировок). К4: разовый резерв −3 п.п. Санити: −15−4−5−3 ≈ −27 ≈ исход. Вывод: прямой гео-вклад ≈ 5 п.п. из 25, т.е. ~20% исхода — GRE-балл двигать умеренно, а не «санкции убили компанию».

## 4. Анти-паттерны

- Начинать с гео-объяснения и искать подтверждения (подгонка).
- Приписывать К2 (ставка/курс) прямому гео-каналу «потому что война всё равно первопричина» — первопричина уже уплачена в безрисковой ставке (C06 §3); повторный счёт запрещён.
- Пропуск теста К1 (нет сравнения с мировыми аналогами) — самый частый дефект.
- «Остаток = гео» — остаток называется остатком.
- Событийные окна без контроля рынка (бумага упала в день санкций, но весь рынок упал так же).

## 5. Шаблон вывода

«Исход [метрика, Δ, период]. Разложение: К1 … / К2 … / К3 … / К4 … / К5 … (метод по каждому). Необъяснённый остаток: … Доля прямого гео: ~X% [ОЦЕНКА, conf.]. Следствия для GRE/оценки: …»

# Корпоративное управление — ПАО «Ставропольэнергосбыт» (STSB)

*Дата среза: 26.06.2026. Качество данных: low (источники — деловые СМИ и агрегаторы, первичное раскрытие требует ручной проверки). Общий флаг управления: **weak**. Итоговый балл: **2,3 / 5**.*

## Вердикт

**Управление слабое (weak).** Это де-факто госкомпания: контроль над ~72,31% голосующих акций перешёл государству (Росимущество) не рыночной сделкой, а через судебную конфискацию активов прежнего бенефициара. Миноритарий обыкновенной акции остаётся без влияния, без денежного возврата и при низкой ликвидности (листинг 3-го уровня).

## Почему такой балл (ключевые факты)

- **Факт.** Контроль 72,31% получен по антикоррупционному иску Генпрокуратуры (решение Черёмушкинского суда, 29.08.2025; активы по группе ~14,3 млрд ₽) — сработал триггер деприватизации/национализации контролирующего собственника (активный красный флаг).
- **Факт.** По обыкновенным акциям дивиденды не платятся вообще (DPS = 0 за 2021–2025). Распределяет прибыль только класс префов (STSBP, формула 10% ЧП РСБУ) — но и по нему за 2025 г., по доступным данным, выплаты не было при наличии прибыли (потенциальное отклонение от уставной формулы — второй красный флаг, требует подтверждения по ГОСА).
- **Оценка.** Реализованный исторический related-party риск (тот самый, что стал основанием конфискации) плюс непрозрачные текущие внутригрупповые потоки и неподтверждённый состав совета удерживают баллы группы «денежные потоки» и «совет/раскрытие» на уровне 2 (ниже нормы сектора 3).
- **Вывод для миноритария.** Держателю обыкновенной акции структурно не достаётся ни влияния, ни дивиденда; денежный возврат есть только в префах, и тот стал менее предсказуем после смены контроля. Выход из позиции затруднён малоликвидностью.

## Распределение баллов

Группа I (контроль и права, вес 45%): защита миноритариев 2, разрыв голос/капитал 3, квазиказначейские 3. Группа II (денежные потоки, 35%): связанные стороны 2, стабильность дивидендов 2, контролирующий акционер 2. Группа III (совет и раскрытие, 20%): независимость совета 2, прозрачность и корпоративные действия 2. Норма сектора (электроэнергетика) по каждому фактору = 3; компания ниже нормы по 6 из 8 факторов.

## Связь с оценкой

**Оценка → ставка.** Балл 2,3 (weak) при двух активных красных флагах (деприватизация контролирующего собственника; потенциальное нарушение дивполитики префов) ложится в нижний, выпуклый участок маппинга — управление повышает ставку дисконтирования и снижает справедливую цену. Конкретное число премии к ставке (`premium_to_wacc_pp`) рассчитывает финансовый слой по `config/governance_mapping.json` на основе балла и красных флагов — субагент число не зашивает. (assessment)

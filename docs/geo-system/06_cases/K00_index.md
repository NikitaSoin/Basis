# K00. Реестр кейсов (32): гео-события → российский рынок, 2014–2026

**Слой:** знания. Кейсы работают трижды (Принцип 3): обучающие примеры для агентов, якоря шкал B01/C07, материал бэктеста V01. Формат реестра: каждый кейс — код, событие с датами, тип, затронутые оси, рыночный эффект, главный урок, статус. Статус: все 32 кейса развёрнуты отдельными файлами по формату K04 (9 разделов); аннотации реестра самодостаточны для быстрого якорения шкал без открытия файла. **Правило приёмки:** «см. кейс» в отчёте агента допустимо только со ссылкой на конкретный раздел развёрнутого файла либо на строку этого реестра.

**Эпистемический стандарт:** цифры в аннотациях имеют метки; при использовании в боевом отчёте любая цифра старше версии базы перепроверяется к дате отчёта. **Оговорка смещения выборки:** библиотека смещена к громким публичным эпизодам; «тихие» адаптации (компании, незаметно перестроившие логистику/расчёты) недопредставлены → выводы вида «гео-шок всегда виден в цене за день» запрещены, survivorship-поправка обязательна (V01 §8).

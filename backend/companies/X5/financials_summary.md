## 2. Баланс, долг и аренда IFRS 16

Структура баланса X5 определяется двумя особенностями:

**Реорг.резерв -312 190 млн** — технический элемент капитала, возникший при редомициляции (2024). Балансовая арифметика корректна (assets = equity + liabilities — сверено), но балансовый P/B занижен. Ощутимый капитал (tangible equity = total_equity − goodwill − intangibles) стал положительным только с 2023: 55 млрд → 98 млрд (2025).

**Аренда IFRS 16 ~667 млрд руб.** — ключевая статья для leverage. Традиционный долг (без аренды) к 2025 = 287 млрд, net_debt_trad = 26 млрд (почти нулевой). Но с арендой: net_debt_incl_lease = 693 млрд, ND/EBITDA = 1,57x — в рамках дивидендной политики (целевой коридор 1,2-1,4x, чуть выше). EV с арендой = 1 343 млрд руб.

Ликвидность улучшается: current ratio 0,47 (2021) → 0,77 (2025). Cash + ST investments выросли с 26 млрд до 261 млрд (2025) — компания накопила подушку.

## 3. Качество денежного потока

ОДДС не извлечён (CFO/CFI/CFF = null) — главное ограничение анализа. FCF нельзя рассчитать напрямую. Единственная точка: capex PP&E 2024 = 59 554 млн (1,5% выручки). Proxy FCF 2025 ≈ 62 млрд (adj NP + DA_PP&E_est − capex_est) — широкая неопределённость.

DCF = insufficient_data: при попытке построить FCFF excl lease EV_dcf (~352 млрд) оказывается меньше net_debt_incl_lease (693 млрд) при Ke=22,25% → цена отрицательная. Это не значит, что компания стоит отрицательно — это означает, что при высокой ставке дисконтирования и IFRS16-аренде DCF требует корректного разбиения DA (lease vs PP&E) из первичного ОДДС.

## 6. Оценка справедливой стоимости

Маршрутизация: ритейл → P/E(adj) + EV/EBITDA (исторический) + дивидендная + CAPM. DCF = insufficient_data (нет ОДДС).

| Метод | Результат | Апсайд |
|-------|-----------|--------|
| DCF | insufficient_data | — |
| Исторический P/E adj (forward) | 3 479 руб. | +45,5% |
| Исторический P/E adj (backward) | 3 025 руб. | +26,5% |
| Исторический P/B tangible | 3 633 руб. | +51,9% |
| EV/EBITDA hist (outlier) | 5 841 руб. | +144% |
| CAPM 12м | 2 310 руб. | −3,4% |
| Дивидендная Гордон | 3 506 руб. | +46,6% |

**Коридор справедливой стоимости: 2 310 — 3 633 руб. Базовая цель (среднее без EV/EBITDA outlier): ~3 191 руб. (+33%).**

Расхождение методов объясняется:
1. CAPM (2310) — отражает краткосрочный ценовой возврат при дивиденде 25,6% > Ke 22,25%. Технически говорит: «весь доход уже в дивиденде, апсайда цены нет».
2. EV/EBITDA (5841) — outlier, исключён из базы: историческая средняя включает 2021-2022 при ставках ОФЗ ~8-9%.
3. Кластер P/E / дивидендная (3025-3633) сходится — это основной ориентир.

**Ненадёжность оценки:** отсутствие ОДДС делает impossible надёжный DCF; P/B метод ненадёжен из-за технического реорг.резерва; прогноз механический без консенсуса. Данный анализ — ориентир, не точный расчёт.

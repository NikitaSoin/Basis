### 🟡 Вердикт — справедливо, с лёгким уклоном в недоплату

Облигация платит ~12,7 п.п. сверх ОФЗ при риске уровня B+ (микрокэп-производитель кофе, одна площадка, тонкая ликвидная подушка). Для группы B/single-name это близко к норме, но без явного запаса; call-структура и слабая ликвидность срезают и без того скромную премию *(суждение)*.

### Природа доходности

YTM 29,18% при цене 101,17% и купоне 26,5% — цена чуть выше номинала логична (высокий купон), **не** near-offer артефакт *(факт)*. Но в выпуске есть **call-оферта с 12-го и 24-го купона (право эмитента, не put)** → корректная база — yield-to-worst: при снижении ставок эмитент гасит по номиналу, держатель теряет высокий купон, апсайд цены ограничен сверху; права досрочного выхода у держателя нет *(оценка)*.

### Арифметика (3 шага)

Фактический спред: YTM 29,2% − ОФЗ ~16,5% ≈ +12,7 п.п. *(оценка)*. Требуемый: медиана B / пищепром-FMCG микрокэп / 1,5–2,5 г (малая, деградация до «рейтинг B + потребсектор») ~10–11 п.п. + неликвид/дебют (+1) + слабая ликвидная подушка/нет линий (+0,5–1) ≈ 11,5–13 п.п. *(оценка)*. Премия ≈ 0…+0,5 п.п. → 🟡 справедливо.

### Аргументы (факт → механизм → вывод)

1. **Долг/OIBDA ~2,6 на 30.06.2025 (прогноз НКР ~2,4)** *(факт)* → для FMCG умеренная нагрузка, обслуживание НКР называет сильным → блок A средне-рисковый, не преддефолт.
2. **Низкие остатки кэша и отсутствие действующих кредитных линий** *(факт, НКР)* → нет подушки на рефинансирование; дефолты ВДО случаются на рефинансе → главный риск-фактор, оправдывает надбавку.
3. **Одна производственная площадка + низкие рыночные позиции + умеренно волатильная выручка** *(факт)* → концентрационный/операционный риск: простой одной точки рвёт поток → блок B повышен.
4. **Выручка 556,6 млн ₽, ЧП 48,7 млн (9м2025), рентабельность выше средней** *(факт)* → бизнес прибыльный, но микрокэп; импорт зелёного кофе → валютная зависимость, любой шок сырья бьёт сразу → запас прочности тонкий.

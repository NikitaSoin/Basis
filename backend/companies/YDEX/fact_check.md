### 3. Разрешение противоречий
- **DCF 1589₽ (низко) vs мультипликаторы 5059–6518₽ (высоко), >3×.** РАЗРЕШЕНО:
  при Ke 28.2% множитель Гордона сжат (implied exit EV/EBITDA 1.85x — нереалистично
  для IT с ростом 20%+); DCF = техническая нижняя граница при текущих ставках, НЕ
  база. База — мультипликаторы + консенсус брокеров 5000–7300₽. Перевешивают
  мультипликаторы; ключевая развилка — траектория ключевой ставки.
- **«Дорого» vs «дёшево».** РАЗРЕШЕНО: при ОФИЦИАЛЬНОЙ скорр. ЧП 141.4 млрд (не
  ошибочных 88) текущий P/E adj = 11.55x — для растущего на 20%+ IT это умеренно/
  недорого, а не дорого (как при прежних 18.6x). SBC-only пол 88 млрд (P/E 18.6x)
  оставлен консервативным сценарием — реальное размытие.
- **Гео-выигрыш vs макро-негатив.** РАЗРЕШЕНО: структурный сдвиг доли поиска
  (гео, наблюдаемый горизонт) перевешивает циклическое давление рекламы (макро,
  обратимо при снижении ставки).

### 4. Итоговый синтез-вердикт
Цена 4134₽; база справедливой **~5400₽** (апсайд ~30%; диапазон 3900–7300₽).
**СПРАВЕДЛИВО-К-УМЕРЕННО-ДЁШЕВО.** Качество бизнеса высокое: долга по сути нет
(ND/EBITDA 0.07x — ставка бьёт по оценке, не по прибыли), ROE 25%, FCF растёт
(136 млрд). Прибыль концентрирована на доноре поиск/реклама (~80% EBITDA);
вертикали дают оборот при тонкой марже. Governance mixed (+1пп к ставке), гео
net-positive в наблюдаемом горизонте (отложенный технориск чипов — флаг), макро
смешанный (циклический встречный ветер сейчас, попутный при снижении ставки).
DCF — нижняя техническая граница, база — мультипликаторы и консенсус. Без «купить/
продать». Уровни достоверности: числа FY2025 — факт; нормализация adj и маржа
сегментов — оценка; траектория ставки и реализация guidance — суждение.

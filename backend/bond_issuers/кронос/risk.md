### 🔴 Вердикт — риск НЕ оплачен даже спредом ~17 п.п.; это цена близости к проблемам материнской группы

Эмитент — квазикэптивная структура ПАО «ЕвроТранс» (>95% выручки от группы), а ЕвроТранс с апреля 2026 в техническом дефолте по облигациям, под исками на 10–14 млрд ₽, с рейтингом ruC (Эксперт РА) и отозванным CC.ru (НКР). ### Природа доходности

Спред такого размера для формального BB+ означает: рынок оценивает фактический риск намного выше рейтинга и закладывает вероятность перехода проблем ЕвроТранса на Кронос.

### Арифметика (3 шага)

Фактический спред: YTP 33,2% − ОФЗ ~16,3% ≈ +17 п.п. *(оценка)*. Требуемый: медиана BB / торговля / ~1 год ≈ 7–9 п.п., но по стоп-правилу группа берётся по худшей — фактический профиль ближе к B/CCC → реалистично 15–20+ п.п. *(оценка)*. Премия формально мала/отсутствует. Спред высок не из-за «недооценки», а потому что отражает преддефолтный риск группы → красный по стоп-правилам, не по арифметике.

### Аргументы (факт → механизм → вывод)

1. **>95% выручки Кроноса — контракты с группой ЕвроТранс** *(факт, НКР 03.02.2026)* → при проблемах материнской структуры выручка и поток Кроноса исчезают мгновенно → низкий собственный долг (NetDebt/EBITDA 0,12x) обманчив: кредитоспособность Кроноса = кредитоспособность ЕвроТранса.
2. **ЕвроТранс — техдефолты по облигациям (апрель 2026), Эксперт РА снизил до ruC, НКР отозвало CC.ru, иски 10–14 млрд, блокировки счетов ФНС** *(факт)* → дефолтная материнская структура не может стабильно платить «дочке» и склонна стягивать ликвидность вверх (займы связанным сторонам — канал вывода) → блоки C и D критичны, перекрывают хорошие формальные метрики.
3. **НКР 30.04.2026 изменило прогноз Кроноса со «стабильного» на «на пересмотре с возможностью понижения»** *(факт)* → формальное предупреждение о вероятном понижении вслед за группой → действующий риск-профиль уже не BB+ «стабильный» из БД.
4. **Выпуск малый (200 млн ₽), эмитент дебютный, без обеспечения** *(факт)* → неликвид (невозможность выйти при новостях) + необеспеченность (высокий LGD) → надбавка к требуемому спреду, дисконт к recovery.

### Проверка ожидаемыми потерями

Если трактовать риск как B/CCC (связь с дефолтным эмитентом тянет туда): PD для B ~5–10%, для CCC >20%; LGD ~70% → минимально при B ~3,5–7 п.п., при CCC ~14+ п.п. Спред ~17 п.п. покрывает B-сценарий, но в CCC-сценарии (реализация дефолта группы) едва покрывает EL без премии за нервы/неликвид. Якоря расходятся → консервативный вывод *(суждение)*.

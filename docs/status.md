- **Дизайн-система** (заповедник): токены, светлая (дефолт) + тёмная темы,
  кобальт-акцент, «цвет только в данных», примитивы `/_design`, читаемый дизайн
  (LeadStatement/KeyTakeaway/Disclosure), две интонации (маркетинг vs аналитика).
- **Портфельная аналитика (Этапы 0-3)**: история котировок 10 лет + бенчмарки
  (IMOEX/RTSI/MCFTR); дивиденды + total return; `company_metrics`; волатильность,
  гибрид-бета (MOEX + расчёт с поправкой Диммсона), корреляции, R², VaR,
  downside-волатильность; ОФЗ G-curve; Шарп/альфа/Сортино/CAPM; графики, секторная
  диаграмма, концентрация; индекс качества + субиндексы + интерпретация корреляций.
  Вкладки: Состав, Агрегирующая, Доходность/оценка, Риск, Корреляции, ИИ-Диагноз
  (заглушка), Стресс-тест (заглушка).
- **Облигации**: полный охват ~3100 (борды TQOB/TQCB/TQOY/TQOD/TQRD); двойной
  рейтинг (рынок по спреду + агентский smart-lab) + расхождение; типы купонов
  (фикс/флоатер/линкер/структурная); муни; дефолтные; связка выпуск→эмитент +
  блок долговой нагрузки «сможет ли расплатиться»; глубокие разборы-якоря
  (ТГК-14/Сегежа/ЕвроТранс). docs/bonds-methodology.md.
- **Фьючерсы**: срез FORTS ~560 (плечо/ГО/экспирация/срочная структура); контекст
  базового актива (рубль/нефть). docs/futures-methodology.md.
- **Фонды**: БПИФ/ETF 102 (тип/TER-в-деньги/ликвидность); разбор среза. Состав

## Очередь (приоритет — в CLAUDE.md). Детали:
1. **Облигации вглубь** — РЕАЛЬНЫЙ анализ ВСЕХ (не примеры): «сможет ли
   расплатиться» (долговая нагрузка), насколько доходность соответствует риску
   (для ВДО 30-50%), АРБИТРАЖ (расхождение цены и риска), риски мошенничества/
   руководства/споров с властью (кейсы как ТГК-14). Публичные эмитенты — долг из
   financials.json + гео/управление; ВДО без публичного эмитента — агентский
   рейтинг как оценка кредитоспособности + кредит-аналитик на ликвидный срез.
   Сортировка по реакции на ставку (дюрация) — готово.
2. **Фьючерсы вглубь** — подробный анализ базового актива («что дальше с
   валютой/рынком/сырьём» через макро/ДКП/рынок), оценка СПРАВЕДЛИВОСТИ контанго/
   бэквордации vs теоретической (cost-of-carry/паритет ставок) + АРБИТРАЖ,
   ПАРНЫЕ стратегии (акция + шорт фьючерса) — владелец подтвердил, делать
   аналитически без сигналов «купи/продай». Читаемо (рендер markdown готов).
3. **Новые классы** (РЕШЕНИЯ ВЛАДЕЛЬЦА 2026-06-09): валюта и металлы — ОТДЕЛЬНЫЙ
   класс (НЕ лёгкая вкладка); ОПЦИОНЫ — разрешены (в бэклоге); фонды углубить
   (состав, ошибка слежения, дублирование портфеля look-through).
4. **Обозреватель рынка** — 5 под-блоков (масштаб как портфель, разбить):
   (а) новости рынка; (б) макрообзор (статистика + интерпретация в духе ЦМАКП);
   (в) анализ выходящих отчётностей с приземлением в карточку (триггер пересчёта);
   (г) дивидендный календарь; (д) календарь СД/собраний/отчётностей.
5. **Ревизия карточек** — fact-checker + читаемость по всем блокам; не пересобирать
   вкладки. Требует владельца для эталонного чек-листа дефектов.
6. **Скрининг** — сортировка/фильтрация по `company_metrics`; расширяется на каждый
   класс активов. Опирается на готовое.
7. **Лендинг** — глубокая переработка (маркетинговая интонация), когда есть что
   показать.
8. **ИИ-ассистент + RAG + монетизация** — отдельное направление, с продуктовой
   постановки (чат, знающий данные Basis; регистрация, платные/бесплатные фичи).
9. **Стресс-тесты/симуляции** (компании и портфель) — в конце, опираются на всё.

## Бизнес-контекст (неизменное)
Независимый аналитический слой и «второе мнение» для самостоятельного частного
инвестора: цельная картина (компания, рынок, макро, гео, портфель) вместо тумана.
НЕ брокер, не сделки, не «купить/продать». Снижает неопределённость, не хайп.
Ценность — доверие, логика, полнота рассуждений.

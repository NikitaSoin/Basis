## 1. Динамика выручки и прибыли

Выручка: 6044 → 7172 → 7950 млн руб (+18.7% / +10.8%). Рост замедляется. Модель asset-light: почти нет себестоимости в классическом смысле, EBITDA-маржа исключительно высокая — 55.8% / 64.5% / 61.6%. Чистая маржа (ROS): 39% / 49% / 47%.

**Adjusted = reported**: разовых статей не идентифицировано — нет детализации ЧП по примечаниям. Мост пустой. ETR неизвестна (данные по налогу отсутствуют). Аномалия: CFO 2023 = 4786 млн при ЧП = 2360 млн (CFO/ЧП = 2x) — вероятно специфика движения клиентских средств (обеспечение заявок на платформе); без детализации ОДДС изолировать невозможно — флаг.

Finance_income 2024 = 1302 млн — регулярный инвестдоход (~15% совокупных доходов); риск снижения при падении ключевой ставки.

## 6. Справедливая стоимость

Методы по сектору IT-площадки:

| Метод | Цена | Статус |
|---|---|---|
| DCF (Гордон, FCF среднее 2024-25) | 78.8 руб | нижняя граница; implied EV/EBITDA 2.4x = очень жёсткая ставка 27.2% |
| DCF (FCF от 2025) | 98.2 руб | альтернативная база |
| Исторический P/E (7.6x × EPS_fwd) | 172.8 руб | средний P/E 3л |
| Relative peers EV/EBITDA (7.0x) | 197.7 руб | со скидкой за ликвидность |
| CAPM 12m | 145.9 руб | таргет 12 месяцев |
| Дивидендная (Гордон) | 96.0 руб | консервативная |
| P/B | — | not_applicable |

**Коридор справедливой цены: 96 — 159 — 198 руб**

Расхождение методов ~2.5x (DCF vs relative) объясняется высокой ставкой дисконтирования (27.2%): при ставке ЦБ 14.5% мультипликатор Гордона сжимается, DCF даёт нижнюю границу. По мере снижения ставок оценка смещается к верхнему кластеру. Conservative = дивидендная/DCF (96-100 руб), base = CAPM+P/E (159 руб), optimistic = peers (198 руб). Текущая цена 129.91 руб — в диапазоне, ближе к conservative-base.

**Надёжность оценки ограничена**: только 3 года истории, нет брокерского консенсуса, баланс неполный. Ставки на снижение ключевой ставки критически влияют на DCF и дивидендную модель.

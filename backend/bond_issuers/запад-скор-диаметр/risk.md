- Премия: **около нуля** — для класса AAA это норма, светофор 🟢 (риск оплачен по уровню класса, аномалии нет).

- **Госгарантия РФ по основному долгу → выпуски рейтинга AAA(RU) (АКРА, подтверждено 2025)** {факт} — при дефолте эмитента основная сумма погашается из федерального бюджета → LGD по телу стремится к нулю, кредитный риск держателя квазисуверенный.

- **Магистраль — безальтернативная транспортная артерия СПб, барьеры входа высокие** {суждение} — устойчивость денежного потока на горизонт бумаги высокая, спрос неэластичный.

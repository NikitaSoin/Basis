# Ручная выгрузка Росстата (fedstat) — как заполнить

**Почему руками.** fedstat.ru закрыт антибот-WAF: возвращает `403 Forbidden` на
любой машинный запрос — на главную, на страницу показателя, на data.do, при любом
User-Agent (проверял Chrome, Googlebot, без UA) — **даже с боевого сервера в РФ**
(подтверждено диагностикой, не предположение). Поэтому автоматический ингест
невозможен, и история Росстата закрывается файлом, который выгружаешь браузером.

ЦБ-показатели (ставка, инфляция, инфл. ожидания, прогноз ЦБ, курсы, M2),
FRED и World Bank грузятся **машинно автоматически** — их трогать не нужно.

## Что заполнять
Скопируй `rosstat_manual.example.csv` в `rosstat_manual.csv` и впиши строки.
Колонки: `indicator,period,value,metric`

- **indicator** — код: `unemployment` (безработица, %), `ppi` (цены производителей, г/г %),
  `real_wage` (реальная зарплата, г/г %), `real_income` (реальные доходы, г/г %),
  `gdp` (ВВП, г/г %).
- **period** — `ГГГГ-ММ` (месяц), `ГГГГ-Qn` (квартал), `ГГГГ` (год).
- **value** — число (точка или запятая).
- **metric** — можно не заполнять (возьмётся по умолчанию из конфига).

## Где взять числа на fedstat (в браузере)
- Безработица — fedstat.ru/indicator/43062
- Цены производителей (ИЦП) — fedstat.ru/indicator/57609
- Реальная начисленная зарплата — fedstat.ru/indicator/43245
- ВВП — fedstat.ru/indicator/33223
- Реальные доходы — fedstat.ru/indicator/33464

На странице показателя есть кнопка выгрузки таблицы — перенеси период+значение.

## Safeguards (что делает бэк сам)
- Валидация диапазона (min/max в конфиге) — мусорное число не попадёт.
- `ingested_via='rosstat'` (официальный приоритет): Лента новостей **не перетрёт**.
- Идемпотентно: повторный прогон не дублирует. Неизвестный код / пустое значение —
  пропуск, существующие данные не портятся.

Файл подхватывается автоматически при старте и в ежедневном задании.

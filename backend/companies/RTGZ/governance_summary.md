# Корпоративное управление — RTGZ (ПАО «Газпром газораспределение Ростов-на-Дону»)

*Эмитент: региональная газораспределительная организация (ГРО) в составе вертикали
«Газпром межрегионгаз». Данные — открытые источники (РСБУ, агрегаторы, MOEX,
e-disclosure). Качество данных: низкое. Дата: 01.06.2026.*

## 1. Кто контролирует
**Факт.** Контролирующий акционер — АО «Газпром газораспределение» с долей ~74,99%
капитала (≈106 581 из 142 108 акций). Конечный контур контроля — группа «Газпром»
(государство) косвенно. Прочие ~25% (free float + неустановленные пакеты) в открытых
источниках по составу не раскрыты; реально торгуемый биржевой free float существенно
меньше 25%. Два класса акций: обыкновенные **RTGZ** (120 246 шт., 84,6% капитала,
1 голос) и привилегированные **RTBGP** (21 862 шт., 15,4% капитала).
**Оценка.** Контур группы прозрачен, офшоров/номиналов не видно (прозрачность 3/5),
но точная контролирующая доля по разным раскрытиям расходится (75% капитала / до
86,65% обычки) — сверить по списку аффилированных лиц.
**Вывод для миноритария.** Все ключевые решения (дивиденды, инвестпрограмма)
принимает группа «Газпром»; миноритарий на них не влияет.

## 2. Дивиденды
**Факт.** По обыкновенным акциям дивиденды не платятся с выплаты за 2011 г. (231,85 ₽,
май 2012). Последние выплаты по обычке — 2006–2011 гг. За 2023 и 2024 годы ГОСА прямо
решало «не выплачивать». При этом компания в большинстве этих лет была прибыльной
(РСБУ: 0,79–2,26 млрд ₽ в 2020–2023, 0,74 млрд ₽ в 2025). Историю выплат по префам
RTBGP в открытых данных реконструировать не удалось.
**Оценка.** Дивидендная предсказуемость для миноритария — 1/5: 13 лет подряд без
выплат по обычке при наличии прибыли, целевого payout нет.
**Вывод.** Ждать дивиденда по обыкновенной акции оснований нет; доходность бумаги —
только курсовая, при малой ликвидности. Это сознательная модель ГРО: прибыль
удерживается/идёт в газификацию, центр распределения прибыли группы — не на уровне ГРО.

## 3. Отношение к миноритариям
**Факт.** Агрессивных действий не зафиксировано: принудительного выкупа (squeeze-out
при контроле ~75% недоступен — порог 95%), делистинга, допэмиссий/размытия в открытых
данных нет. Главный негатив — пассивное удержание прибыли (нет дивиденда по обычке).
Сделки со связанными сторонами внутри вертикали вероятны по характеру бизнеса (сбыт
газа ведёт отдельная компания группы), но в РСБУ-раскрытии не детализированы.
**Оценка.** Защита миноритария 2/5: интересы системно вторичны, но без прямого
«грабежа». Тренд скорее стабильно-нейтральный, без улучшения.
**Вывод.** Риск для миноритария — не размытие, а вечное удержание прибыли «вверх» по
группе. Подтвердить/опровергнуть вывод прибыли через связанные стороны по открытым
данным нельзя (низкая уверенность).

## 4. Управление и прозрачность
**Сильные стороны.** Понятный контролирующий собственник; естественная монополия с
регулируемым тарифом ФАС (низкий риск банкротства); отсутствие выкупов и размытий.
**Главные риски.** (1) Нет дивидендов по обычке более 10 лет — *high*; (2) низкая
прозрачность: только РСБУ, без построчных EBITDA/CapEx/долга/денежных потоков, МСФО
нерегулярна, брокерского покрытия нет — *medium*; (3) возможный вывод прибыли через
связанные стороны группы — *medium*, не подтверждён; (4) низкий free float — *medium*.
Состав и независимость совета директоров по открытым источникам не подтверждены
(оценка 3/5 — нейтрально, низкая уверенность).

**Общий вывод (overall 2/5).** Слабое управление с точки зрения миноритария:
доминируют отсутствие дивидендов по обыкновенной акции и информационная асимметрия при
контроле ~75% у группы «Газпром». Смягчает оценку отсутствие агрессивных действий и
понятный бенефициар на уровне группы. Оценка дана с пометкой о низкой уверенности —
ключевые пробелы: точная контролирующая доля, состав 25% миноритарного блока, история
дивидендов по префам, структура сделок со связанными сторонами и состав совета
директоров (требуют сверки с уставом, годовым отчётом и списком аффилированных лиц на
e-disclosure.ru).

# Корпоративное управление — ПАО «Саратовэнерго» (SARE)

**Общий вывод (assessment):** управление слабое с точки зрения миноритария по
обыкновенной акции — компания полностью подконтрольна государственной группе, а
денежный поток акционеру по обычке отсутствует.

## 1. Кто контролирует компанию
**Факт:** контролирующий акционер — ПАО «Интер РАО ЕЭС» (~83,5% капитала; доля
требует сверки по e-disclosure.ru). «Интер РАО» — госконтролируемый холдинг,
поэтому конечный бенефициар — государство.
**Вывод:** это госкомпания-«дочка». Все ключевые решения (распределение прибыли,
стратегия, сделки) принимает мажоритарий; миноритарий по обыкновенным акциям не
влияет на собрание. Логика решения по бумаге определяется не корпоративными
правами, а политикой материнской группы.

## 2. Дивиденды: политика на бумаге vs практика
**Факт:** по обыкновенным акциям (SARE) дивиденды системно не выплачиваются
(2020-2024). Реальные выплаты идут только по привилегированным акциям SAREP,
которые имеют приоритетное право на фиксированный дивиденд по уставу.
Прозрачной дивполитики по обычке компания не раскрывает.
**Вывод (главный, честный):** держатель обыкновенной акции не получает участия в
прибыли наличными. Доходность по SARE возможна только через рост котировки, не
через дивиденд. Накопленный денежный буфер (~1,86 млрд ₽ на конец 2024)
контролируется мажоритарием и до держателя обычки может не дойти.

## 3. Отношение к миноритарию (конкретные прецеденты)
- **Факт:** при наличии прибыли в 2023-2024 и денежного буфера распределения по
  обычке не было. **Вывод:** прибыль удерживается, миноритарий по обычке не
  участвует в её распределении.
- **Факт:** собственный капитал был отрицательным в 2020-2022 (накопленный
  убыток), вышел в плюс лишь в 2023. **Вывод:** был период, когда распределение
  было невозможно в принципе, а устойчивость держалась на поддержке группы.
- **Факт:** контроль ~83,5% даёт мажоритарию квалифицированное большинство.
  **Вывод:** инструментов защиты у миноритария по обычке практически нет.

## 4. Прозрачность и менеджмент
**Факт:** раскрытие минимальное — публикуется РСБУ-отчётность, но детальная
структура расходов, разбивка капитала по классам акций и состав совета
директоров непрозрачны; брокерского покрытия нет (третий эшелон).
**Оценка:** менеджмент и совет формируются «Интер РАО», независимость совета как
фактор защиты миноритария ограничена. Это нормальная для подконтрольной «дочки»
картина, но для частного инвестора означает низкую наблюдаемость.

## 5. Ключевые риски управления
1. **Отсутствие дивидендов по обычке** (severity: high) — доход только от цены.
2. **Единоличный госконтроль ~83,5%** (high) — интересы мажоритария и
   миноритария по обычке могут расходиться.
3. **Малоликвидность и слабое раскрытие** (medium) — третий эшелон, мало данных.
4. **Историческая финансовая хрупкость** (medium) — отрицательный капитал
   2020-2022, волатильная прибыль, зависящая от резервов по дебиторке.

## Что не раскрыто (data_flags)
- Доля «Интер РАО» и разбивка капитала по классам не сверены по e-disclosure.ru.
- Точный free float по обычке не подтверждён Мосбиржей.
- Дивполитика по обычке как документ отсутствует; вывод — по истории выплат.
- Состав совета директоров, независимые директора, казначейский пакет не подтверждены.
- Параметры дивиденда по префу SAREP здесь не детализированы (преф выводится отдельно).

*Примечание: привилегированные акции SAREP отдельным файлом не создаются — их
управленческий профиль выводится из обыкновенных, ключевое отличие — наличие
приоритетных дивидендов по уставу.*

## ВЕРДИКТ {light:red}
Техдефолт 02.06.2026 по купону + рейтинг CC: вопрос больше не про «оплачена ли доходность» — а про возврат вложенного.

## АРИФМЕТИКА
- Состоявшийся техдефолт → светофор-премия не строится (стоп-правило №4). Доходность как «премия за риск» нерелевантна — деньги уже не пришли в срок.
- Цена ~71,6% номинала (716 руб) {факт} — рыночная оценка ожидаемого исхода: устранение техдефолта и обслуживание, либо реструктуризация/полный дефолт с recovery. Дисконт ~28% = заложенная вероятность невозврата части тела.
- «YTM ~69%» в терминалах {оценка} — артефакт дистресса (доходность от обвалившейся цены к номиналу), НЕ реальная доходность.

## ГЛАВНЫЕ АРГУМЕНТЫ
- **Технический дефолт 02.06.2026: из купона 7,479 млн руб выплачено лишь ~0,8 млн, недоплата 6,679 млн руб, срок устранения — до 17.06.2026** (НКР, V1.ru, 03.06.2026) {факт} — нехватка денег на купон в 7,5 млн руб у эмитента с активами 401 млн руб → острый дефицит ликвидности (блок A=5).
- **ВТОРОЙ техдефолт за 9 месяцев: первый — сентябрь 2025, устранён в срок** {факт} — повторяемость = хронический кассовый разрыв → преддефолтное состояние (блок C — история обслуживания плохая).
- **Рейтинг CC.ru, «на пересмотре с возможностью понижения» (НКР, 03.06.2026), снижен с B+.ru** {факт} — траектория BB-.ru (2024) → B+.ru (09.2025) → CC.ru (06.2026) → высокая вероятность полного дефолта (блок F=5).
- **Руководитель (Е. Остапенко): в 2025 на обслуживание кредитов ушло ~100 млн руб (60 млн — проценты), спрос в машиностроении упал на 30–70%** (V1.ru, 09.06.2026) {факт} — процентная нагрузка съедает поток на фоне падающего сбыта → структурная проблема (блок A/D).

## ПРОВЕРКА ПОТЕРЯМИ
Для группы CC PD на год >20–40%, с уже состоявшимся техдефолтом PD полного дефолта ~0,5–0,7. LGD для необеспеченного ВДО ~70%, у мелкого промпредприятия со специфичными активами (станки, запасы) ликвидационная стоимость ниже → ожидаемая потеря может превышать текущий дисконт (~28%). Цена 71,6% может НЕ полностью отражать риск при банкротстве. Речь о recovery, не премии. 🔴 {оценка}

## ТРИГГЕРЫ
- Устранение техдефолта до 17.06.2026 (доплата 6,679 млн руб) → из «дефолт» обратно в «дистресс CC», цена может частично восстановиться → потолок 🟠.
- Неустранение к 17.06.2026 → фактический дефолт → дело о банкротстве, вопрос recovery → 🔴.
- Активность ПВО, заявления о реструктуризации (продление, снижение купона) → сигнал по recovery. {суждение}

## БЛОК A — Платёжеспособность {score:5}
- **A1 Долговая нагрузка.** Активы 401,3 млн руб (2025) против 289,0 млн (2024), +38,8% (Boomin/cbonds); облигационный долг 100 млн + банковские кредиты (обслуживание ~100 млн руб/год в 2025) — банковский долг кратно больше облигационного. {факт} — рост активов на заёмные деньги при падающем сбыте = пирамида.
- **A2 ICR.** Проценты ~60 млн руб/год при выручке I кв.2026 ~81 млн руб; неоплата купона 7,5 млн руб. {факт} — ICR фактически <1, проедает себя.
- **A3 Ликвидность и стена погашений.** Техдефолт ровно по нехватке 6,7 млн руб; ближайший call 01.12.2026, амортизация 30.11.2027. {факт} — подушки нет; каждый купонный период = новый риск.
- **A4 Качество денежного потока.** Выручка есть (~81 млн руб/квартал), но кэша на купон нет; авансовые налоги 24,5 млн при I кв. — оборотка заморожена. {факт/оценка}
- **A5 Анти-рисование.** Только РСБУ → ограничено. Рост активов +38,8% за год на фоне дефолта по купону подозрителен — проверка на рост запасов/дебиторки/НМА; данных нет. {суждение} → A=5.

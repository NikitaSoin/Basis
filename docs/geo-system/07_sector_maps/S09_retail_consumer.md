# S09. Ритейл и потребительский сектор: секторная гео-карта

**Периметр:** X5, Магнит, Лента, Fix Price, М.Видео, Henderson, Novabev, Черкизово/Русагро-класс.

## 1. Гео-профиль сектора
Прямой гео минимален (E1≈1–1,5: продовольствие — carve-out-класс даже риторически), действующие оси — опосредованные: **инфляционно-ставочная** (G12: продовольственная инфляция = номинальный рост выручки при давлении на трафик и маржу; ставка = долг и капекс — K27-скрининг применим: М.Видео как вечный якорь), **импортно-платёжная** (E4/E5 для non-food: K17-трение = себестоимость техники/одежды; параллельный импорт = решённая, но платная задача — М.Видео-урок K27 §3: гео-адаптация может быть успешной при смерти от долга), **периметральная** (E9: X5 — эталон K12; Fix Price/прочие переезды — по лекалам), **курсовая** (E8: импортная корзина non-food против рублёвой выручки — пис-укрепление = маржинальный подарок, редкий сектор с ПОЛОЖИТЕЛЬНОЙ курсовой пис-бетой).

## 2. Трансмиссия
G12 → триада «инфляция-трафик-ставка» (доминанта; продуктовые — бенефициары умеренной инфляции, жертвы долговой ставки); K17/G4 → non-food себестоимость и оборотный капитал (предоплаты = замороженная ликвидность); C04 → импортная маржа; кадровый канал (K11) → персонал магазинов/логистики (зарплатная спираль в OPEX); агро-подгруппа → экспортные пошлины/квоты (G12-фискальный) + мировые цены (G13).

## 3. Сценарная карта
S1: тройной попутный ветер — ставка вниз (долг/капекс/мультипликаторы), рубль крепче (импортная маржа non-food), потребительская уверенность вверх; сектор — тихий бенефициар мира без пис-корзинного статуса (беты умеренные, устойчивые). S2: ставочно-потребительский плюс. S3 (база): продуктовые — защитный кэш-флоу с фискально-зарплатным трением; non-food — K17-налог и слабый дискреционный спрос. S4: продуктовые — максимально защитные (спрос неэластичен), non-food — удар по дискреционке; регуляторные риски цен (заморозки/соглашения — меню отработано).

## 4. Мониторинг
Недельная инфляция (Росстат — прямой драйвер), ставка/сигнал, LFL-раскрытия (трафик vs чек — где инфляция, где спрос), K17-параметры (сроки/комиссии платежей non-food), зарплатные индексы персонала, экспортно-пошлинные решения (агро), регуляторные сигналы по ценам (ФАС-риторика = ранний маркер заморозок).

## 5. Анти-паттерны / red-team флаги
«Гео-невинность» = отсутствие анализа (C09 §6в: ставочно-инфляционный канал — их главная жизнь). Инфляционная выручка как «рост бизнеса» (расщеплять номинал/трафик). М.Видео-класс через санкционную рамку (K27: смотри баланс — родовая травма проекта). Игнор положительной курсовой пис-беты non-food (редкий знак — использовать в сценарных строках). Агро без пошлинно-квотной строки (фискальная мишень при мировых ралли — «где толсто» работает и тут).

## 6. Дифференциация
X5 — пост-K12 эталон: чистый периметр, ставочно-операционная история, сектор-бенчмарк; Магнит — корпоративно-владельческие сюжеты поверх операционки (частично ИНСТ); Лента/Fix Price — ставочно-форматные истории (K27-скрининг долга); М.Видео — вечный учебник опосредованного канала (баланс > гео); Henderson/Novabev — потребительская дискреционка с импортно-курсовой осью; Черкизово/Русагро-класс — агроэкспортёры: пошлины+мировые цены+внутренняя инфляция кормов (тройная декомпозиция).

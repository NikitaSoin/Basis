**(3) Противоречия.**
- Расхождение методов ~2,4× — структурное, не ошибка: дивидендный якорь префа (5,4–6,4 ₽ при payout ~22%) vs активный кластер (NAV-disc 39,2 ₽; EV/EBITDA-rel 34,2 ₽) vs доходный (DCF 26,6; P/E-adj fwd 26,3). Причина — governance-замок: кэш 20,7 млрд физически есть, миноритарию недоступен.

- EV отрицателен (кэш > mcap) → EV/EBITDA-история бессмысленна, помечено.

**(4) Синтез-вердикт по ПРЕФУ (дивдоходность + NAV + спред к обычке).**
- **Дивдоходность (основной метод для префа):** при равном дивиденде 0,9639 ₽ и цене 12,10 ₽ преф даёт **7,97%** против 7,34% у обычки WTCM (13,14 ₽). Преф дешевле обычки на **~7,9%** — умеренный спред, типичный для РФ; за равный дивиденд преф объективно выгоднее обычки как дивбумага. Справедливая цена по дивмодели при req.yield 15-18%: **5,4–6,4 ₽** (нижний якорь при текущем payout 22%); при росте payout до 50-60% → ~13-16 ₽.

- **Вердикт:** глубоко недооценён по активам (P/NAV 22%, P/B ~0,22), но дисконт оправдан governance-замком — кэш заперт, payout низкий, бумага неликвидна. **Справедливый диапазон префа: cons 16 ₽ (P/B-история, +32%) / base 25,5 ₽ (доходный кластер, +111%) / opt 39 ₽ (NAV-disc при улучшении payout, +222%).** Это история стоимости активов с governance-замком, не дивидендная: дисконт снимется только при изменении дивполитики ТПП РФ. Преф предпочтительнее обычки за счёт более высокой дивдоходности при равном дивиденде и почти равной (pari passu) доле в активах. Низкая ликвидность ограничивает реализуемость апсайда. Достоверность medium.

> листинга, исполнитель ГОЗ, раскрывает отчётность ограниченно. Баланс и ОДДС
> не получены из источников (null). Числа — по МСФО через агрегатор Smart-Lab

## 1. Динамика выручки и прибыли

Выручка нестабильна и зависит от госзаказа: 2450 млн (2022) → 3750 (2023,
+53%) → 3560 (2024, -5%) → 3590 (2025, +1%). Рывок 2023 года — вероятно,
наращивание ГОЗ; дальше стагнация на уровне 3,5-3,6 млрд.

Компания убыточна на чистом уровне во всех четырёх раскрытых годах. Убыток
устойчиво сокращается: -1340 → -660 → -453 → -258 млн руб. Это факт, и тренд
положительный. Но само сокращение идёт медленно: при текущем темпе (+43%/год
по улучшению ЧП) выход в ноль — ориентировочно 2026-2027 при сохранении тренда.

Нормализация (adjusted): не применялась. Причина — нет первичной отчётности
(pre_tax_profit, income_tax, разовые статьи недоступны через агрегатор). adjusted
= reported для всех лет. Мост пустой.

Положительный сдвиг в 2024-2025: EBITDA вышла в плюс (300 млн в 2024, 401 млн
в 2025; маржа 8.4% и 11.2%). Операционная прибыль — ноль в 2024, +201 млн в
2025. Убыток на чистом уровне — полностью из-за процентных расходов по долгу
(~439 млн в 2024, 2025 не раскрыт, оценочно 350-450 млн).

Аномалия: gross_margin 2025 = 33.1% (vs 14.0% в 2024). COGS упал с 3060 до
2400 млн при стабильной выручке — возможно, изменение структуры заказов (более
маржинальные продукты) или разовый эффект. Без первичного отчёта подтвердить
невозможно.

## 2. Баланс, долг, ликвидность

Баланс не распарсен — большинство строк null. Из косвенных источников:

- Чистый долг (из business_model.md): 4390 млн (2023) → 4100 млн (2024) →
  оценочно 3829 млн (2025, judgement: тренд -6.6%/год).
- Собственный капитал (из governance.json): -4570 млн руб. на 2025. Отрицательный
  — результат накопленных убытков. P/B бессмысленен. Юридически исключает
  выплату дивидендов.
- Net debt/EBITDA: 13.7x (2024) → 9.5x (2025). Формально снижается, но по-прежнему
  критически высокий. Норма для машиностроения — 2-3x; 9.5x — экстремально.

Долговая нагрузка — главный структурный риск: операционно завод почти вышел
в ноль (EBITDA +401 млн, opEx +201 млн в 2025), но процентные расходы по долгу
4 млрд руб. при ставке ЦБ 14.5% превращают операционный плюс в чистый убыток.

## 4. Мультипликаторы

P/E: не применим (убытки во всех годах).
P/B: не применим (капитал отрицательный).
P/S (текущий): 1.02x — нейтрально для производителя, но без прибыли не информативен.

EV/EBITDA (при EV ~ 7497 млн, EBITDA 2025 = 401 млн): 18.7x — существенно выше
норм для убыточного машиностроения (нормальный диапазон 4-8x). Это означает, что
рынок уже закладывает ожидание роста EBITDA в 3-4 раза, не снижение долга.

EV/Sales (текущий): 2.09x — для убыточного производителя очень высоко (норма
убыточных промышленников 0.3-0.8x). Текущая цена предполагает нереалистичный
мультипликатор.

Исторические средние мультипликаторы не рассчитаны: P/E во всех годах убыточен,
P/B недоступен, EV/EBITDA положительен только в 2024-2025 (база 2 года — CV не
применима из-за малой выборки).

## 6. Справедливая стоимость: методы и коридор

Применены пять методов; большинство неприменимы или дают нулевой результат.

| Метод | Результат | Статус |
|-------|-----------|--------|
| DCF | — | not_applicable (FCF недоступен, убытки) |
| Исторический P/E | — | not_applicable (все годы убытки) |
| Исторический P/B | — | not_applicable (equity < 0) |
| Relative EV/EBITDA | 0 ₽ | ok — долг > EV_fair при любом разумном мультипликаторе |
| CAPM (12м) | 8.36 ₽ | условно ok — это «цена риска», не intrinsic value |
| Дивидендная | — | not_applicable (DPS = 0) |

**Ключевой вывод оценки:** При EV/EBITDA сектор-медиане 5x:
- EV_fair = 5 × 401 = 2005 млн руб.
- Чистый долг оценочно 3829 млн → Equity_fair = 2005 − 3829 = −1824 млн → 0 ₽.

Для EV/Sales: equity_fair < 0 при EV/Sales < 1.07x (отраслевая норма 0.3-0.8x).

По фундаменталу долг полностью поглощает операционную стоимость бизнеса.
Текущая цена 6.52 ₽ = рынок платит за «опцион» на разворот (снижение долга
при реструктуризации / снижении ставки ЦБ / резком росте EBITDA).

CAPM 8.36 ₽ (требуемая доходность 28.1% за 12 мес.) — это условная «цена
риска», а не intrinsic value. При отрицательном капитале этот таргет нельзя
использовать как ориентир.

**Итоговый коридор (спекулятивный, широкий):**
- Conservative: 0 ₽ (фундаментальный потолок — все методы дают ≤ 0)
- Base: 2 ₽ (допущение: частичное снижение долга + сохранение EBITDA)
- Optimistic: 4 ₽ (допущение: реструктуризация долга до ~2 млрд + рост EBITDA)

Расхождение методов объяснено: CAPM не учитывает структуру баланса.

2. Чистый долг 2025 — оценочный (judgement); точное значение может отличаться.
3. Gross_margin 2025 нуждается в верификации первичным отчётом.
4. Компания под санкциями 27+ стран; ГОЗ ограничивает раскрытие.
5. Акции крайне малоликвидны (3-й уровень) — котировка не отражает реальную
   ликвидную стоимость, волатильность высокая.

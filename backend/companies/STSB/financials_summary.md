# Финансы и оценка — ПАО «Ставропольэнергосбыт» (STSB, обыкновенные акции)

> Качество данных: **medium**. Источник: smart-lab агрегированные таблицы МСФО 2021–2025.
> Первичный PDF e-disclosure повреждён — примечания недоступны. ОДДС, налог, детализация
> баланса отсутствуют. Перед принятием решений сверить с официальной отчётностью.
> Версия методики: v2 (нормализация reported/adjusted, CV-правило, explain по каждому методу).

## 1. Динамика выручки и прибыли

Выручка росла стабильно: 21,1 → 22,3 → 24,6 → 27,4 → 32,1 млрд руб. (CAGR 11% за 2021–2025).
Рост объясняется индексацией тарифов и сбытовой надбавки в Ставропольском крае.

Чистая прибыль: 298 → 530 → 655 → 1430 → 1340 млн руб. Резкий скачок в 2024 (+2,2x) — по
бизнес-модели обусловлен тарифным пересмотром надбавки; разовых факторов в агрегированном
источнике не обнаружено. В 2025 ЧП снизилась (-6,3%) при росте выручки +17%: операционные
расходы опережают темп выручки.

**Нормализация: adjusted = reported** (чек-лист по всем годам — явных one-off не выявлено;
ограничение: первичный PDF повреждён). Мост пустой. ETR не раскрыт.

EBITDA маржа: 2,8–5,3%; ROS: 1,4–5,2% — крайне тонкие, характерны для энергосбыта
(посредник, зарабатывает только сбытовую надбавку). Волатильность ROS снижает надёжность
форвардных прогнозов.

## 2. Баланс: долг, ликвидность, капитал

Баланс агрегированный: детализация долга, дебиторки, кредиторки и ОС недоступна.

**Долг:** в агрегаторе не раскрыт. По бизнес-модели: чистая денежная позиция отрицательная
(кэша больше долга) — принято net_debt = -cash. 2025: cash 1890 млн, net_debt = -1890 млн.
EV = капитализация 3270 + (-1890) = 1380 млн руб. — т.е. кэш составляет 58% от капитализации.

**Капитал** вырос в 3,8 раза (1220 → 4690 млн руб.) за счёт накопления нераспределённой
прибыли (дивиденды по обычке не платятся). ROE 25–41% — высокий для регулируемого сбыта.
Tangible equity = total equity (НМА не раскрыты; предположение).

Net debt/EBITDA: -1,27x в 2025 (чистая кэш-позиция растёт). Долговой нагрузки нет.

## 4. Мультипликаторы: дорого или дёшево?

Текущие мультипликаторы (от цены 3,842 руб.):
- P/E = 2,46 (исторический медиан 5,04) — ниже в 2 раза
- P/B = 0,70 (исторический медиан 1,38) — ниже балансовой стоимости
- EV/EBITDA = 0,93 (исторический средний 1,46) — минимум за 5 лет

По всем мультипликаторам акция торгуется ниже своих исторических значений. Для компании
с ROE 25–41% и отрицательным долгом это нетипично. Вероятные причины дисконта:
— По обычке дивидендов нет (DPS=0), возврат только через курсовой рост;
— Госконтроль с 08.2025 (72,31% Росимущество): будущая политика по обычке не определена;
— Малоликвидность (3-й уровень листинга); размытый free-float (27,69%).

## 6. Справедливая стоимость

Методы оценки (применено 4 из 6):
- **Исторический P/E** (медиана 5.04 × EPS_forward 1.30 руб.) = **6.56 руб.**
- **Исторический P/B** (медиан 1.38 × BVPS 5.46 руб.) = **7.53 руб.**
- **EV/EBITDA** (среднее 1.46 × EBITDA 1490 млн + кэш) = **4.73 руб.**
- **CAPM** (Ke=20.9%, beta=0.7, implied P/E=4.78) = **7.46 руб.**
- DCF: insufficient_data (нет ОДДС/capex)
- Дивидендная: not_applicable (DPS=0 по обычке)

**Справедливый коридор: 4,73 — 7,86 руб. Базовый сценарий: 6,83 руб.**

Разброс методов ~66% — расширенный коридор из-за: (а) data_quality=medium; (б) нет FCF;
(в) волатильная ROS; (г) неквантифицированный дисконт за отсутствие дивидендов и госконтроль.

**Ненадёжность оценки:** EV/EBITDA метод даёт минимум (4,73 руб.) из-за исторически низкого
мультипликатора — возможно, рынок систематически дисконтировал бумагу. Методы P/E и P/B
(6,56–7,86 руб.) учитывают рост прибыли и капитала, но опираются на исторические
мультипликаторы, рассчитанные от текущей цены (условно).

## Отчёт по методике

**Период:** 2021–2025 (МСФО, агрегатор smart-lab)

**Нормализация:** adjusted = reported; мост пустой (явных one-off не обнаружено при
data_quality medium; ограничение — первичный PDF повреждён). ETR не раскрыт — нормализация
по ставке 25% не проводилась (нет базы).

**Методы применены:** historical_pe (forward + backward), historical_pb, ev_ebitda_historical, CAPM

**Методы not_applicable/insufficient_data:**
- DCF: insufficient_data — ОДДС и capex null, FCF невозможен
- Дивидендная: not_applicable — DPS=0 по обычке за 2021–2025

**Расхождение методов:** ~66% (4.73–7.86 руб.) — превышает 30%, коридор расширен. Причина
задокументирована в methods_divergence_note.

**data_flags ключевые:** нет ОДДС; нет детализации баланса; PDF повреждён; beta суждение;
дисконт за дивиденды/госконтроль не квантифицирован.

**peers.json:** не обновлялся (нет peers-данных для сектора utilities_energy_retail).

# S01. Нефть (ВИНКи): секторная гео-карта

**Периметр:** Роснефть, Лукойл, Газпром нефть, Татнефть, Сургутнефтегаз, Башнефть.

## 1. Гео-профиль сектора (доминирующие оси)
E1 — от 5 (Роснефть, Лукойл, ГПН, Сургут — SDN: K19/K23) до 2,5–3 (Татнефть, Башнефть — вне блокировок, но в «очереди» по правилу кандидатской лестницы K19 §6.1). E7/G5 — главный операционный канал: реализованная цена = Brent − дисконт (декомпозиция C05 §1 обязательна). E3 — маршрутная зависимость: балтийское плечо (уязвимо к B04 L4–L5) vs ВСТО-премия. E10 — только Лукойл исторически (заложник, K23). E8 — классический экспортный: слабый рубль = маржа, пис-укрепление = встречный ветер (половина E14-механики). Фискальный слой (совместно с ИНСТ): демпфер, НДПИ-настройки, «налоговый дисконт» (K10 §2) — государство постоянно перераспределяет маржу.

## 2. Трансмиссия
G5 → цена реализации (через фильтр K10 §4: фискально-курсовой!); G1-тыл → переработка/демпфер (K26: связка «сырая↔продукты» — недозагрузка НПЗ высвобождает экспорт сырой); G3/G4 → комплаенс-премия покупателей и платёжные лаги; G12 → изъятия («где толсто»); G13 → мировой цикл (в декомпозиции всегда отдельной строкой — K02/K29-дисциплина).

## 3. Сценарная карта
S1: делистинг мейджоров — вероятный первый козырь сделки (K23 §6.5) → двойной эффект: санкционная нормализация (+) против крепкого рубля и падения риск-премии нефти (−) — нетто умеренно-положительный с дисперсией; Татнефть-класс выигрывает меньше (нечего снимать). S2: слабый плюс (ставка/рубль). S3 (база): боковик под пилой G5 и фискальными качелями; ВСТО-профили устойчивее балтийских. S4: рубль-компенсатор смягчает (war-бета относительная), но L5-блокада — хвостовой убийца балтийского экспорта (B04 §2).

## 4. Мониторинг (сектор-специфично, поверх B03)
Дисконт Urals/спред ESPO (еженедельно), GL-календарь по Лукойлу (телеграф C02 §2), листинг-волны флота, загрузка переработки и демпферные пороги (K26), «налоговый дисконт» в бюджетных пакетах, статус потолка ($47,6/спор о $44), индийские месячные закупки.

## 5. Анти-паттерны / red-team флаги
«Дисконт → котировки» напрямую (мимо фискально-курсового фильтра — K10 §4). Мировая цена в компанейском гео-дисконте (двойной счёт G13). Игнор очереди кандидатов у «чистых» имён. Дивидендные модели без строки фискальной дискреции (K08/K30). Сургутская «кубышка»: курсовые переоценки ≠ операционный результат (расщеплять всегда, C04 §4).

## 6. Дифференциация (главная гео-ось каждого)
Роснефть — E1=5 + ВСТО-логистика (устойчивее прочих под санкциями); Лукойл — E1=5 × E10-заложник (международные активы = отдельная сценарная ветка); ГПН — E1=5 + максимальная доля переработки (K26-чувствительность); Сургут — E1=5 + кубышка (ставочно-курсовая история поверх нефтяной); Татнефть — «чистый» кандидат: премия за неблокированность = измеримый санкционный спред сектора; Башнефть — миноритарная защита слабая (ИНСТ-слой доминирует над гео).

# K28. Сегежа: тройная петля (рынок ЕС × ставка × курс) и разводнение как финал

**Статус:** якорный кейс, полный формат. Эталон многоканальной атрибуции (C08 №3) и нелинейного взаимодействия каналов.

## 1. Паспорт
Тип: комбинированный кейс — прямой гео-удар по выручке, добитый макрофинансовым каналом через баланс. Период: 2022 – начало 2025 (SPO). Оси: E2 (потеря премиального рынка), E3 (переворот логистики), G12/K27 (ставка), C04 (курс), C06 (разводнение как форма реализации риска). Роли: бэктест — out-of-sample резерв V01; учебный кейс «взаимодействие каналов».

## 2. Хронология [ФАКТ, если не отмечено]
2021 — IPO на пике цикла пиломатериалов (глобальное ковидное ралли цен); агрессивная долговая экспансия (M&A: ИФР, НЛХК). 2022 — 5-й пакет ЕС: запрет импорта российской древесной продукции (лето) → потеря премиального европейского рынка (значимая доля выручки ключевых сегментов [ОЦЕНКА: порядка трети по группе, выше — по фанере/пиломатериалам премиум]); одновременно разворот логистики на восток: плечо и стоимость доставки кратно вверх, маржа экспорта в Китай структурно ниже европейской; крепкий рубль 2-й половины 2022 дожимает экспортную маржу (C04). 2022–2023 — мировые цены пиломатериалов нормализуются с ковидных пиков (глобальный цикл — НЕ гео). 2023–2024 — ставка (K27) превращает долг ~120+ млрд ₽ в жёрнов: OIBDA перестаёт покрывать проценты, реструктуризации, котировка с ~10 ₽ (пик 2021) к ~1,5 ₽. Начало 2025 — SPO-спасение: ~113 млрд ₽ по 1,8 ₽/акция, основной объём — конвертация требований банков-кредиторов и участие контролирующего акционера; количество акций вырастает в разы → доля прежних миноритариев размыта кратно [структура — ФАКТ, кратности — ОЦЕНКА].

## 3. Механика по каналам — декомпозиция (доли — [ОЦЕНКА] для метода, пересчитываются в V01)
Канал A, прямой гео (E2+E3): потеря премиального рынка + логистический переворот — необходимое условие кризиса; вклад в разрушение юнит-экономики ~30–40%. Канал B, глобальный цикл: нормализация мировых цен с аномальных пиков — ~20–30% (проверка по не-российским пирам: их маржа тоже сжалась — контрольная группа C08). Канал C, макрофинансовый (ставка×долг): ~30–40% и **решающий для ФОРМЫ исхода** — именно он превратил «плохие годы» в разводнение. Канал D, менеджмент/структура: долговая экспансия на пике цикла — предсуществовавшая уязвимость, множитель всех прочих. **Главная формула кейса: гео-шок × leverage = нелинейность.** Ни один канал по отдельности не был смертелен (безлевереджный аналог пережил бы и ЕС-запрет, и цикл); произведение — смертельно для акционерной стоимости.

## 4. Рыночная динамика
Медленная смерть, не шок: −80%+ растянулись на 2,5 года серией ступеней (каждая реструктуризация/отчёт — ступень) → окно выхода существовало долго, сигналы читались: ковенантные новости, перенос купонов, риторика «поддержки акционера» — стандартная лестница к разводнению (каталог сигналов — в правило C06). SPO-цена (1,8 ₽) стала фактическим «справедливым» якорем, обнулив все довоенные ориентиры — терминальный урок о необратимости.

## 5. Развязка
Компания жива, баланс расчищен разводнением; для прежних миноритариев исход эквивалентен потере большей части капитала при формальном избежании дефолта. **Разводнение = форма реализации риска для акционера** — фиксируется как отдельный исходовый класс в C06 (не «выжила/дефолт», а трёхчлен «выжила/разводнение/дефолт»).

## 6. Уроки для шкал
1. **Взаимодействие каналов — умножение, не сложение:** GRE-профиль с высоким E2/E3 при долговой нагрузке >3× получает нелинейную надбавку риска (правило в C07-агрегирование: критические комбинации помечаются, не усредняются).
2. **E2-урок терминальности:** «премиальный рынок» в оценке несёт вероятность политического обнуления в S3 (параллель K22-Газпром); терминальные допущения DCF по экспортёрам в недружественные направления — с этим дисконтом.
3. **Каталог предвестников разводнения** (ковенанты→переносы купонов→«поддержка акционера»→SPO ниже рынка) — в обязательные проверки КОМПАНИИ для закредитованных.
4. **Контрольная группа обязательна:** без сверки с мировыми пирами цикл (канал B) был бы приписан гео — классическая сверхатрибуция.
5. **Цена спасения задаёт новый якорь:** после рекапитализации все прежние ценовые ориентиры недействительны — запрет «отскока к историческим уровням» в аргументации.

## 7. Ex-ante тест
Точка T = лето 2022 (ЕС-запрет): корректный выход системы — «прямой удар по трети бизнеса переживаем сам по себе, но в комбинации с долгом на пике цикла — профиль кандидата на реструктуризацию при развороте ставки; watchlist: ковенанты, рефинансирование 2023–24» → жертва идентифицируется за 1,5–2 года до SPO без знания дат. Проверка атрибуции: заставить агента разложить −80% по каналам A–D и сверить с §3 (сумма ≤100%, гео не доминирует единолично).

## 8. Перенос
Шаблон для всех «экспортёров с плечом»: скрининг E2×leverage в каждом секторном обзоре; для будущих санкционных ударов по новым отраслям (потенциальные: металлы, СПГ) — первым делом сортировка жертв по балансу, потому что форму исхода определит он.

## 9. Источники и метки
Отчётность группы (МСФО), проспект/параметры SPO, тексты 5-го пакета ЕС, ценовые ряды пиломатериалов (глобальные бенчмарки). Доли выручки и вклады каналов — [ОЦЕНКА], метод фиксирован в §3.

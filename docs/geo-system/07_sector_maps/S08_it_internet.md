# S08. ИТ и интернет: секторная гео-карта

**Периметр:** Яндекс (МКПАО), Ozon, HeadHunter, VK, Группа Позитив, Астра, Диасофт, Аренадата, Софтлайн, ЦИАН.

## 1. Гео-профиль сектора
Гео-профиль лёгкий по прямым осям (E1≈1,5–2: блокировки точечны и редки) при трёх специфических: **E9-периметр** — сектор прошёл главную волну редомициляций (Яндекс-раздел 2024: уникальная сделка «актив за периметр со скидкой»; Ozon/HH-переезды по K12-лекалам — навесы отработаны, methodология C03 применена); **кадровый канал** (K11 §6.2) — релокационные волны, дефицит и зарплатная инфляция = давление на юнит-экономику разработки; **G12-дюрация** — рост-мультипликаторы: главная бета сектора ставочная (K27-зеркало со знаком плюс при смягчении). Отдельная подгруппа — **бенефициары изоляции**: импортозамещение софта (Позитив, Астра, Диасофт) = выручка, СОЗДАННАЯ уходом западных вендоров и регуляторным принуждением (реестры, запреты западного ПО в КИИ) — их гео-профиль инвертирован: peace-РИСК (возврат вендоров/конкуренции), war/изоляция-комфорт.

## 2. Трансмиссия
G12 → мультипликаторы и потребительский спрос (Ozon/ЦИАН — ставочно-потребительские); G6 → двусторонне: железо/облака дорожают (издержки) ↔ софт-замещение растёт (выручка подгруппы); кадры (G1/K11) → фонд оплаты; E9-события → навесы/разблокировки по C03-календарям; регуляторный слой (КИИ-дедлайны, реестровые преференции) — квази-гео внутренняя политика (стык с ИНСТ по владельческим сюжетам VK-класса).

## 3. Сценарная карта
S1: РАСЩЕПЛЕНИЕ сектора — потребительско-ростовые (Яндекс, Ozon, HH, ЦИАН) получают двойной плюс (ставка+мультипликаторы+кадровый возврат), импортозаместители — peace-риск (темп возврата конкуренции медленный из-за гистерезиса T09/T15, но переоценка мгновенная — рынок торгует вероятность); знак ВНУТРИ сектора противоположный — единая «ИТ-корзина» в сценарном анализе запрещена. S2: ставочный плюс всем. S3: изоляционная рента заместителей + дюрационное давление на рост. S4: минус фронтально (мультипликаторы), заместители относительно лучше.

## 4. Мониторинг
Ставка/сигнал (дюрация), КИИ-дедлайны и реестровые решения, кадровая статистика (зарплатные индексы ИТ — HH-данные как прокси), E9-календари оставшихся структур, признаки возврата западных вендоров (S1-маркер для заместителей: лицензионные возобновления — ловить первые прецеденты), G6-железо (серверные наценки — C03-канал издержек).

## 5. Анти-паттерны / red-team флаги
Единый сектор-знак в S1 (расщепление обязательно — противоположные беты). Импортозамещение по анонсам (T12 §5: считать по внедрениям/выручке). «Гео-невинность» без кадрово-ставочной строки (C09 §6в: достаёт всех). Экстраполяция изоляционной ренты в терминальность (peace-риск заместителей — не хвост, а сценарная строка). Пропуск владельческо-регуляторных сюжетов (это ИНСТ — маршрутизировать, K31-дисциплина).

## 6. Дифференциация
Яндекс — пост-периметральный флагман: ставка+продуктовый цикл, гео-остаток мал; Ozon — дюрационно-потребительский с E9-хвостом завершённым; HH — прокси рынка труда (K11-канал = его выручка: дефицит кадров ему платит); VK — регуляторно-владельческая история (ИНСТ-доминанта); Позитив/Астра/Диасофт/Аренадата — изоляционная рента с peace-риском (анти-пис-беты сектора); Софтлайн — дистрибуция: G6-трение как бизнес; ЦИАН — девелоперская производная (S10-родство).

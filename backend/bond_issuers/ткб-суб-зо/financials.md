## Финансы эмитента: сможет ли ТКБ расплатиться (и риск суборда)

**Кредитный рейтинг эмитента:** Эксперт РА — **ruBBB**, прогноз стабильный, подтверждён 10.12.2025; НКР — **BBB.ru**, стабильный *(факт)*. Уровень BBB по нацшкале — умеренная надёжность (середина, не верхний эшелон). Достаточность капитала Н1.0 на 01.10.2025 — 13,2%, заметно выросла к 2024 г.; за год активы +15%, кредитный портфель +12%, агентства отметили адекватное качество активов и ликвидность *(факт)*.

**ВАЖНО — природа бумаги *(факт/суждение)*: ТКБ СУБ-ЗО — СУБОРДИНИРОВАННАЯ облигация.** Это означает два специфических риска поверх обычного кредитного:
1. При банкротстве/ликвидации суборд гасится **ПОСЛЕ** всех старших кредиторов и вкладчиков — держатель в очереди почти последний, перед акционерами.
2. **Риск списания / конвертации в капитал:** по российскому регулированию суборд может быть полностью или частично списан (либо мена в акции), если норматив достаточности капитала падает ниже триггерного уровня или вводятся меры по предупреждению банкротства. Купон по суборду эмитент при определённых условиях вправе не платить без объявления дефолта.

Дополнительно: бумага **БЕССРОЧНАЯ (perpetual)** — заранее определённой даты возврата тела нет; погашение возможно только через call по решению эмитента *(факт)*.

**Флаги *(суждение)*:** жёлтый по эмитенту (BBB — средняя надёжность, исторически тяжёлая санация ИТБ); дополнительный риск-флаг по инструменту — суборд + бессрочность + только для квалинвесторов: повышенная доходность здесь есть плата за риск списания и субординацию, а не «бесплатный» доход.
